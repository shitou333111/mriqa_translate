DWI b 值

b 值是什么意思？ 
我该如何选择它？   
b 值 是一个反映用于生成弥散加权图像的梯度强度和时序的因子。b 值越高，弥散效应越强。具体来说，如果 So 是基线时的 MR 信号，ADC 是表观弥散系数，那么施加弥散梯度后的信号 (S) 由下式给出：

S = Soe−b•ADC

因此，项 e−b•ADC 的行为非常类似于在许多其他脉冲序列中发现的 T2 加权项 e−TE/T2。b 的值由操作者在成像前选择。这个选择控制着观察到的弥散加权程度，类似于选择 TE 影响 T2 加权的方式。因此，弥散可以被认为是除 T1 和 T2 之外的另一种弛豫机制。在没有额外弥散梯度的脉冲序列中，这种弛豫机制相对不重要，对最终信号的影响不超过 5%。然而，当施加弥散梯度时，弥散效应被显著放大，并成为组织对比度的主导机制。

在 之前的问答中我们曾说明，由于弥散是粒子在一段时间内穿过一个表面的通量，ADC 的单位形式为 [面积/时间]。为了使 exp(−b•ADC) 无量纲，b 的单位应该是 ADC 的倒数。换句话说，b 应表示为 [时间/面积]。现代 MRI 扫描仪上可用的典型 b 值范围从 0 到大约 4000 s/mm²。
术语“b 值”源自 Stejskal 和 Tanner 于 1965 年发表的里程碑式论文，他们在文中描述了他们的脉冲梯度弥散方法。这项技术至今仍是大多数现代 DWI 脉冲序列的基础，它由两个强梯度脉冲组成，其幅度 (G) 和持续时间 (δ) 相同，并由时间间隔 (Δ) 分隔。仅针对这种特定实现方式的 b 值公式，如右图所示。 

Stejskal-Tanner 脉冲梯度弥散方法。

b = γ² G² δ² (Δ−δ/3)
b 值取决于这些脉冲梯度的强度、持续时间和间隔。通过增加梯度幅度和持续时间，以及加宽梯度脉冲之间的间隔，可以获得更大的 b 值。由于 MR 系统可能使用不同形状和持续时间的脉冲梯度，因此可能难以精确比较不同研究之间的 b 值。
b 值的“最佳”选择没有明确定义，取决于场强、信号平均次数、解剖学特征和预期的病理变化。下面使用 b 值为 0、1000 和 3000 s/mm² 的三幅脑部图像显示，随着弥散加权逐渐增强（表现为皮质脊髓束更亮），噪声也更多。随着 b 值的增加，机械振动伪影也可能成为一个问题。 在实际应用中，目前大多数常规临床 DWI 使用 0 到 1000 之间的 b 值，在中枢神经系统外使用的值略低。 
使用 3 种不同 b 值（0、1000 和 3000 s/mm²）的脑部 DWI 图像
新生儿和婴幼儿的脑部含水量远高于成人，其 T2 值和 ADC 值要长 25-40%。对于这些患者，b 值通常设置得更短，在 600-700 s/mm² 的范围内。 一个有用的经验法则是选择 b 值，使得 (b × ADC) ≈ 1。

进阶讨论 (显示/隐藏)»

尽管上面给出了历史上 Stejskal-Tanner 的 b 值公式，并且大多数教科书都有收录，但这种特定的实现在现代临床或实验系统中很少使用。更常用的是正弦波或梯形脉冲，而不是纯矩形脉冲。对于正弦波脉冲，公式变为 b = 4γ²G²δ²(Δ−δ/4)/π²。如果使用上升时间为 ξ 的梯形脉冲，则 b = γ²G²[δ²(Δ−δ/3) + ξ³/30 −δξ²/6]。

除了脉冲梯度方法外，也研究了其他弥散方案，b 值的计算取决于所用梯度方案的细节。例如，如果在时间 TE 内施加单个长弥散梯度 G，则 b = γ²G²TE³/12。如果使用具有 n 个回波的快速自旋回波技术，则 b = γ²G²TE³/12n²。
