脂肪/水化学位移

脂肪/水化学位移指的是什么？它是如何计算的？  
正如之前的问答中所述，水分子中强电负性的氧原子会拉走覆盖氢原子核的保护性电子云。这种去屏蔽效应使氢质子暴露在比原本更强的局部磁场中。因此，它们比甘油三酯（脂肪）分子中屏蔽更强的质子共振得稍快一些。水和脂肪之间的这种共振频率差异，会随着磁场强度的增加而增加，其缩放因子称为化学位移。
电负性氧对水中氢原子核的去屏蔽作用

脂肪与水之间的化学位移经测量约为 3.5 百万分之一 (ppm)。后缀 ppm 与 x10−6 可互换，就像符号 % 与 1/100, x0.01 或 x10−2 可互换一样。化学位移以比率形式给出，是因为实际的频率差（以 Hz 为单位）取决于所施加的磁场。

计算实际的脂肪-水频率差很简单——只需将你所用场强下的拉莫尔频率乘以 3.5 ppm 或 3.5 x 10−6。
由于不均匀性效应，大解剖区域内脂肪和水的谱峰被展宽。平均化学位移差约为 3.5 ppm。

因此，在 1.5T 扫描仪（工作频率为 64 MHz）中，脂肪-水频率差 (Δf) 将为

Δf = (64 MHz)(3.5 ppm) = (64 x 106 Hz)(3.5 x10−6) ≈ 220 Hz

在拉莫尔频率为 128 MHz 的 3.0T 场强下，脂肪-水频率偏移将是其两倍，约为 440 Hz。

进阶讨论（显示/隐藏）»

水分子有两个化学位移相同（δ=4.65）的等效 ¹H 原子核。然而，甘油三酯至少包含十几个具有不同 δ 值的非等效 ¹H 原子核。脂肪中最丰富的 ¹H 原子核是亚甲基（−CH2−， δ=1.3）和甲基（−CH3， δ=0.9）基团中的那些。因此，脂肪具有宽谱，并不存在单一的脂肪-水化学位移值。通常引用的是 3.3 至 3.5 ppm 范围内的加权平均值。 
甘油三酯分子中多个化学非等效的 ¹H 原子核使其具有宽谱。两个主要峰是亚甲基 (−CH2−) 和甲基 (−CH3)，其化学位移 (δ) 分别为 1.3 和 0.9。
