钆对 T1 的缩短作用

为什么钆似乎只缩短 T1？它对 T2 难道没有影响吗？  
由于其顺磁性特性，钆能促进纵向和横向磁弛豫，从而缩短其聚集组织的 T1 和 T2。然而，其主导效应（是缩短 T1 还是 T2）取决于无对比剂时组织的基线弛豫时间，以及达到的最大组织浓度。

在弛豫化学中，讨论弛豫率通常比讨论弛豫时间更方便。弛豫率以 sec−1 为单位，就是弛豫时间的倒数。化学过程的总弛豫率通常是各个组成步骤弛豫率的总和。

因此，注射对比剂后观察到的弛豫率，1/T1obs 和 1/T2obs，可以分别计算为来自天然组织和顺磁性对比剂的弛豫率之和，
1/T1obs = 1/T1t + 1/T1c             和              1/T2obs = 1/T2t + 1/T2c
其中 1/T1t 和 1/T2t 是无对比剂时组织的弛豫率，而 1/T1c 和 1/T2c 是归因于顺磁性对比剂的弛豫率。此外，1/T1c 和 1/T2c 与顺磁性对比剂的浓度 [C] 呈线性关系，根据以下关系式
1/T1c = r1 [C]                   和                    1/T2c = r2 [C]

其中 r1 和 r2 被称为顺磁性对比剂的比弛豫率，单位为 L/mmol-s。尽管这个公式严格来说仅适用于溶解在纯溶剂中且无溶质-溶质相互作用的对比剂，但临床测量已证明，对于大多数含钆细胞外对比剂在各种人体组织和肿瘤中的弛豫效应，这仍然是一个合理的近似。

大多数细胞外钆对比剂的比弛豫率（r1 和 r2）是相当的；对于低分子量钆螯合物，测量值在 3 - 5 L/mmol-s 范围内。由于 r1 和 r2 比弛豫率如此相似，人们可能会错误地得出结论，认为钆对比剂会同等程度地缩短 T1obs 和 T2obs。然而，情况并非如此，因为具有可加性且与钆浓度成正比的是弛豫率，而不是弛豫时间。此外，大多数生物组织的基线弛豫时间存在显著的自然差异；T1t 通常比 T2t 长 5 到 10 倍，这进一步加剧了钆对观察到的弛豫率的不均衡影响。
为了说明低剂量钆在表观上优先缩短 T1 的效应，我们选择正常脑组织作为基线组织，其 T1t = 800 msec (0.8 sec)，T2t = 80 msec (0.08 sec)。现在假设血脑屏障被破坏，比弛豫率为 r1 = 4 L/mmol-s 和 r2 = 4 L/mmol-sec 的钆对比剂在该组织中累积，浓度为 0.1 mmol/L。根据上述数据和公式，观察到的弛豫率将为 

1/T1obs = (1/0.8) + (4)(0.1) = 1.65 sec−1, 和
1/T2obs = (1/0.08) + (5)(0.1) = 13.0 sec−1

由此我们计算出 T1obs = 606 msec，T2obs = 77 msec。我们可以看到，钆使观察到的 T1 值减少了近 25%，而 T2 仅减少了 4%。因此，在大多数 T1 通常远长于 T2 的组织中，低浓度的细胞外钆对比剂对 T1 的缩短作用将相对大于对 T2 的缩短作用。

进阶讨论 (显示/隐藏)»

这里提供的弛豫率简单处理方法被称为“快速交换”模型，该模型假设所有对 MR 信号有贡献的质子，在时间尺度上（小于或等于介质的 T1 弛豫时间）都能平等且不受限制地接触到顺磁性中心。显然，对于存在多个具有不同交换速率（甚至正向和反向速率可能不同）的隔室的生物组织来说，情况并非如此。尽管如此，这里的简单处理方法在很大程度上解释了涉及组织和溶液中对比剂行为的基本现象。
