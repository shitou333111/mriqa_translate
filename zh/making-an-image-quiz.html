频率编码与相位编码

  1. 角频率（ω）的单位是什么？
    1. 赫兹 (Hz)
    2. 每秒周期数 (cps)
    3. 每周期弧度数
    4. 弧度每秒

    角频率（ω），也称为径向频率或圆频率，衡量单位时间内的角位移。因此其单位为度（或弧度）每秒。 链接至问答讨论

  2. 一弧度大约等于多少度？
    1. 30.0º
    2. 45.0º
    3. 57.3º
    4. 71.7º

    根据定义，2π 弧度跨越一个圆，即 360º。因此 1 弧度 = 360/2π ≈ 57.3º 链接至问答讨论

  3. 180º 脉冲有时用弧度表示为
    1. π/2-脉冲
    2. π-脉冲
    3. 3π/2-脉冲
    4. 2π-脉冲

    180º 脉冲是半圆，即 π-脉冲。 链接至问答讨论

  4. 当梯度开启和关闭时，自旋的角频率（ω）随时间（t）变化，由函数 ω(t) 表示。在时间点 A 和 B 之间，ω(t) 对 t 曲线下的面积代表
    1. 自旋在 A 和 B 之间积累的相位偏移
    2. 自旋在 A 和 B 之间的频率变化
    3. 自旋在 A 和 B 之间获得的能量
    4. 自旋在 A 和 B 之间经历的场强变化

    ω(t) 对 t 曲线下的面积代表自旋在特定时间间隔内积累的相位。从单位角度来看，频率以弧度或度每秒为单位，而时间以秒为单位，因此它们的乘积 [ω(t) x t] 具有弧度（或度）的单位。 链接至问答讨论

  5. RF线圈接收到的MR信号信息可归类为
    1. 频率调制
    2. 幅度调制
    3. 相位调制
    4. 脉冲调制

    MR信号信息（通常编码在 50-100 kHz 范围内）以幅度调制方式在 RF 载波（例如 64 MHz）上到达接收器。信号处理的第一步称为解调，它去除载波。 链接至问答讨论

  6. 在接收器 I 和 Q 通道中，MR 信号的实部和虚部在特定时间点测得分别为 3.0 和 4.0 au（任意单位）。信号的实际幅度是多少？
    1. 3.5 au
    2. 5.0 au
    3. 7.0 au
    4. 12.0 au

    幅度 (M) = [Re² + Im²]½ = [3² + 4²]½ = [25]½ = 5.0 au。 链接至问答讨论

  7. 关于MR信号的傅里叶表示，以下哪项陈述是错误的？
    1. 信号的傅里叶展开可以用三角形式或指数形式表示。
    2. 为了精确表示任何真实信号，其傅里叶表示中必须包含无限多个频率分量。
    3. 傅里叶变换将基于时间的信号转换到频域，但此变换不可逆。
    4. MR图像的傅里叶级数表示总是被截断的。

    除了（c）之外，其他都是正确的。傅里叶变换是连接时域信号 s(t) 与其频域表示 S(ω) 的数学过程。逆傅里叶变换将 S(ω) 转换回 s(t)。 链接至问答讨论

  8. 对应于 S(ω) 中均匀/矩形频带的时域信号 s(t) 被称为
    1. 双指数函数
    2. Sinc函数
    3. 高斯函数
    4. 洛伦兹函数

    Sinc 函数，sinc(t) = [sin t]/t，傅里叶变换成均匀的频带，例如用于定义常规 2D MR 成像中层面轮廓的频带。 链接至问答讨论

  9. 以下哪位科学家未因其在 NMR 方面的工作获得诺贝尔奖？
    1. Felix Bloch

    4. Peter Mansfield

  10. 以下哪项不是 MR信号的空间定位方法？
    1. 频率编码
    2. 相位编码
    3. 接收线圈的位置
    4. 幅度调制

    选项 (d) 幅度调制，指的是发射或接收 RF 脉冲的形状，但其本身不是一种空间定位方法。 链接至问答讨论

  11. 关于频率编码，以下哪项陈述是错误的？
    1. 它通常用于 2D 成像的层面内定位。
    2. 它通常用于 3D 成像的层面选择。
    3. 每个体素实际上包含一个频率范围，而不仅仅是一个单一频率。
    4. 对于大多数应用，期望使用线性频率编码梯度。

    除了 (b) 之外，其他都是正确的。频率编码通常用于 2D 成像的层面选择，但相位编码通常用于 3D 成像的层面选择。 链接至问答讨论

  12. 1.5T MR 扫描仪的典型总接收带宽是多少？
    1. 1 kHz
    2. 50 kHz
    3. 250 kHz
    4. 500 kHz

    接收带宽是一个操作者可选择的参数，由技师在扫描开始前选择。总接收 BW 的可用值范围约为 5-100 kHz，典型值为 50 kHz。 链接至问答讨论

  13. 如果视野为 25.6 cm，并使用 512 个频率编码步长，则频率编码方向上的像素宽度是多少？
    1. 0.5 mm
    2. 1.0 mm
    3. 2.0 mm
    4. 5.0 mm

    像素宽度 = FOVf ÷ 频率编码步数 = 256 mm ÷ 512 = 0.5 mm。 链接至问答讨论

  14. 如果在 5.12 ms 的时间内采样了 256 个数字化的 MR 信号复数值，驻留时间是多少？
    1. 5 μs
    2. 10 μs
    3. 20 μs
    4. 200 μs

    驻留时间是数字化采样之间的时间间隔。因此，如果在 5.12 ms 内获得了 256 个样本，则驻留时间为 5.12 ms/256 = 20 μs。 链接至问答讨论

  15. 如果数字化样本之间的间隔（驻留时间）为 25 μs，则总接收带宽是多少？
    1. 2,500 Hz
    2. 25,000 Hz
    3. 40,000 Hz
    4. 50,000 Hz

    总接收 BW 与 MR 信号的数字化速率相同。以方程形式表示，BW = 1/td = 1/(25 μs) = 40,000 Hz。 链接至问答讨论

  16. 在 1.5T 的临床 MR 成像中，设置接收器为“窄带宽”的典型频率范围是多少？
    1. 1 – 5 kHz
    2. 5 – 20 kHz
    3. 50 – 52 kHz
    4. 100 – 101 kHz

    窄接收带宽的典型范围是选项 (b)，5 – 20 kHz。 链接至问答讨论

  17. 关于窄带宽，以下哪项陈述是错误的？
    1. 窄带宽意味着采样时间缩短。
    2. 窄带宽可提高信噪比。
    3. 窄带宽会加剧化学位移伪影。
    4. 窄带宽会加剧磁敏感伪影。

    选项 (a) 是错误的。带宽与采样时间成反比。“窄带宽”的同义词是“延长采样时间”。 链接至问答讨论

  18. 关于发射带宽，以下哪项陈述是正确的？
    1. 一旦指定接收带宽，发射带宽就会自动确定。
    2. 发射带宽本身并非由技师在指定扫描参数时直接调整。
    3. 发射 BW 的典型值为 50-100 kHz。
    4. 大多数用于 2D 层面选择的现代 RF 脉冲呈 sinc 形。

    发射带宽通常不可由用户调节，因此选项 (b) 正确。典型值为 1000-2000 Hz，且与接收带宽无关。简单的 sinc 脉冲不再用于层面选择，已被使用 SLR 算法的更复杂设计所取代。 链接至问答讨论

  19. 串扰可以通过除以下哪项之外的所有方法来减少：
    1. 增加层面间隙
    2. 增加层面选择梯度的强度
    3. 执行层面交叉采集
    4. 改善 RF 层面轮廓

    增加层面选择梯度的强度只会产生更薄的层面，而不会改善串扰。 链接至问答讨论

  20. 适用于单通道 RF 线圈的最早商用同步层面激发版本（由 GE 在 1990 年代推出）被称为
    1. Phase Offset MultiPlanar (POMP)
    2. MultiBand (MB)
    3. Simultaneous Multi-Slice (SMS)
    4. HyperBand (HB)

    现代 MB/SMS 方法的历史前身是 POMP（Phase Offset MultiPlanar）成像。POMP 技术使用复合 RF 脉冲同时激发两个层面，每个层面仅在总视野（FOV）的一半上进行相位编码。相位交替允许两个层面被单独重建而不重叠。与 SMS 不同，POMP 不利用并行成像技术，因此适用于单通道 RF 线圈。 链接至问答讨论

  21. 关于 MultiBand/Simultaneous Multi-Slice 成像，以下哪项陈述是错误的？
    1. 需要使用并行成像阵列。
    2. 加速因子通常为 2-4。
    3. 层面必须间隔较远（2-3 cm）。
    4. 该技术在信噪比方面会带来显著的损失。

    MB 利用线圈编码以及梯度或 RF 编码来解析沿层面选择（z）轴的数据。由于现代线圈阵列在 z 方向上通常只有少数几个线圈单元，因此沿该轴的线圈灵敏度差异相当差。因此，同时激发的层面必须间隔较远（通常至少 25−30 mm）。与标准的并行成像加速技术不同，MB/SMS 加速在信噪比方面几乎没有损失。（选项d是错误的）。这是因为在常规并行成像加速方法中缩短的回波链长度、相位编码步数或 k 空间样本数量并未减少。 链接至问答讨论

  22. 使用 192 个相位编码步长和 512 个频率方向样本采集的 25 x 25 cm FOV 图像的近似平面内像素尺寸是多少？
    1. 1.3 x 0.5 mm
    2. 2.6 x 0.5 mm
    3. 1.3 x 5.0 mm
    4. 2.6 x 5.0 mm

    相位编码方向的像素宽度为 250 mm ÷ 192 ≈ 1.3 mm，而频率编码方向的像素宽度为 250 ÷ 512 ≈ 0.5 mm。 链接至问答讨论

  23. 关于 2D 傅里叶变换成像，哪项陈述是错误的？
    1. 独特回波信号的数量等于相位编码步数。
    2. 连续回波之间的相位偏移是 180º 的倍数。
    3. 对于低幅度相位编码，MR 信号强，但提供的空间细节信息很少。
    4. 对于高幅度相位编码，MR 信号弱，且提供的图像对比度信息很少。

    选项 (b) 是错误的。行之间的相位偏移不是简单的 180º 倍数，而是根据相位编码步长的大小在回波之间变化。采集的回波信号数量等于相位编码步数，通常为 192 或 256。对于低阶相位编码，MR 信号强。频率投影近似于物体的大致形状，但缺乏边缘清晰度。高阶相位编码步长的 MR 信号较小，但提供更多关于空间细节的信息，例如边缘的位置。 链接至问答讨论

  24. 为什么相位编码方向通常选择沿着最短的解剖维度？
    1. 这可以减少卷折伪影。
    2. 这可以减少流动相关伪影。
    3. 这可以减少由患者大幅运动引起的伪影。
    4. 这是并行成像的要求。

    当所成像身体部位的大小超过相位编码方向定义的视野（FOV）时，会发生卷折（也称为混叠）。这导致 FOV 外的解剖结构折叠到图像的主要部分上。为避免卷折，通常选择最短的解剖维度作为相位编码方向。通过改变相位编码方向可以移动流动相关和其他运动伪影，但最佳方向取决于这些伪影的位置和类型，其不一定是最短维度。对于并行成像，可能对允许的相位编码方向有一些限制，但将相位编码沿最短轴设置并非必需。 链接至问答讨论

扫描与质量控制

  1. 设置 MR 扫描的以下步骤中，正确的顺序是？
    1. 设置定位 → 执行预扫描 → 采集定位像 → 放置层面
    2. 采集定位像 → 放置层面 → 设置定位 → 执行预扫描
    3. 设置定位 → 采集定位像 → 放置层面 → 执行预扫描
    4. 执行预扫描 → 采集定位像 → 设置定位 → 放置层面

    选项 (c) 显示了正确的顺序。 链接至问答讨论

  2. 下面列表中的哪个过程不是自动预扫描的一部分？
    1. 比吸收率（SAR）估算
    2. 快速匀场
    3. 中心频率调整
    4. 发射增益调整
    5. 接收增益调整

    比吸收率（SAR）由扫描仪根据患者体重、所选脉冲序列和脉冲序列参数进行估算。这发生在自动预扫描之前，允许操作员调整参数以使 SAR 保持在既定限值内。自动预扫描本质上是实际扫描开始前的最后一步。 链接至问答讨论

  3. 在以下哪种情况下，几乎总是需要手动匀场（除了预扫描快速匀场之外）？
    1. 弥散加权成像
    2. MR 波谱
    3. 快速自旋回波成像
    4. Dixon 水脂成像

    对于波谱，必须进行非常详细的高阶匀场。尽管波谱匀场可以自动化，但设置总是需要 MR 技师进行审查并经常调整。 链接至问答讨论

  4. 在以下哪种情况下，中心频率调整最为关键？
    1. 使用空间饱和脉冲时
    2. 使用流动饱和脉冲时
    3. 使用动脉自旋标记脉冲时
    4. 使用脂肪饱和脉冲时

    中心频率可以设置为水质子、脂肪质子或两者的平均值。当使用频谱脂肪饱和脉冲以确保脂肪峰被充分抑制时，精确的中心频率设置尤为重要。 链接至问答讨论

  5. 如果在预扫描期间接收器衰减值设置过低，会出现什么伪影？
    1. 数据截断
    2. 卷折
    3. 重影
    4. 弥漫性噪声

    如果衰减值设置过低（即接收增益设置过高），则信号将使模数转换器过载，并发生数据截断。由于 MR 信号的最大峰值通常是 k 空间中心的低阶相位编码步长，数据截断会干扰图像对比度。结果图像会呈现“诡异”的外观。 链接至问答讨论

  6. 预扫描期间虚拟循环的目的是什么？
    1. 调整 RF 电压以实现完美的 90º 脉冲
    2. 允许发射和接收电路预热
    3. 使线圈阻抗与患者的阻抗达到最佳匹配
    4. 允许建立稳态的纵向和横向磁化强度

    一旦 RF 激发开始，纵向和横向磁化强度需要时间达到其稳态值。在预扫描期间，扫描仪会播放几个周期的序列而不记录信号。这些称为虚拟循环、禁用或丢弃的采集（DDA）。虚拟循环的确切数量取决于 TR 和所选序列。仅在系统接近平衡时才开始数据采集。 链接至问答讨论

  7. 要在美国销售 MR 扫描仪，公司必须获得美国食品药品监督管理局的何种上市前许可？
    1. 401(k)
    2. 510(k)
    3. 1099
    4. 457(b)

    在美国，食品药品监督管理局（FDA）拥有监管 MRI 设备销售和使用的法定权力。MRI 扫描仪被视为 II 类设备，意味着它们可能对人体造成伤害，需要在上市前获得 510(k) 上市前许可。FDA 已发布指导文件，其中包含获得此上市前批准的非强制性（但强烈建议的）标准——包括与硬件、软件、性能、场地规划和安全性相关的标准。其他数字都是与 MRI 无关的 IRS 表格和投资账户。 链接至问答讨论

  8. 以下哪个组织不在美国提供 MRI 认证？
    1. 美国放射学会 (ACR)
    2. 美国医疗保险和医疗补助服务中心 (CMS)
    3. RadSite
    4. 联合委员会 (TJC)

  9. 以下哪项质量控制测量必须至少每周在 MR 扫描仪上执行一次？
    1. 中心频率
    2. 磁场均匀性
    3. 层面厚度准确性
    4. 显示器分辨率

    每日 QC 活动包括目视检查所有扫描仪硬件、安全和通信设备的功能，以及图像质量的总体评估，包括识别伪影。至少每周一次，将专用的 MR 体模放入扫描仪中，并进行各种测量和记录。此类测量包括定位准确性（床位置）、中心频率、图像均匀性、发射增益或衰减、几何失真、空间分辨率、伪影评估和信噪比。其他更复杂的测试应由医学物理师每半年或每年进行一次。 链接至问答讨论

  10. 几何误差最常见的原因是
    1. MR 体模变形
    2. RF 线圈阻抗异常
    3. 一个或多个成像梯度校准错误
    4. 中心频率漂移

  11. 测量体模中信噪比的两种最常用方法的详细程序和标准来自哪个组织？

    2. 国际电工委员会 (IEC)
    3. 国际非电离辐射防护委员会 (ICNIRP)
    4. 美国材料与试验协会 (ASTM International)

  12. 当复数值的 MR 信号数据被转换为幅度图像时，噪声统计最好用以下哪种分布描述？
    1. 高斯（正态）分布
    2. 泊松分布
    3. 伽马分布
    4. 莱斯分布

    由于感兴趣区（ROI）测量通常在幅度重建图像上进行，因此必须对统计数据进行一些校正。回想一下，“原始” MR 数据是具有实部和虚部的复数。当转换为仅幅度图像时，对应于噪声的像素值不再是高斯分布，而是偏斜成所谓的莱斯分布。 链接至问答讨论

  13. MR 体模中由紧密间隔的线或孔组成的部分，其中填充了与背景信号强度差异很大的材料，用于测量
    1. 低对比物体可探测性
    2. 几何失真
    3. 高对比空间分辨率
    4. 图像均匀性

    MR 体模的高对比部分包含紧密间隔的线、边缘或孔，其中填充了与背景信号强度差异很大的材料。低对比分辨率指的是识别体模中与背景仅略有弛豫时间差异的小孔的能力。 链接至问答讨论

  14. MR 体模中由以已知角度定向的三角形斜坡或楔形组成的部分用于测量
    1. 层面厚度
    2. 层面位置
    3. 线性准确性
    4. 低对比分辨率

    估算层面厚度最常用的方法是使用包含三角形斜坡或楔形的体模，其表面与层面平面成已知角度（θ）。当层面穿过斜坡时，会产生一个拉伸的“阴影”图像，可以估算其半高全宽（FWHM）。 链接至问答讨论
