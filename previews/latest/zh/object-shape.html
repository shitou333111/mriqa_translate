形状效应

我不理解物体的形状如何影响其磁化。  
有这种困惑的不止你一人。物体形状的影响远比大多数人想象的复杂得多！
抗磁性、顺磁性或弱铁磁性材料
由磁化率较小的材料（即 χ < 1）构成的物体，包括大多数外科手术不锈钢以及锡、铝、铜或铅等有色金属，在外磁场中仅发生极微小的位移，不会构成抛射物危险。其磁化过程中的任何形状效应基本无关紧要，可以忽略不计。
中等到强铁磁性材料
由具有较大磁化率（χ >> 1）的铁、镍和/或铬制成的物体，当置于外磁场（Bo）中时，会逐渐被磁化。随着外磁场增强，这种内部极化/磁化（M）也随之增强，因为电子自旋开始排列，磁畴边界发生重塑。此过程持续进行，直到M 达到材料的磁饱和点，此后无法再发生进一步的磁化。（请注意，磁极化 M源于微观电流中的电子自旋，与别处讨论的核磁化M无关）。
铁磁性材料置于外磁场（Bo）中时，会产生强烈的内部磁化（M），自身表现得像小磁体。其暴露于外磁场的边缘变得像北极和南极，从而产生一个内部的退磁场（D），该磁场与M方向相反。
`作为此过程的结果，一个由中等到强铁磁性材料构成的物体本质上变成了它自己的“小磁体”，其两端可被视为北极和南极“极”。  这些“极”代表了在物体两端归因的磁荷密度积累，那里存在突然的磁场不连续性（即，M 从物体内部的正值立即变为外部的零）。由此产生了一个内部的退磁场（D） ，它从物体的北极指向南极，方向与 M 和 Bo 两者都相反。  假设物体是均匀的且对小磁场变化呈线性响应，则通常认为退磁场（D）在每个方向上与 M 成正比，比例常数在 0 到 1 之间，称为 退磁因子（N）。  具体而言， D = −NM。  请注意，N 仅取决于几何形状，而非材料。

这一结果的一个重要推论是，对于铁磁性材料，达到临界磁饱和 (Bsat) 所需的外场强度 (Bext) 比预期的要小得多。  具体来说，
例如，考虑一个饱和磁感应强度 Bsat = 2.0 T 的硅钢球体靠近一台 1.5 T 的 MR 扫描仪。球体在每个方向上的退磁因子为 ⅓，因此当外磁场 (Bext) 达到 N ​• Bsat = (⅓)(2.0) = 0.67 T 时，球体将完全磁化。  令人惊讶的是，这个球体在 1.5T 扫描仪的边缘场中就已经达到了其完全的（2.0T）饱和磁化强度！实际上，这并不应该太令人惊讶，因为对铁磁性材料的常见通俗描述是，它们倾向于“集中”外部磁场的磁力线，从而增加其内部的磁通密度。
通常，N 和 χ 不是单一数值，而是二阶张量（对应于不同方向的数值阵列）。对于像椭球体这样的对称三维形状，可以选择与物体主轴对齐的坐标系，从而得到三个 退磁因子， Nx， Ny，和 Nz，分别对应每个基本方向。这些 都是介于 0 和 1 之间的无量纲数，并具有附加属性 Nx + Ny + Nz = 1。

请注意，具有最高 N 值的物体是那些在 Bo 方向上具有较大相对表面积的物体。这类物体的典型代表是垂直于 Bo 的薄垂直板，其 N ≈ 1.00（可能的最大值）。相反，当薄板旋转 90º 时，N ≈ 0.00（可能的最小值）。图中所示的各种椭球体和圆柱体也展示了同样的原理。
那么，为什么会发生这种情况呢？原因在于，对于最大面积面向 Bo 的较薄物体，其表面的虚拟磁南北极距离很近；因此去磁场 (D) 非常强。相反，当虚拟磁极相距较远时（如平行于 Bo 的浅蓝色圆柱体），两端的磁极相距甚远。在这种情况下，内部的 N-S 极对相互抵消，仅在两端留下相距较远的磁荷密度。因此，圆柱体的这种取向会导致极弱的去磁场 (D) 以及相应较小的去磁因子 (Nz) 值。

高级讨论（显示/隐藏）»

根据定义，磁化强度 M 通过被称为磁化率的无量纲量 (χ) 与内部磁场 (H) 相关联：

M = χ H

在抗磁性和弱顺磁性材料中，H = Ho，其中 Ho 是外部磁场。

在铁磁性材料中，形状起着至关重要的作用，因为内部磁场 H 会被反向的去磁场 (D) 削弱。对于均匀线性材料，去磁场可表示为 D = − NM，其中 N 是特定方向的去磁因子。

因此，在铁磁性材料中：

H = Ho − D = Ho − NM

但由于 H = M / χ，我们可以写成：

M / χ = Ho − NM

或

Ho = M / χ + NM = M (1/χ + N) = M (1 + Nχ) / χ

因此，表观磁化率 χapp 为：

χapp = M / Ho = χ / (1 + Nχ)

分子分母同时除以 χ，我们得到：

χapp = 1 / (1/χ + N)

因此，当 χ → ∞ 时，1/χ → 0，剩下来的结果为：

χapp = 1 / N

因此，对于较大的 χ 值（如通常在铁磁性材料中发现的那样），对 χ 的依赖性消失了，表观磁化率仅取决于去磁因子的倒数 (1/N)。

导致磁饱和 (Msat) 所需的外部磁场强度 (Hext) 可以推导为：

Hext = Msat / χapp = Msat / (1/N) = NMsat

将该方程两边乘以真空磁导率 (μo = 4π x 10−7 T m A−1) 转换为国际单位制 (SI)，结合 Bext = μoHext 和 Bsat = μoMsat，我们得到：

Bext = N Bsat
