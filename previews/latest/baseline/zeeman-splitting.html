塞曼分裂

在之前的问答中，我们介绍了1H原子核的两种自旋态概念，通常表示为 |+½⟩ 和 |-½⟩ 或  |↑⟩ 和 |↓⟩。我们还展示了如何像斯特恩和盖拉赫在1922年所做的那样，通过使自旋穿过磁场（Bo）来分离和测量这些态。由于自旋倾向于排列成与外部磁场大致同向或反向，这两种自旋态通常被称为“自旋向上” 或 “平行” 以及 “自旋向下” 或 “反平行”。  
警告：尽管我们有时可以测量到对应于自旋系统“纯”（本征）态的离散属性（如能级或角动量），但推断自旋在物理上仅存在于这些状态之一是不正确的，因为大多数自旋存在于混合态中。 
​磁势能：经典分析
将原子核自旋视为嵌入大外部（N-S）磁场中的翻转条形磁铁的塞曼分裂直观模型。
在斯特恩-盖拉赫装置中，自旋向上和自旋向下的粒子通过非均匀磁场对其磁矩施加的力而在空间上分离。这种力源于塞曼相互作用，该相互作用在磁场中产生自旋态之间的能量分裂。相关的能量分裂被称为（核）塞曼效应，以荷兰物理学家彼得·塞曼的名字命名，他于1896年观察到了钠焰光谱线随磁场变化的分裂现象。

上面的动画图像揭示了为什么自旋向上和自旋向下态在能量上被分开。在自旋向下态中，小磁铁的红色磁极朝向大磁铁的红色磁极。要实现这种排列，需要输入能量以克服同极磁铁的自然排斥力。系统处于不太稳定的、能量较高的构型。相反，在自旋向上态中，小磁铁和大磁铁的相反磁极相邻并相互吸引，导致系统磁势能较低。  ​
这些概念可以使用经典电磁理论进行更定量的描述，我们将原子核自旋建模为磁偶极子 (µ)。 ​µ 相对于 Bo 的扭矩和能量关系已在之前的问答中详细阐述。总结如下： 
  • 偶极子（µ) 所具有的磁势能 (E)由矢量点积给出：    E = − µ • Bo  =  −µ Bo cos (θ)，其中θ 是µ 和 Bo 之间的夹角。
  • 当µ 与 Bo 平行时，θ = 0​° 且 cos (θ) = 1， 因此系统能量处于最低水平，为负值 (= −µBo)。  
  • 当µ 和 Bo ​ 反平行时，​​θ = 180​° 且 cos (θ) = −1。 两个负号抵消，系统能量现在为正 (+ µBo)，处于可能达到的最高水平。
  • 因此，µ 相对于Bo 的平行和反平行构型之间的能量差 (ΔE) 为 ΔE  =  µBo − (−µBo)  =  +2 µBo 
  • 这一结果与塞曼的观察结果一致，即分裂程度与磁场强度（Bo）成正比。
  • 对于磁矩为 μ = γℏ/2 的自旋-½原子核，这变为 ΔE  = 2µBo = γℏBo​
延伸至量子力学 
将这个经典偶极子类比延伸到量子领域，需要假设自旋的磁矩（µ）与其角动量 I直接相关。   这种关系明确定义为
​µ = γ I 
其中 γ 是一个常数，称为 旋磁比。  旋磁比由实验确定，对于每个旋转粒子都不同。例如，1H原子核的 γ 值为 42.58 MHz/T，而 ​ 3He 则为 −32.43 MHz/T。自旋与角动量之间的线性关系源自运动电荷的经典电磁物理学，并在量子力学中得以保留，因为磁偶极矩和角动量具有相同的旋转变换特性。

在量子力学中，算符 是用于表示自旋角动量（I）等物理可观测量的数学实体。其中最重要的之一是哈密顿算符 (ℋ) 。系统的哈密顿量通过本征值方程确定其量子化能级。对于简单系统，例如在沿 z 轴的静态磁场 Bo 中的单个自旋-½ 原子核，哈密顿量为 Hz = −γℏBoIz。求解相应的本征值问题得到两个可观测的能级， [−½​γℏBo] 和 [+½​γℏBo]，其能级差定义了塞曼分裂：

ΔΕ = γ ℏ​ Bo

这里我们使用的旋磁比 (γ) 形式，其单位为角速度除以磁场强度 (rad/T)。该方程引入了另一个参数 (ℏ)，称为约化普朗克常数，这个数字在量子力学计算中频繁出现，反映了亚原子世界的颗粒性。其测量值约为 1.05457 x 10-34 焦耳·秒。与每个原子核都不同的旋磁比 (γ) 不同，约化普朗克常数 (ℏ) 对所有原子核都是通用的。

约化普朗克常数 (ℏ) 与更为人熟知的（非约化）普朗克常数 (h) 通过方程 ℏ = h/(2π) 相关联。因子 2π 将线性频率（每秒周数，Hz）转换为角频率（每秒弧度，rad/s），后者更广泛用于描述量子力学中的旋转和波动现象。

简而言之，上述塞曼分裂关系表明，沿磁场方向，自旋向上 |+½⟩ 和自旋向下 |−½⟩ 态之间的能隙 (ΔE) 与磁场强度 (Bo) 成正比。这对核磁共振具有重要意义，因为更高的磁场会导致能级间距增大，从而产生更强的信号，并能够分离不同化学基团之间的谱线。结合（下文讨论的）普朗克定律，塞曼关系也直接引出了核磁共振的基本概念之一，即著名的拉莫尔方程。

德国物理学家马克斯·普朗克是量子力学的奠基人之一。在20世纪初，他提出原子系统的能量交换只能以离散的“包”或量子的形式发生。这一概念是作为解释黑体辐射的实验性假设而出现的，并非源自更深刻的经典物理定律。它随后被多个独立实验证实，现在被视为量子理论结构中的一个基本假设。

普朗克发现能量交换 (ΔE) 是通过特定频率的量子介导的，该频率用希腊字母“nu”ν 或 fo = ωo/2π 表示。这种关系体现在著名的普朗克方程中（现在已在高中物理课程中讲授，是现代量子力学的基石）：

ΔE  =  h ​ν  =  h fo  =  h (ωo/2π)  =  (h/2π) ωo  =  ℏ ωo
普朗克方程和塞曼方程可以结合，产生一个有趣且重要的结果，对于熟悉磁共振成像的人来说会很眼熟：
ΔE  =  ℏ ωo  =  γ ℏ Bo
或
ωo   =  γ Bo
这就是著名的拉莫尔方程，它表明核磁共振共振频率 (ωo) 仅仅是旋磁比 (γ) 乘以磁场强度 (Bo)。普朗克常数 (h) 从解中消失这一事实意味着，使用经典物理学对核磁共振进行非量子解释既是可能的，也是合理的。 
几个包含塞曼名字的技术术语已进入现代核磁共振术语库（尽管并非由他具体描述），包括 塞曼分裂、塞曼能级、塞曼能量 和 塞曼库。  这些术语在下文的进阶讨论中为有兴趣的学生进行了定义和解释。

基于他在1896年的实验，荷兰物理学家彼得·塞曼得出结论，钠焰的光谱线分裂与外加磁场强度成正比。这种比例关系是现在称为塞曼效应的核心实验结论。尽管他的发现后来与自旋-½系统和核磁共振联系起来，但塞曼用经典术语解释了他的观察结果——即原子中的电子表现为微小的振荡电荷，其运动受磁场影响。需要记住的是，1896年量子力学尚不存在，电子和核自旋的概念也未知，甚至原子能级也未完全被接受。现代将塞曼效应视为源于角动量量子化态的观点是在几十年后随着量子力学的成熟而发展起来的。

除了塞曼效应分裂，彼得·塞曼的名字还与现代核磁共振理论中的许多相关概念联系在一起：

塞曼能级 是自旋或原子在塞曼分裂产生的磁场中的一个离散量子化能态。因此，塞曼能级是一个允许的单一能级，对应于沿磁场方向角动量的特定允许投影。在核磁共振中，自旋占据塞曼能级，射频辐射诱导它们之间的跃迁。

塞曼能量 指的是磁矩与外磁场之间的相互作用能。每个塞曼能级都具有塞曼能量。由于主磁场通常定义为沿 z 轴，适用的哈密顿量（与能量测量相关的量子力学算符）为 Hz = − μ • Bo = − γℏBoIz（对于沿 z 轴的自旋-½ 原子核）。每个允许的本征态都具有塞曼能量 Em = − γℏBo，其中 m = ±½。

塞曼分裂 是由外加磁场产生的塞曼能级之间的能隙，通常表示为 ΔE。对于自旋-½ 原子核， ΔE = E+½ − E−½ = γℏBo。当没有外加磁场时，塞曼分裂为零。

塞曼库 是包含在与静态磁场对齐的自旋布居中的能量储存。换句话说，它是与塞曼哈密顿量 (Hz) 相关的热力学子系统，与初始自旋布居数差和净纵向磁化强度 (Mz) 相关。

射频脉冲主要重新分配自旋系统内部的能量。一个理想的短脉冲不会改变塞曼布居数，但会将塞曼有序转换为横向相干性。塞曼库被认为被“排空”到横向相干性中，而不是进入周围的晶格。随后的 T1 弛豫机制（将在后续问答中讨论）通过重新建立塞曼布居数差并恢复 Mz，使系统恢复到热平衡状态。因此，T1 是衡量塞曼库被“重新填充”并与晶格重新平衡的速度的指标。

学生有时会发现以下类比很有帮助：将塞曼库视为储存磁对齐的电池。充电时，Mz 很大；放电时，Mz 很小。射频脉冲消耗电池，将其转换为信号，而 T1 效应则用来自晶格的新鲜自旋为电池充电。


玻尔兹曼分布与塞曼有序的区别

玻尔兹曼分布 描述了平衡状态下单个自旋在塞曼能级中的分布方式。它是关于单个自旋的概率规则。它描述了“测量为自旋向上 (N+/N) 与自旋向下 (N−/N) 的自旋比例是多少？”

在大多数核磁共振场景中（其中塞曼能量沿 z 轴纵向定义），塞曼有序 指的是这种由玻尔兹曼统计决定的相同布居数差。如果布居数不相等 (N+ ≠ N−)，系统就具有塞曼有序。使用“有序”一词是因为这种分布比可能的状态更不随机（具有更小的熵）。如果布居数相等 (N+ = N−)，这将是一个最大熵状态，没有塞曼有序。缺乏塞曼有序的系统被称为具有“空”的塞曼库。因此，塞曼有序是自旋系统内部的一种热力学结构。能量施加的统计有序的非核磁共振例子包括铁磁体中的磁化、引力场中的气压差以及电场的极化。
