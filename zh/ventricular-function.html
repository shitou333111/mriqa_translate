心室功能

如何利用电影 MRI 计算射血分数？  
除了在电影成像研究中目视检查心壁运动和增厚情况外，大多数中心也常规进行整体心室功能的定量分析。 将数据下载到独立工作站后，技师会追踪整个心动周期中心肌的内缘。目前已有半自动软件辅助此过程，但必须始终由人工检查其准确性。心肌外缘也可能被追踪。 输入患者的体重和身高，以便将计算出的功能参数归一化到标准体表面积 (BSA) 1.8 m²。 
短轴电影成像上的自动心室功能轮廓（由西门子提供）
典型的左心室 (LV) 功能研究始于获取单层、TrueFISP/平衡稳态自由进动 (balanced SSFP) 电影图像，包括两腔心、三腔心和四腔心切面。对于大多数无显著心律失常且能配合的患者，这些图像均可使用回顾性门控在一次屏气内完成采集。
接下来是核心的左心室功能序列——在垂直于左心室长轴的短轴方向上，从二尖瓣到心尖，获取覆盖整个心动周期的 8-12 层电影图像。这些图像通过多次屏气配合回顾性门控获得。应旋转视野 (FOV) 以避免卷褶伪影。 
总左心室 (LV) 容积的计算方法是将每个短轴切面的横截面积乘以层厚加上层间距（辛普森法则）。通过此过程生成的全局心室功能参数包括：
  • 舒张末期容积 (EDV) = 舒张期（松弛期）心室腔的最大尺寸
  • 收缩末期容积 (ESV) = 收缩期（收缩期）心室腔的最小尺寸
  • 每搏输出量 (SV) = EDV−ESV = 一次心跳排出的血液体积
  • 射血分数 (EF) = SV/EDV = 一次心跳排出的血液百分比
  • 心输出量 (CO) = SV x 心率 = 一分钟内排出的血液总体积
  • 心脏指数 (CI) = CO/BSA = 归一化到体表面积的心输出量
大多数 MRI 设备供应商和独立公司都提供半自动心室功能分析，如图中来自西门子 Argus 系统的示例。
通常常规只计算左心室容积，但该方法也可应用于右心室 (RV)。需要测量右心室的临床情况包括评估先天性心脏病、肺动脉高压和致心律失常性右心室心肌病 (ARVC) 患者。由于右心室位于胸骨正下方，超声心动图可能难以评估其功能。
典型的右心室功能研究始于获取单层、回顾性门控的垂直长轴和右心室流出道电影图像。核心测量序列是一组覆盖整个右心室和流出道的 8-12 幅横断面图像。
心脏 MRI 目前通常被认为是临床测量左心室和右心室功能的“金标准”，优于超声心动图、核素显像和导管心室造影。多项人体和动物研究证实，通过 MRI 获得的功能参数具有高重复性、高准确性以及较低的观察者内和观察者间变异性。

进阶讨论（显示/隐藏）»

正如预期的那样，在视觉上定义心室壁的边界和组成部分确实存在一些变异性。第一个挑战是决定在多大程度上将乳头肌和小梁结构纳入心室容积测量。通常这些结构被包含在内，因为围绕整个心腔内血池（其中包含它们）绘制一个椭圆形感兴趣区域更容易。请注意，在收缩期可能难以将乳头肌与心肌壁分开，因为它们看起来都融合在一起。

第二个技术挑战是决定在计算中使用多少层图像。这在心脏基底部可能尤其成问题，由于心脏运动，收缩期时左心房的部分可能出现在短轴切面中，而舒张期时则只有左心室。通常，测量舒张末期容积比收缩期容积需要多一到两层图像。

左心室质量也可以通过绘制心内膜和心外膜轮廓来计算（如上图电影图像中的红色和绿色线条所示）。最常使用舒张末期获得的图像进行此测量。由于心肌密度约为 1.05 g/mL，左心室质量可计算为左心室心肌体积 x 1.05。

区域心肌增厚百分比也可以从这些相同数据中计算出来，并显示在极坐标图（“靶心图”）上，类似于灌注成像所使用的图。
