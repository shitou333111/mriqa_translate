化学位移

什么是化学位移，它如何影响进动频率？  
特定原子核的共振频率并非由外部施加的磁场强度 (Bo) 决定，而是由原子核在原子水平上感受到的局部磁场 (Bloc) 决定。因此，患者体内所有的1H 原子核并非以完全相同的频率共振；共振频率的差异（称为 化学位移）取决于它们所处分子的化学性质。对于1H，这些位移相对较小（在 1.5T 下约为几百 Hz），但是可检测的。基于化学位移进行化合物表征是磁共振波谱 (MRS)的主要课题。
当置于外部磁场 (Bo) 中时，原子核周围电子云的反应性响应会产生一个感应场 (Bind)，该感应场通常与外加磁场方向相反。这被称为“屏蔽”。原子核的共振频率取决于 Bloc = Bo − Bind。
化学位移主要来源于循环电子产生的抗磁屏蔽。这种屏蔽改变了原子核处的局部磁场。理解这一现象最简单的方法是想象原子核周围的循环电子产生一个“感应”磁场 (Bind)，该磁场与 Bo 方向相反。因此，电子云充当了“屏蔽”，保护原子核免受 Bo 全部作用力的影响。屏蔽因子通常用符号 σ 表示，其中 Bind = σBo。因此，原子核感受到的局部有效磁场为 Bloc，其中
Bloc = Bo − Bind = Bo (1 − σ)
较早的文献有时将 σ 称为“化学位移”。这种术语造成了一些混淆，因为 Bind 和 Bloc 并非直接测量值。在实践中，化学家测量的是待测样品与外部参考标准之间的 NMR 频率差。2001年，国际纯粹与应用物理与化学联合会 (IUPAC) 建议放弃使用 σ ，转而采用 δ-标度，并将化学位移 (δ) 定义为
δ = (fsamp − fref ) / fref
其中 fsamp 是待测核种的 NMR 频率，fref 是参考化合物的 NMR 频率。

频率差 (fsamp − fref ) 通常在几百到几千 Hz 的量级，而参考频率 (fref ) 则以 MHz 为单位测量。因此，化学位移 (δ) 是一个很小的数值，以百万分之一 (ppm)为单位表示。后缀 ppm 可与 x 10 −6 互换使用，就如同符号 % 可与 x 0.01 或 x 10−2 互换一样。

化学位移 (δ) 是一个无量纲常数，对于每种类型的原子核 (1H， 31P 等) 和化学构型 (—CH3， —OH， --PO4 等) 都是特定的。化学位移值表很容易获取（现已嵌入计算机/智能手机应用程序），这使得化学家能够在 MRS 实验中快速识别检测到的原子核。

关于化学位移的补充材料

感应场 (Bind) 的起源

在关于化学位移的主要问答中，我主要遵循了“传统”解释，即化学位移源于响应外加磁场 (Bo) 的应用而产生的循环电子电流，这让人联想到楞次定律。但这并不完全正确。事实上，可以证明（通过统计力学的玻尔-范列文定理），这种电流在经典电磁学定律下无法产生如前所述的 Bint 场，也无法产生原子级别的顺磁性或抗磁性。

深层的解释只能在神秘的量子力学世界中理解，但化学位移确实是由电子循环产生的，这一点是真实的。抗磁项（来自场诱导的基态电子循环）和顺磁项（来自激发态电子）共同作用，产生一个难以预测但通常与外加磁场方向相反的综合效应。
化学位移张量
虽然我们将化学位移 (σ) 表示为一个单一数值，但实际上它是一个张量，即一个 3x3 的实数数组，对应于实验室各方向（-xy， -yz, -zz 等）上化学位移的分量。该张量考虑了一个事实，即感应场 (Bind) 通常与外加磁场 (Bo) 不平行。其矩阵形式的关系由点积给出： 
化学位移张量
Bind = σ • Bo
矩阵分量 σyz 表示当外部磁场沿 z 方向施加时，化学位移在 y 方向的分量，其他组合依此类推。通常可以假设化学位移张量关于对角线对称，这意味着 σyz = σzy 等。
对于每个核位置，可以找到一组正交（垂直）的方向，用大写字母 X， Y 和 Z 表示，使得感应场 (Bind) 可以与外加磁场 (Bo) 平行，并且非对角矩阵元素消失。（这在数学上称为“对角化矩阵”）。X， Y 和 Z 被称为化学位移张量的主轴，并且通常与分子结构特征（如化学键的方向）相关。使用这些主轴，可以定义一个密切相关的主值化学位移张量 (δ)，其元素表示为 δXX， δYY 和 δZZ， 并且所有非对角元素如 δXY = 0。 该矩阵三个对角元素的平均值称为各向同性化学位移 (δiso)，由以下方程给出：
δiso  = ⅓ (δXX + δYY + δZZ)
各向同性化学位移是决定 MRS 谱中（至少对于各向同性液体而言）峰相对位置的值。因此，当表中给出的单个数字作为“化学位移”时，例如 δ = 3.2 ppm，那就是所指的值。
化学位移各向异性 (CSA)

每当三个对角元素 (δXX， δYY， 和 δZZ) 相等时，化学位移张量称为各向同性。如果这些值中至少有一个不同，则该张量被称为各向异性。化学位移各向异性 (CSA) 是化学位移与其平均值（各向同性值）的最大偏差的量度。因此，如果最大的化学位移偏差出现在 Z 主轴上（使得 δZZ > δXX 或 δYY），则 CSA 表示为 δCSA 并由下式给出：
δCSA  = δZZ − δiso
CSA 效应在大分子和组织的成像中尤为重要，因为这些结构中存在结构不对称性以及相对较慢的分子运动。CSA 弛豫与场强的平方 (Bo²) 成正比，因此在更高场强下更为重要。它在磷 (31P) 原子核的成像和波谱中也很重要，因为磷原子核相对于其共振频率具有非常大的化学位移。
