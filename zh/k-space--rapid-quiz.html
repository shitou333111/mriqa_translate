k空间

  1. k空间中的一个点对应的是
    1. 整个成像对象中某特定空间频率的量
    2. 成像对象中某特定点处某特定空间频率的量
    3. 成像对象中某特定点所产生信号的幅度和相位
    4. 整个成像对象所产生信号的自旋密度

    k空间是一个代表MR图像中空间频率的数字数组。k空间中的单个点(kx,ky)并不与图像中的单个像素(x,y)一一对应。每个k空间点都包含最终图像中每个像素的空间频率和相位信息。 链接至问答讨论

  2. 关于k空间的以下哪项陈述是错误的？
    1. k空间的kx轴和ky轴通常对应于图像的水平（x-）轴和垂直（y-）轴。
    2. k空间中的每个点都包含最终图像中每个像素的空间频率和相位信息。
    3. 图像中的每个像素映射到k空间中的每个点。
    4. 靠近k空间中心的点对应于图像中心的点。

    k空间中的每个点都包含最终图像中每个像素的空间频率和相位信息。因此选项(d)是错误的。 链接至问答讨论

  3. 关于k空间的以下哪项陈述是正确的？
    1. k空间的中心提供关于成像对象边缘和细节的信息。
    2. k空间的外围提供关于对象整体形状和轮廓的信息。
    3. k空间正中心的原始数据总是比任何其他点的幅度大。
    4. k空间必须以特定的顺序填充数据。

    只有(c)是正确的。k空间中心区域幅度最大的原因有两个。首先，中心行(ky = 0)是在无相位编码梯度的情况下采集的（因此没有相位编码步骤引起的破坏性波干涉）。其次，k空间的中心列(kx = 0)与MR回波的峰值重合。 链接至问答讨论

  4. 在常规解剖MR图像中，k空间的哪些点具有最低的幅度？
    1. 四个角
    2. 中心
    3. 顶部和底部边缘的中点
    4. 左侧和右侧边缘的中点

    四个角上的信号在两个方向上都具有最高的空间频率，并且对应于MR信号最弱的数据位置。为了节省螺旋成像的时间，通常会将四个角从数据采集中排除，而分辨率损失很小。 链接至问答讨论

  5. k空间中的“k”指的是“波数”，是空间频率的指标，即每单位距离的波数或周期数。因此，“k”可接受的单位是
    1. 周期每秒
    2. mm−1
    3. cm/秒
    4. 线对

    波数(k)简单来说是波长的倒数。由于波长以距离为单位测量，波数的单位是(1/距离)，例如1/mm = mm−1。其他可接受的单位包括每毫米线对数和每毫米周期数。 链接至问答讨论

  6. 关于k空间中空间频率位置的哪项陈述是正确的？
    1. 沿kx轴的点对应于图像中的水平条纹
    2. 沿ky轴的点对应于图像中的垂直条纹
    3. 与kx轴成45°角的点对应于图像中45°的条纹
    4. 如果成像对象相对于视野较小，其k空间频率表示将主要局限于中心。

    选项(c)是正确的。选项(a)和(b)颠倒了。关于不正确的选项(d)，成像对象的物理尺寸与其在k空间中的频率范围成反比。换句话说，小物体在k空间外围有波纹延伸，而大物体的频谱能量更集中在中心。 链接至问答讨论

  7. 考虑一个常规的2DFT自旋回波脉冲序列，频率编码沿x轴，相位编码沿y轴。哪项陈述是错误的？
    1. 对于给定的相位编码步(ky)，回波早期的数字化数据放置在k空间的左侧（负kx值）。
    2. 对于给定的相位编码步(ky)，回波早期的数字化数据放置在k空间的右侧（正kx值）。
    3. 对于给定的相位编码步(ky)，回波峰值的数字化数据放置在点(kx=0, ky=0)。
    4. 对于给定的相位编码步(ky)，回波峰值的数字化数据放置在点(kx=0, ky)。

    选项(c)是错误的，而选项(d)是正确的。 链接至问答讨论

  8. 关于常规2DFT自旋回波脉冲序列填充k空间（频率编码沿x轴，相位编码沿y轴），哪项陈述是错误的？
    1. k空间中心点(kx=0, ky=0)对应于图像傅里叶表示中的常数项。
    2. 除了中心行(ky=0)之外，k空间的每一行都是用相同恒定值的相位编码梯度采集的。
    3. k空间的中心行(ky=0)是在无相位编码梯度的情况下采集的。
    4. k空间的中心列(kx=0)与每个相位编码步的MR回波峰值重合。

    选项(b)是错误的。在2DFT自旋回波成像中，相位编码梯度的大小在每个周期中步进变化，而不是保持恒定。其他陈述是正确的。 链接至问答讨论

  9. 常规2DFT自旋回波脉冲序列逐行填充k空间通常被称为哪种轨迹？
    1. 笛卡尔
    2. 径向
    3. 之字形
    4. 梯度脉冲

    传统上，笛卡尔（逐行）方法几乎专门用于所有脉冲序列，但如今许多不同的模式被广泛使用。 链接至问答讨论

  10. 径向采样方法的缺点包括以下所有，除了
    1. 对运动伪影的敏感性。
    2. 排除k空间的角部。
    3. 需要运行校准扫描以估计梯度延迟和失真。
    4. 采集的数据点不是适合快速傅里叶变换的格式。

    在径向采集中，由于重叠的“辐条”反复穿过该区域，k空间中心被过采样并持续更新。如果k空间中心的信号在视图之间发生变化，这种冗余可用于检测和校正运动。此外，所有径向辐条对图像的贡献相等（与笛卡尔采样不同，笛卡尔采样中只有穿过k空间中心的几条线决定了整体图像对比度和噪声水平）。因此，仅一个或几个径向视图上的运动不太可能严重降低图像质量。 链接至问答讨论

  11. 来自径向或螺旋采集的非均匀原始数据必须放入矩形（笛卡尔）矩阵格式以便高效计算。这个过程被称为
    1. 变形
    2. 网格化
    3. 鸽巢式放置
    4. 压缩

    “网格化”是该过程的常用术语。尽管存在多种方法，但典型的网格化算法首先将原始数据乘以一组密度补偿权重，并与网格化核进行卷积。然后将所得数据插值并放入均匀间隔的矩阵（“网格”）中，执行离散傅里叶变换。最后，修剪视野，并将变换后的数据乘以变迹校正函数。 链接至问答讨论

  12. 需要多少个相位周期才能在图像中唯一区分沿某个方向的每个像素？
    1. 0.5
    2. 1.0
    3. 1
    4. 2.0

    需要一个完整的相位周期来唯一区分每个像素。如果视野上有N个宽度为Δw的像素，则需要N个相位周期。 链接至问答讨论

  13. 如果频率编码方向(x)的数据采样率减半，k空间中点之间的间距(Δkx)会发生什么？
    1. 保持不变。
    2. 减半。
    3. 加倍。
    4. 变成四倍。

    这相当于每隔一个kx值采样一次，因此与原始模式相比，间距Δkx将加倍 链接至问答讨论

  14. 如果频率编码方向(x)的数据采样率减半，该方向的视野会发生什么？
    1. 保持不变。
    2. 减半。
    3. 加倍。
    4. 变成四倍。

    我们使用公式Δk = 1/视野。因此Δk加倍意味着视野减半。 链接至问答讨论

  15. 如果频率编码方向(x)的数据采样率减半，该方向的像素尺寸会发生什么？
    1. 保持不变。
    2. 减半。
    3. 加倍。
    4. 变成四倍。

    像素宽度Δw = 1/ kFOV。只要kFOV = 2kmax不变，像素宽度也将保持不变。 链接至问答讨论

  16. 假设频率编码方向(x)的数据采样率不变，但采样时间减少，使得仅采样k空间的中心1/3。该方向的像素宽度和视野会发生什么？
    1. 像素尺寸变为三倍，视野不变。
    2. 像素尺寸不变，视野变为三倍。
    3. 像素尺寸和视野均减少1/3。
    4. 像素尺寸和视野均变为三倍。

    因为数据采样率不变，Δkx不变，点之间的间距与之前相同。所以由于Δk = 1/视野，视野x不变。通过仅采样k空间中心1/3，kFOV现在只有原来的1/3。而由于像素宽度Δw = 1/ kFOV，这意味着像素宽度变为三倍。因此答案(a)正确。 链接至问答讨论

  17. 假设2D自旋扭曲成像的频率编码部分保持不变，但相位编码步数减半（同时保持相位编码梯度的最大和最小幅度不变）。相位编码方向(y)上点之间的间距(Δky)会发生什么？
    1. 保持不变。
    2. 减半。
    3. 加倍。
    4. 变成四倍。

    在此场景下，将相位编码步数(Ny)减半会使连续步之间的增量加倍，因此Δky加倍。 链接至问答讨论

  18. 在上面的问题中，“保持相位编码梯度的最大和最小幅度不变”这句话在y方向的kFOV方面意味着什么？
    1. 保持不变。
    2. 减半。
    3. 加倍。
    4. 变成四倍。

    这句话基本上规定了kFOV不变。 链接至问答讨论

  19. 继续问题17和18中的相同场景，像素宽度和相位视野会发生什么？
    1. 像素尺寸加倍，视野不变。
    2. 像素尺寸不变，视野加倍。
    3. 像素尺寸和视野均减半。
    4. 像素尺寸不变，视野减半。

    最后这三个问题中的场景是创建一个频率:相位比为1:2的矩形视野。由于Δky加倍，视野y减半。此外，由于视野y和Ny均减少一半，相位编码方向的像素尺寸（视野y/Ny）和图像的整体空间分辨率也保持不变。 链接至问答讨论

  20. 关于通过如上所述减少相位编码步数来获得矩形视野，以下哪项陈述是错误的？
    1. 该选项在脊柱和四肢成像时最有益。
    2. 空间分辨率不变。
    3. 信噪比不变。
    4. 相位编码方向更可能出现卷褶伪影。

    选项(c)是错误的。因为采集的相位编码步数更少，从而采集的回波信号更少，信噪比相比全视野成像成比例降低。 链接至问答讨论

  21. 假设数据采集期间未发生相位误差，k空间呈现何种对称性？
    1. x轴对称
    2. y轴对称
    3. z轴对称
    4. 共轭对称

    假设数据采集期间未发生相位误差，k空间具有一种特殊的镜像特性，称为共轭（或厄米特）对称。共轭对称适用于位于k空间原点对角线上的点对（如P和Q）。如果P处的数据是复数[a+bi]，则Q处的数据立即已知为P的复共轭[a−bi]。 链接至问答讨论

  22. 关于相位共轭对称（部分傅里叶）成像，哪项陈述是错误的？
    1. 使用该技术可以实现超过50%的成像时间节省。
    2. 必须采样超过一半的相位编码线以校正相位误差。
    3. 空间分辨率得以保持。
    4. 对于半傅里叶成像，信噪比降低约30%。

    相位共轭对称技术使用k空间上半部分的数据来估计下半部分的数据。理论上，相位共轭对称允许仅使用正常相位编码步数的一半来采集数据，从而可能将成像时间减少一半。然而，为了校正相位误差，必须采样超过一半的相位编码线，因此实际节省的时间少于50%。 链接至问答讨论

  23. 关于读出共轭对称成像，哪项陈述是错误的？
    1. 没有直接的时间节省。
    2. 频率编码方向的流动和运动伪影减少。
    3. 仅采样回波的第一部分。
    4. 该技术允许更短的TE值。

    读出共轭对称的方向是沿读出（频率编码）轴。仅采样回波的最后部分，因此选项(c)是错误的。其他陈述是正确的。 链接至问答讨论

  24. 关于零填充，以下哪项陈述是错误的？
    1. 可用于2D或3D成像
    2. 用于在频率编码方向扩展成像矩阵
    3. 提高图像的视在空间分辨率。
    4. 在第一次用零将矩阵尺寸加倍后，没有额外的益处。

    零填充通常用于在相位编码方向扩展图像矩阵尺寸，该方向可以是层内（2D）或层间/容积选择（3D）。零填充作为对相邻体素信号进行插值的方法，使图像看起来更平滑，减少“像素化”外观。 链接至问答讨论

快速成像（快速自旋回波/回波平面成像）

  1. 快速/涡轮自旋回波成像是哪种原始技术的商业应用？
    1. RARE
    2. MEDIUM
    3. ROAST
    4. SMASH

    快速自旋回波成像，也称为涡轮自旋回波成像，是Hennig等人在1986年首次描述的RARE（弛豫增强快速采集）技术的商业实现。 链接至问答讨论

  2. 以下哪种表示最能描述FSE/TSE技术？
    1. 90°−180°−回波−90°−180°−回波−90°−180°−回波−…..
    2. 90°−180°−回波−180°−回波−180°−回波−180°−回波−…..
    3. 90°−180°−回波−回波−回波−回波−…..
    4. 90°−180°−回波−90°−回波−180°−回波−90°−回波−180°−回波−…..

    FSE/TSE脉冲序列在单个90°脉冲后使用一系列180°重聚焦脉冲来产生一连串回波。 链接至问答讨论

  3. FSE/TSE相对于常规自旋回波成像的优点包括除了
    1. 降低组织能量沉积
    2. 减少磁敏感性伪影
    3. 减少成像时间
    4. 相同成像时间内提高分辨率或信噪比

    FSE/TSE通过几乎同时扫描多条k空间线，显著缩短了成像时间。节省的时间可用于延长TR，使纵向磁化强度有更多时间恢复，从而提高信噪比。可以使用更多的相位编码步数，从而提高空间分辨率。磁敏感性引起的信号损失减少。然而，多个180°脉冲在组织中沉积大量能量，导致比吸收率较高。 链接至问答讨论

  4. FSE/TSE与多回波常规自旋回波序列之间的主要概念结构差异是什么？
    1. TR间隔内90°脉冲的数量
    2. TR间隔内90°和180°射频脉冲的模式
    3. TR间隔内相位编码梯度的模式
    4. TR间隔内频率编码梯度的模式

    FSE/TSE技术为这些回波中的每一个改变相位编码梯度（多回波CSE序列以相同的相位编码采集一连串中的所有回波）。 链接至问答讨论

  5. 什么是有效回波时间？
    1. 90°脉冲和第一个回波之间的时间
    2. 90°脉冲和最后一个回波之间的时间
    3. 回波串中连续回波之间的时间
    4. 90°脉冲与在k空间中心采集的回波之间的时间

    有效回波时间是填充k空间中心线(ky = 0)的时间。 链接至问答讨论

  6. 什么是回波间隔？
    1. 90°脉冲和第一个180°脉冲之间的时间
    2. 90°脉冲和最后一个回波之间的时间
    3. 回波串中连续回波之间的时间
    4. 90°脉冲与在k空间中心采集的回波之间的时间

    回波间隔是任意两个连续回波之间的时间。 链接至问答讨论

  7. 什么是回波链长度或涡轮因子？
    1. 一个TR间隔内的回波数
    2. 一个TR间隔内回波之间的时间
    3. 一个TR间隔内的射频脉冲数
    4. 到达k空间中心所需的射频脉冲数

    ETL或TF简单地是在连续90°脉冲之间FSE链中的回波数。成像时间按此因子成比例减少。 链接至问答讨论

  8. 如果具有特定TR/TE/空间分辨率的常规自旋回波序列需要8分钟完成，而其他方面相同的FSE序列具有8的回波链长度（涡轮因子），需要多长时间？
    1. 1分钟
    2. 2分钟
    3. 4分钟
    4. 8分钟

    当ETL/TF = 8时，每个TR间隔填充8条k空间线，将成像时间减少为1/8。 链接至问答讨论

  9. 随着回波链长度或涡轮因子的增加，不会发生哪种图像变化？
    1. 更强的T2加权
    2. 信噪比增加
    3. 空间模糊
    4. 脂肪信号增高

    更长的ETL与整体信噪比和对比噪声比的降低相关，因为链中较晚的回波幅度较小。 链接至问答讨论

  10. 随着回波间隔的增加，不会观察到哪种图像变化？
    1. 运动伪影增加
    2. 磁敏感性伪影增加
    3. 边缘相关伪影增加
    4. 信噪比增加

    增加回波间隔允许使用更长的TE，但对信噪比和对比噪声比产生负面影响。运动、磁敏感性和边缘相关伪影增加。通常，增加的ESP对图像质量主要产生有害后果；因此在大多数应用中应选择允许的最短ESP 链接至问答讨论

  11. 为什么脂肪在FSE/TSE上看起来比常规自旋回波图像更亮？
    1. FSE/TSE具有更多的内在T1加权
    2. 多个FSE/TSE 180°脉冲破坏了脂肪分子中的J耦合
    3. FSE/TSE中快速的相位编码梯度切换破坏了脂肪酸的扩散
    4. 仅仅是视觉错觉；绝对脂肪信号不变。

    尽管多种因素可能导致FSE“脂肪高信号”现象，但主要机制被认为是由于J耦合相互作用的破坏导致的T2延长，这种相互作用通常发生在相邻脂肪质子之间。 链接至问答讨论

  12. 比较FSE/TSE和CSE，哪项陈述是错误的？
    1. FSE/TSE对磁敏感性变化的敏感性增加。
    2. FSE/TSE表现出增加的磁化传递效应。
    3. FSE/TSE表现出空间模糊
    4. FSE/TSE表现出伪边缘增强

    FSE/TSE序列对磁敏感性变化的敏感性较低，因为以非常短的间隔发生的180°重聚焦脉冲使得磁敏感性诱导的散相时间相对较少。 链接至问答讨论

  13. 如果在MR序列结束时播放，哪种脉冲可被视为驱动平衡脉冲？
    1. +90°
    2. +180°
    3. −90°
    4. −180°

    驱动平衡，也称为快速恢复，是一种在MR序列结束时使用−90°“回翻”脉冲来帮助恢复纵向磁化强度的技术。 链接至问答讨论

  14. 比较射频脉冲的组织能量沉积，哪项是正确的？
    1. 90°和180°脉冲沉积的能量大致相同
    2. 180°脉冲沉积的能量是90°脉冲的两倍。
    3. 180°脉冲沉积的能量是90°脉冲的一半。
    4. 180°脉冲沉积的能量是90°脉冲的四倍。

    射频脉冲的能量沉积与翻转角平方（α²）成正比。因此，180°脉冲沉积的能量是90°脉冲的4倍。 链接至问答讨论

  15. 以下哪项脉冲序列功能在3D-FSE序列（如CUBE/SPACE/VISTA）中通常未按所述方式实现？
    1. 非常短的TE（2-4 ms）
    2. 中等长度的回波链（涡轮因子8-16）
    3. 减小翻转角
    4. 具有零插值填充的部分傅里叶成像。

    3D-FSE序列使用极长的回波链，可能有100-200个回波。其他特性包括非选择性射频脉冲、并行成像和优化的k空间轨迹。 链接至问答讨论

  16. 为什么回波平面成像通常使用之字形k空间轨迹，而不是像自旋回波成像中那样使用一致的笛卡尔（从左到右）轨迹？
    1. 提供更好的空间分辨率。
    2. 伪影更少。
    3. 执行速度更快。
    4. 在相位编码方向而非频率编码方向上进行。

    之字形轨迹很受欢迎，因为它易于编程且速度更快，因为它不需要在每个回波后将读出梯度重新绕回−kmax。空间分辨率相同，但之字形更容易产生伪影，如奈奎斯特N/2鬼影。 链接至问答讨论

  17. 关于单次激发FSE（HASTE）的以下所有陈述都是正确的，除了
    1. 使用梯度回波信号生成以实现速度。
    2. 总回波数可能超过200。
    3. 利用相位共轭对称来限制相位编码步数。
    4. 常见应用包括磁共振胆管成像和磁共振脊髓成像。

    其名称SS-FSE/HASTE中隐含了快速自旋回波技术，意味着记录的是自旋回波，而不是梯度回波。 链接至问答讨论

并行成像

  1. 关于并行成像的以下哪项陈述是错误的？
    1. 它几乎可以与所有脉冲序列一起使用。
    2. 它需要多个接收线圈。
    3. 它可以在任何方向上进行。
    4. 其主要目的是减少所需的相位编码步数。

    允许进行并行成像的方向取决于阵列中线圈的布置。具体来说，线圈必须沿为并行成像选择的方向具有不同的灵敏度分布。并行成像不能与仅有一对正交输出通道的单个表面线圈、头线圈、膝线圈或大体线圈一起使用。 链接至问答讨论

  2. 以下哪项不是并行成像的缺点？
    1. 磁敏感性伪影增加。
    2. 混叠伪影增加。
    3. 线圈校准伪影。
    4. 非均匀方式的噪声增加，信噪比降低。

    由磁敏感性引起的MR信号中的相位相关失真通过并行成像采集和重建过程减少（而不是增加）。这在回波平面序列中尤其有利。 链接至问答讨论

  3. 比较图像域并行成像技术（如SENSE/ASSET）和k空间并行成像技术（如GRAPPA/ARC），哪项陈述是错误的？
    1. 对于高R（加速）值，图像域并行成像技术提供稍高的信噪比。
    2. 图像域并行成像在异质身体区域（如胸部）表现更好
    3. k空间并行成像技术显示更少的混叠，并且在需要小视野时更有用。
    4. k空间并行成像技术在最小化磁敏感性失真方面更有效。

    图像域并行成像技术如SENSE/ASSET在异质身体区域表现不佳。这是因为可能难以获得准确的线圈灵敏度图。肺是一个典型的例子。 链接至问答讨论

  4. 获取图像域并行成像图像的正确步骤顺序是什么？缩写：A = 采集部分k空间数据；G = 生成线圈灵敏度图；R = 重建单个线圈图像；U = 展开/组合部分视野图像。
    1. A — G — R — U
    2. G — A — U — R
    3. G — A — R — U
    4. A — G — U — R

    正确的顺序是生成/采集/重建/展开 (c)。 链接至问答讨论

  5. 获取k空间并行成像图像的正确步骤顺序是什么？缩写：A = 采集部分k空间数据；C = 将数据组合成幅度图像；E = 估计缺失的k空间线；G = 生成单个线圈图像
    1. A — C — E — G
    2. A — E — G — C
    3. E — A — G — C
    4. E — A — C — G

    正确的顺序是采集/估计/生成/组合 (b)。 链接至问答讨论

  6. 关于CAIPIRINHA序列的以下哪项陈述是错误的？
    1. 它可以归类为图像域并行成像技术。
    2. 加速因子（R）通常为4。
    3. 最常用于3D动态肝脏成像。
    4. 它使用移位的k空间采样模式来减少像素混叠和重叠。

    CAIPIRINHA显然是一种k空间（而非图像域）并行成像技术，因为它依赖于重建前的k空间数据采样和处理。 链接至问答讨论

  7. 当加速因子（R）从2增加到8时，并行成像研究中的成像时间会发生什么？
    1. 减少为原来的1/4
    2. 减少为原来的1/2
    3. 减少为原来的1/√2
    4. 减少为原来的1/√2

    成像时间与R成反比，因此如果R变为四倍（从2到8），成像时间减少为原来的1/4 链接至问答讨论

  8. 当加速因子（R）从2增加到8时，并行成像研究中的信噪比会发生什么？
    1. 减少为原来的1/4
    2. 减少为原来的1/2

  9. 关于g因子的哪项陈述是错误的？
