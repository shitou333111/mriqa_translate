自旋

  1. 关于核自旋（I），以下哪项陈述是正确的？
    1. 自旋是由于原子核绕其轴旋转而产生的。
    2. 质子有自旋，但中子没有。
    3. 自旋只能取整数或半整数值。
    4. 自旋的另一个名称是“进动”。

    自旋是亚原子粒子的基本量子属性，并非由其物理旋转产生。除质子外，许多亚原子粒子（包括中子和电子）也具有自旋。自旋是量子化的，只能取有限数量的离散值，因此c)正确。当置于外磁场中时，核自旋会导致进动，但自旋和进动不是一回事。 链接至问答讨论

  2. 原子核不可能具有以下哪种自旋（I）？
    1. 0
    2. 1/2
    3. 3/4
    4. 9/2
    5. 5

    量子力学将核自旋限制为仅取整数或半整数（1/2, 3/2, 5/2等）值，因此I = 3/4 是不允许的。 链接至问答讨论

  3. 关于核自旋（I），以下哪项陈述是错误的？
    1. “自旋”一个更长但等价的名称是“自旋角动量”。
    2. 对于氢（¹H）MRI，通常可以互换使用术语“原子核”、“自旋”和“质子”。
    3. 常规临床 MRI 仅测量来自氢（¹H）原子核的信号。
    4. 氢（¹H）原子核包含一个质子和一个电子。

    氢（¹H）原子核仅包含一个质子，周围环绕着电子云，因此d)是错误的。其他陈述均正确。 链接至问答讨论

  4. 关于核自旋（I），以下哪项陈述是错误的？
    1. 质子和中子各自的自旋均为½。
    2. 要确定净核自旋（I），只需将质子和中子的数量相加，再除以2。
    3. 同一元素的不同同位素通常具有不同的核自旋。
    4. 每种元素至少有一种同位素具有非零自旋。

    净核自旋（I）确实取决于质子和中子的总数，但不存在用于计算I的简单公式，因为必须考虑更基本成分（夸克和胶子）之间的相互作用。其他陈述——a)、c)和d)——均正确。 链接至问答讨论

  5. 关于核自旋（I）和NMR，以下哪项陈述是错误的？
    1. 除了质子和中子均为偶数的原子核外，所有原子核都能发生NMR。
    2. 元素周期表中的每种元素至少有一种同位素可以发生NMR。
    3. 在元素周期表中，可以找到核自旋（I）值从0到8的同位素。
    4. I = 0的原子核容易发生NMR。

    只有自旋非零的原子核才能发生NMR，因此d)是错误的。其他陈述均正确。 链接至问答讨论

  6. 关于磁偶极矩，以下哪项陈述是错误的？
    1. 它将原子核建模为一个具有南北极的小条形磁铁。
    2. 另一种表示方法是来自小电流环的矢量（μ）。
    3. 像指南针指针一样，偶极矩会趋向于与外施磁场对齐，以达到其最低能态。
    4. 置于外磁场中时，偶极矩会进动。

    磁偶极矩（μ）会经历一个力矩，使其与外施磁场对齐，但不会绕磁场进动。 链接至问答讨论

  7. 关于置于外磁场中的磁偶极子，以下哪项陈述是错误的？
    1. 当偶极子指向与磁场相反的方向时，其处于最低能态。
    2. 偶极子所受的力矩与外磁场的强度成正比。
    3. 偶极子所受的力矩取决于偶极子与外磁场之间的夹角。
    4. 偶极子-外磁场系统的能量取决于偶极子与外磁场之间的夹角。

    偶极子（μ）所受的力矩（τ）由矢量叉积τ = μ x B0给出。就标量大小而言：||τ|| = ||μ|| ||B0|| sin θ，其中θ是它们之间的夹角。能量（E）由点积定义：E = −μ • B0 = −||μ|| ||B0|| cos θ。当偶极子指向与磁场相反的方向时（θ = 180°），能量最大（而非最小），因此答案a)是错误的。 链接至问答讨论

  8. 偶极磁矩（μ）与核自旋（I）成正比，它们通过一个称为______的常数联系起来。
    1. 旋磁比（γ）
    2. 普朗克常数（h）
    3. 核磁化率（χ）
    4. 化学位移（δ）

    两者通过旋磁比（γ）联系起来，关系式为：μ = γI 链接至问答讨论

  9. 关于旋磁比（γ），以下哪项陈述是错误的？
    1. 它可以用 MHz/Tesla 为单位表示。
    2. 它可能为负值。
    3. 每种元素都不同。
    4. 对于给定元素的所有同位素都相同。

    旋磁比（γ）是偶极磁矩（μ）与核自旋（I）之间的比例常数。通常以 MHz/T 为单位表示。虽然通常是正数，但也可能是负数。γ为负值意味着磁矩和自旋方向相反。旋磁比（γ）对于每种元素的每种同位素都不同，因此答案d)是错误的。 链接至问答讨论

  10. ¹H原子核的自旋（I）= ½。当置于外磁场中时，其可测量的自旋态（本征态）数量为
    1. 0
    2. ½
    3. 1
    4. 2

    自旋为I的原子核的可观测自旋态数量等于 2I +1。因此，¹H原子核有 2(½) + 1 = 2 种可能的自旋态。 链接至问答讨论

  11. 除了¹H之外，NMR波谱中常研究的两种原子核是³¹P和¹³C。这三种同位素都具有相同的核自旋（I）= ½，这意味着当置于外磁场中时：
    1. 它们将具有完全相同的共振频率。
    2. 它们各自都会表现出两个离散的可测量能态。
    3. 每种同位素的能态差异相同。
    4. 它们将具有相同的旋磁比（γ）。

    所有自旋-½粒子在置于外磁场中时都会表现出两个离散的能态，因此答案b)正确。然而，这些同位素具有不同的旋磁比（γ）、不同的共振频率和不同的能级。 链接至问答讨论

  12. ²³Na是另一种用于成像研究的NMR活性同位素。其核自旋（I）= 3/2。与I = 1/2的¹H原子核相比，在外磁场中：
    1. ²³Na将具有比¹H更高的旋磁比（γ）。
    2. ²³Na将具有比¹H更高的共振频率。
    3. ²³Na将表现出3个离散的能态。
    4. ²³Na将表现出4个离散的能态。

    自旋为I的原子核的可观测能态数量等于 2I +1。因此，²³Na原子核有 2(3/2) + 1 = 4 种可能的自旋态，所以答案d)正确。旋磁比（γ）和共振频率不依赖于I。 链接至问答讨论

  13. 展示自旋-½粒子如何通过磁场被物理分离成两组的著名实验是1922年由______进行的。
    1. 爱因斯坦
    2. 海森堡
    3. 斯特恩和格拉赫
    4. 普朗克和狄拉克

    这是对斯特恩-格拉赫实验的描述，该实验通常作为核角动量量子化在宏观世界可见的有形证据。 链接至问答讨论

  14. 以下哪个名称指的是磁场中¹H原子核的最高能自旋态？
    1. 自旋向上
    2. 自旋向下
    3. 平行
    4. |+½>

    最高能态出现在自旋方向与主磁场方向相反时，称为自旋向下（正确答案）、反平行或|−½>。 链接至问答讨论

  15. 为什么不是所有核自旋都简单地落入其最低能态以使系统总能量最小化？
    1. 它们确实如此。上述陈述是错误的。
    2. 这种高度偏斜的能态分布为量子力学所禁止。
    3. 热分子碰撞倾向于使原子核在较低能态和较高能态之间的分布均等化。
    4. 由于海森堡不确定性原理，能态分布无法获知。

    自旋落入较低能态的自然趋势被倾向于均衡两个能级的热碰撞所抵消。其结果是由玻尔兹曼分布预测的折衷。 链接至问答讨论

  16. 根据量子力学理论和实验，哪项陈述是正确的？
    1. 像质子这样的自旋-½粒子必须仅存在于自旋向上或自旋向下态中。
    2. a)部分中描述的量子力学限制的决定性证据由斯特恩-格拉赫实验提供。
    3. 科学家已经能够捕获自旋并将其操控于向上和向下态的叠加态中。
    4. 科学界几乎普遍同意量子力学的哥本哈根诠释是正确的。

    量子力学并不要求质子必须仅存在于其两个主要（本征）态之一，而只要求是两者的线性组合。因此答案a)是错误的。斯特恩-格拉赫实验仅表明，当进行测量时，一组自旋-½粒子会物理显现两种结果之一。然而，这并不说明测量前的潜在量子现实，因此b)也是错误的。Haroche和Weinland独立工作，因其捕获自旋并将其操控于叠加态的实验而共同获得2012年诺贝尔物理学奖，因此c)是正确的。量子力学的哥本哈根诠释虽然仍最流行，但远未获得普遍接受，因此d)是错误的。最近的量子物理学家民意调查发现多世界诠释正在获得显著进展。当然，两者可能都不正确！ 链接至问答讨论

进动

  1. 核进动可以看作是自旋角动量与外施磁场相互作用时受到“扭转力”或力矩（τ）作用的结果。这个力矩的方向是：
    1. 与自旋共线。
    2. 与自旋共线但方向相反。
    3. 与磁场共线。
    4. 垂直于自旋和磁场。

    力矩（τ）垂直于自旋角动量和磁场。这产生了一个可以理解为“从侧面推动自旋”的“扭转力”，导致核进动而非与磁场对齐。 链接至问答讨论

  2. 进动可以用角频率（ω0）或循环频率（f0）表示。两者通过方程______相关。
    1. ω0 = 2 π f0
    2. f0 = 2 π ω0
    3. ω0 = π f0
    4. f0 = π ω0

    正确的方程是a)。角频率（ω0）以弧度每秒为单位，其中2π弧度/秒 = 360°/秒 = 1周（或转）/秒 = 1赫兹（Hz）。因此，循环频率（f0）必须乘以2π才能得到角频率（ω0）。 链接至问答讨论

  3. 角频率（ω0）的常用单位是：
    1. 每秒周期数（cps）
    2. 赫兹（Hz）
    3. 弧度/秒
    4. 每分钟转数（rpm）

    正确答案是c)，弧度每秒。其他选项用于表示循环频率（f0）。 链接至问答讨论

  4. 以下哪项是拉莫尔方程的正确形式？
    1. f0 = γ B0
    2. f0 = γ / B0
    3. f0 = B0 / γ
    4. 以上方程均不正确。

    正确答案是a)，f0 = γ B0。 链接至问答讨论

  5. ¹H原子核的旋磁比（γ）大约是多少？
    1. 10.7 MHz/Tesla
    2. 42.6 MHz/Tesla
    3. 64.0 MHz/Tesla
    4. 128 MHz/Tesla

    正确答案是b)，42.6 MHz/Tesla。我认为你不需要记住这个数字，但可以轻松推算出来，因为我确实相信每个参与MRI的人都应该知道1.5T扫描仪的共振频率约为64 MHz。因此γ ≈ 64 MHz ÷ 1.5 Tesla，得出γ约为42.6 MHz/T。 链接至问答讨论

  6. 如果1.5T扫描仪中¹H的共振频率约为64 MHz，那么在7T时¹H的共振频率大约是多少？
    1. 128 MHz
    2. 256 MHz
    3. 298 MHz
    4. 426 MHz

    共振频率与场强成正比。因此，7.0T时的¹H共振频率可计算为64 MHz x (7.0/1.5) ≈ 298 MHz。 链接至问答讨论

  7. ¹³C原子核的旋磁比（γ）约为10.7 MHz/T。在3.0T时，¹³C的共振频率是多少？
    1. 10.7 MHz
    2. 21.4 MHz
    3. 32.1 MHz
    4. 64.2 MHz

    根据拉莫尔方程：f0 = γ B0 = (10.7 MHz/T) x 3.0T = 32.1 MHz。 链接至问答讨论

  8. 约瑟夫·拉莫尔爵士推导出他的著名方程，以便在不同磁场中计算NMR共振频率。
    1. 正确
    2. 错误

    错误。拉莫尔在19世纪末推导他的方程时，NMR甚至尚未被发现。他当时在研究磁场对光谱的分裂作用，他的方程描述的是轨道电子的进动。20世纪，人们发现拉莫尔方程适用于任何具有自旋角动量的粒子，因此完全适用于NMR。 链接至问答讨论

  9. 关于核进动，以下哪项陈述是正确的？
    1. 核进动直到施加射频脉冲才会开始。
    2. 维持核进动需要环境不断输入能量。
    3. 海洋中的每一滴水和北极的每一片雪花中的质子此刻都在进动。
    4. 使用地球磁场无法获得MR图像，因为它太弱了。

    当质子置于任何磁场中时，核进动会自发发生。启动或维持进动不需要能量输入。即使在最微弱的磁场（包括地球磁场）中也会发生进动，因此答案c)正确。事实上，仅使用地球磁场的粗略MR图像已被获取。 链接至问答讨论

  10. 不同分子环境中的¹H原子核之间观察到的微小共振频率差异是由于：
    1. 不同的旋磁比
    2. 不同的局部磁场
    3. 不同的弛豫时间
    4. 不同的自旋量子数

    这个问题涉及化学位移的起源。分子电子云对¹H原子核的可变屏蔽和去屏蔽导致每个原子核经历的局部磁场略有不同。这些不同的局部磁场会略微改变共振频率。 链接至问答讨论

  11. 化学位移（δ）通常以______为单位报告。
    1. 高斯 (G)
    2. 毫特斯拉每米 (mT/m)
    3. 百万分之一 (ppm)
    4. 百分比 (%)

    化学位移通常以ppm报告，这是一个与场强无关的单位。注意，ppm和%一样，是无量纲的。 链接至问答讨论

  12. 缩写“ppm”代表：
    1. 质子顺磁矩
    2. 质子-质子磁化强度
    3. 每分钟进动次数
    4. 百万分之一

    正确答案是d)，“百万分之一”，这是报告化学位移的常用场强无关方法。 链接至问答讨论

  13. 在1.5T时，水和脂肪质子之间的化学位移（δ）约为3.5 ppm。在3.0T时，它们的化学位移是多少？
    1. 1.75 ppm
    2. 3.5 ppm
    3. 7.0 ppm
    4. 10.5 ppm

  14. 两种脑代谢物，N-乙酰天冬氨酸（NAA）和肌酸（Cr）的甲基质子，其化学位移相差1.0 ppm。在场强为1.5 T（水的拉莫尔频率为64 MHz）时，它们的频率差约为：
    1. 64 MHz
    2. 1.0 MHz
    3. 64 kHz
    4. 64 Hz

    频率差 = 拉莫尔频率 x 化学位移 = 64 MHz x 1 ppm = (64 x 106 Hz) x (1.0 x 10−6) = 64 Hz。 链接至问答讨论

  15. 在1.5 T时，水和脂肪质子之间的化学位移导致约220 Hz的频率差。在3.0 T时，它们的频率差是多少？
    1. 110 Hz
    2. 220 Hz
    3. 440 Hz
    4. 660 Hz

    场强加倍，因此拉莫尔频率加倍，化学位移导致的频率差也加倍。因此正确答案是c)。 链接至问答讨论

  16. 关于净磁化强度（M），以下哪项陈述是错误的？
    1. M可以看作是数百万个独立核自旋的矢量和或平均值。
    2. 使用M允许使用经典物理（而非量子物理）分析NMR现象。
    3. 在平衡状态下，M与外磁场对齐。
    4. 在平衡状态下，M绕外磁场方向进动。

    除了d)之外，其他陈述均正确。在平衡状态下，M是静止的，并与外磁场对齐。需要施加RF脉冲才能使M偏离与B0的对齐，此时它将开始进动。 链接至问答讨论

  17. 在远离强外磁场的未磁化样品中，净磁化强度（M）：
    1. 在所有方向上都有效地为零。
    2. 具有显著非零的纵向分量。
    3. 具有显著非零的横向分量。
    4. 自发进动。

    在没有强外磁场的情况下，单个质子自旋在空间中是随机取向的，它们的矢量和在所有方向上基本上为零。然而，实际上总存在一些小的外磁场，即使仅仅来自地球本身。因此M在所有方向上只是有效地（但从未完全）为零。 链接至问答讨论

  18. 为什么置于磁场中的未磁化样品材料，其净磁化强度（M）最初不会产生横向分量？
    1. 这个问题的前提是错误的；M从一开始就产生横向分量。
    2. 横向分量起初观察不到，因为它们需要几秒钟才能形成。
    3. 横向分量被M的进动抵消了。
    4. 构成M的各个自旋具有随机分散的横向分量，缺乏相位相干性。

    答案d)正确，因为自旋具有随机的横向角动量分量。需要RF脉冲或接近拉莫尔频率的能量输入来产生一定的相位相干性和横向分量。 链接至问答讨论

  19. 决定净磁化强度（M）纵向分量最初发展速率的内在组织参数是什么？
    1. 自旋密度（ρ）
    2. T1
    3. T2
    4. T2*

    纵向磁化强度（Mz）的初始增长是一个简单指数函数，时间常数为T1。 链接至问答讨论

  20. 关于净磁化强度（M），以下哪项陈述是错误的？
    1. 当未磁化的组织样品置于外磁场中时，会产生净磁化强度（M）。
    2. 最初M沿纵向增长，因为各个自旋试图与B0对齐。
    3. 当M被倾斜偏离与B0的对齐时，它将以与其组成的各个原子核相同的共振频率进动。
    4. 即使当M完全反转并指向−z方向（即与B0相反）时，它也会继续进动。

    只有d)是错误的。即使构成M的各个自旋总是在进动，M本身只有在被倾斜偏离与B0的对齐，从而产生非零横向分量时才会进动。一旦M相对于B0完全反转180°，它就不再具有横向分量，也不再进动。如果在这种反转后不施加任何干预，M将简单地沿着z轴重新增长，恢复到其初始的与B0对齐的取向和大小。 链接至问答讨论

  21. 关于核磁共振，以下哪项陈述是错误的？
    1. 将净磁化强度（M）从其最初与B0的对齐状态倾斜需要自旋系统吸收能量。
    2. 在MRI中，启动NMR所需的能量源通常由称为B1的旋转/振荡射频场提供。
    3. 这种对（M）的倾斜是NMR现象的一种表现。
    4. 核进动和共振本质上是相同的。

    除了d)之外，其他陈述均正确。所有非零自旋粒子置于外磁场中时都会经历核进动，且不需要能量输入。NMR是自旋系统的一种特殊状态，需要在狭窄的频率范围内吸收和释放能量。 链接至问答讨论

  1. 哪位科学家在1930年代首次通过实验证明了NMR现象并为其命名？

  2. 关于费利克斯·布洛赫和爱德华·珀塞尔的NMR发现，以下哪项陈述是正确的？

  3. 用于向自旋系统注入能量以诱导核共振的射频（RF）场被称为：
    1. B0
    2. B1
    3. B2
    4. Mxy

    正确答案是B1。B0是主磁场，Mxy是由B1诱导的净磁化强度的横向分量。没有称为B2的场。 链接至问答讨论

  4. 为了发生共振，B1必须精确地施加在拉莫尔频率上。

    2. 错误

    2. 错误

  6. 关于使用传统RF脉冲的翻转角，以下哪项陈述是错误的？

  7. （高级）关于90°脉冲后立即的自旋系统，以下哪项陈述是正确的？
    1. 如果测量所有质子的z分量角动量，将观察到相等数量的自旋向上和自旋向下态。
    2. 90°脉冲导致自旋绕B1进动。
    3. 所有自旋都彼此锁定在相位相干状态。
    4. 每个质子的自旋角动量都转变为水平方向，指向B1的方向。

    这是一个相当棘手的问题，真正考验一个人对NMR现象的理解。只有a)是正确的。如果测量所有自旋的纵向角动量（例如，让它们通过斯特恩-格拉赫装置），将观察到自旋向上和自旋向下态的均等分布。请记住，自旋并不存在于纯本征态中，因此不能说存在相等数量的自旋向上和自旋向下质子（尽管这个说法在教科书中很常见）。但其思想大致相同。选项b)是错误的，但只差一点。90°脉冲之后的进动发生在B0方向周围，与RF脉冲之前一样。在施加RF脉冲期间，进动发生在B0和B1两者周围，在关于旋转坐标系的问答中有更详细的描述。选项c)是错误的，因为并没有发生自旋“神奇地”锁定在一起。我们所说的“相位相干”仅仅是在RF脉冲作用下，最初偏斜的纵向自旋能量分布被旋转到了横向平面。最后，选项d)是错误的。根据海森堡不确定性原理，我们无法知道自旋“指向”的方向，因此该陈述毫无意义。但我很确定，如果我们能够知道，自旋不会都水平“指向”某个方向！ 链接至问答讨论

  8. 当被RF激发刺激到更高能级时，质子会自发释放这些能量并回落到较低能级。
    1. 正确
    2. 错误

  9. 当净磁化强度矢量（M）同时受到B0和B1作用时，其复杂运动可以通过在______中考虑系统来简化。

    4. 绝热参照系。

  10. （高级）如果B1场未精确施加在拉莫尔频率上，会发生什么？

    2. 自旋将停止围绕B0进动，并开始围绕B1进动。

  11. （高级）关于绝热激发，以下哪项陈述是错误的？
    1. 与纯粹幅度调制的“传统”RF脉冲不同，绝热RF脉冲也是频率调制的。
    2. 脂肪抑制技术SPAIR使用绝热反转。

  11. 预计以下哪种生物材料的T2值最短？
    1. 尿液
    2. 跟腱
    3. 脾脏
    4. 肝脏
    5. 股四头肌

    当存在使自旋失相位的静态局部磁场时，T2弛豫发生最快。组织越“固体”、越“干燥”，其T2值越短。在上述选项中，肌腱的T2最短。 链接至问答讨论

  12. 预计以下哪种生物材料的T1值最短？
    1. 头皮脂肪
    2. 脓液
    3. 肝脏
    4. 脾脏
    5. 红骨髓

    3. 膀胱中的尿液
    4. 肾脏

  15. 以下哪种分子机制对¹H NMR中的T1和T2弛豫最重要？
    1. 偶极-偶极相互作用
    2. 化学位移各向异性
    3. 化学位移各向异性

  16. 因为电子和质子的自旋均为½，它们在产生偶极-偶极相互作用方面同样有效。
    1. 正确
    2. 错误

    由于其尺寸小，电子具有更大的电荷质量比，因此具有大得多的旋磁比（γ）。因此，在偶极-偶极相互作用中，电子的威力比质子强数千倍。 链接至问答讨论

  17. 分子内偶极-偶极相互作用比分子间相互作用更强。

    2. 错误

  18. 在偶极-偶极相互作用中，对于______，T1弛豫最有效（T1值最短）。

    由于T1弛豫需要在拉莫尔频率进行刺激性能量交换，因此以接近拉莫尔频率翻滚的分子上的自旋在T1弛豫方面最有效。 链接至问答讨论

  19. 在偶极-偶极相互作用中，对于______，T2弛豫最有效（T2值最短）。
    1. 快速翻滚的小分子。
    2. 翻滚频率接近拉莫尔频率的分子。
    3. 缓慢移动的大分子。
    4. 结合在刚性生物支架或基质上的大分子

  20. 考虑到大分子对弛豫时间的影响，以下哪项是错误的？

    2. 在大分子附近通常存在一层运动受限的“结合”或“水合”水分子。
    3. 这些“结合”水质子仅与大分子相互作用，而不与“自由水”池相互作用。
    4. 结合水分子翻滚更慢，导致T1和T2缩短。这在一定程度上解释了组织相对于自由水T1和T2值降低的原因。

  21. 在使用常规MRI序列记录的脑实质信号中，以下哪种含¹H分子几乎占100%？
    1. 水
    2. 甘油三酯
    3. 髓鞘
    4. N-乙酰天冬氨酸（NAA）

    甘油三酯存在于头皮脂肪中，但不存在于脑实质本身。髓鞘存在于脑中，但作为一种不动的生物大分子，其T2极短，因此其信号衰减太快，无法使用常规MRI序列检测。NAA是脑代谢物，在脑的MR波谱中显示最大的峰值，但其浓度比水低数千倍。 链接至问答讨论

  4. 胆固醇引起的T1缩短在很大程度上解释了以下哪种疾病中看到的MR信号？
    1. 岩尖胆固醇肉芽肿
    2. 胆固醇性胆结石
    3. 颅咽管瘤
    4. 以上所有
    5. 以上都不是

    胆固醇在所有状态下都具有短T1值和与其他脂质相似的化学位移范围。然而，胆固醇的T2值根据状态而有显著差异。液态胆固醇酯的T2值相当短（~5-10毫秒），但这些时间足够长，可以通过常规MRI检测/成像。相比之下，与细胞膜相关的固态和结晶形式胆固醇具有极短的T2时间（以微秒计）；其信号衰减太快，无法使用常规MR技术检测。在颅咽管瘤和胆固醇肉芽肿中，胆固醇处于固态/结晶状态，高信号可能是由于血液产物所致。胆固醇性胆结石也处于固态，通常不具有高MR信号。 链接至问答讨论

  5. 为什么微钙化区域有时在T1加权图像上显得明亮？
    1. 钙原子核也发生NMR并与¹H信号混合
    2. 你看到的是磁敏感性伪影，而非真正的弛豫时间变化
    3. 与多孔钙表面的盐相互作用减慢了水分子的旋转
    4. 微钙化内部通常含有含脂肪的骨髓

    高信号并非来自钙本身，因为Ca是偶数元素，自旋为零，没有内在MR信号。信号来自水质子，其分子旋转速率通过与钙盐的表面相互作用而减慢到接近拉莫尔频率。这种旋转减慢导致T1缩短，从而在T1加权图像上表现为高信号。 链接至问答讨论
