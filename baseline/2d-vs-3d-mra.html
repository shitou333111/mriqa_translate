2D 与 3D MRA

你提到 TOF MRA 可以在 2D 或 3D 模式下进行。我应该使用哪一种？  
大多数 MRA 技术，包括时间飞跃法 (TOF)，都可以使用 2D 或 3D 采集模式进行。2D TOF 通常用于对垂直于成像平面走行的较长血管节段（如主动脉或股动脉）进行成像。3D 模式则用于血管走行方向多样的、相对紧凑的解剖区域（如颈动脉分叉、Willis 环或肾动脉）。 
颈部动脉的 2D TOF 图像
2D TOF 磁共振血管成像。 在垂直于目标血管走行方向的平面上，获取多个薄层（1-2 mm 厚）并堆叠起来。 TR 值保持较短（<30 ms）以抑制背景组织并增强流动相关强化。2D TOF MRA 的层片是顺序采集的；在移动到下一层之前，会收集完当前层的完整数据。通常会在每个层片的上游侧放置一个约 4 cm 宽的移动饱和带 ，以抑制来自静脉流入的污染信号。 

2D 方法成像时间短，对慢血流具有极佳的敏感性。 通过简单地增加层片数量，可以对较长的血管节段（如整个主动脉或下肢动脉）进行成像。2D 成像的缺点包括对层内流动不敏感，以及患者运动伪影可能导致配准错误。 
3D TOF 磁共振血管成像。3D TOF 模式用于成像解剖结构涵盖区域相对较小且血管走向多样的部位。3D 方法提供高空间分辨率和高信噪比。可以获得近乎各向同性（立方体）的体素 ，从而允许在任何方向进行重建。
3D TOF MRA 的缺点包括对慢血流相对不敏感，以及饱和效应限制了每次采集的最大层块厚度。其成像时间也比 2D TOF 技术长。虽然在 2D TOF MRA 中运动伪影仅影响单个层片，但在 3D 采集中，所有层片都会受到影响，可能导致整个研究无法解读。
