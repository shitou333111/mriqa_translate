ASL 成像参数

应如何选择成像参数以优化 ASL 采集？  
最佳成像参数的选择取决于所使用的具体 ASL 方法（2D 与 3D，PICORE 与 FAIR 与 pCASL 等）。即使是序列的微小变化（例如背景抑制的类型和脉冲整形）也会影响参数选择。还应考虑预期的血流量，例如儿童的循环比成人更旺盛。许多选择是经验性的，因此应始终从设备供应商或脉冲序列设计者推荐的参数开始。 尽管如此，一些通用规则适用于每种实现方案。
与成人相比，儿童具有更高的相对血流量和更短的平均通过时间
  1. 成像平面。当在垂直于成像层的横断面上进行采集时，所有 ASL 方法的效果最佳。
  2. 设备。使用可用的最高场强。ASL 技术受信噪比 (SNR) 限制，因此在 3.0T 下的性能均优于 1.5T 或更低场强。必须使用多通道线圈。低水平的并行加速 (R≈2) 可能对 3D 采集有用，但由于其会降低 SNR，在 2D 方法中应谨慎使用。
  3. 重复时间 (TR) 应足够长，以便标记的自旋在两次采集之间有充分的弛豫时间，通常大于 3500 ms。 
  4. 最小回波时间 (TE) 应始终被使用，以保留因 T2/T2* 衰减而损失的信号。
  5. 空间分辨率。避免“高分辨率”成像。在当前技术状态下，SNR 的考量要求采集的图像空间分辨率低于标准诊断所用的图像。对于 2D ASL，典型的平面矩阵在 64x64 到 128x128 范围内，相应的层厚为 4-6 mm。对于 3D ASL，可以实现各向同性体素，但由于图像采集时间的限制，通常仍选择各向异性体素。
  6. 信号平均。 为了在合理的成像时间（4-6 分钟）内保持可接受的 SNR，需要进行多次信号平均。对于 2D 技术，在 3.0T 下需要 30 到 50 次信号平均，在 1.5T 下则需要更多。对于 3D 技术，需要 2-4 次信号平均。
  7. 标记持续时间与标记后延迟反转时间 (TI)。这些参数取决于场强、ASL 方法和预期的血流速度。对于 1.5T 下的脑成像，PASL 的典型标记持续时间为 700 ms，pCASL 为 1500 ms，TI 值通常为 1800 ms。由于血液在 3.0T 下的 T1 值更长，因此需要稍长的 TI。 当预期动脉血流缓慢时，例如老年患者或有血管病变或心输出量低的患者，也应增加 TI。相反，由于儿童的循环时间更快，建议使用较短的 TI 值。 
  8. 扰相梯度。其使用存在争议。虽然它们可以减少流动相关伪影，但需要延长最小 TE，从而降低 SNR。它们还会在 ASL 图像中引入更多的 T2 (T2*) 对比度，这在定量血流测量中必须加以考虑。扰相梯度还可能从 ASL 图像中去除重要的临床信息，例如延迟或侧支血流的存在。
  9. 背景抑制。对于每个 TR 使用单次激发的分段 3D 成像方法是必需的。对于多层 2D 方法效果/效率较差，可视为可选。背景抑制会略微降低 ASL 信号（约 5%），如果需要进行定量血流测量，应考虑到这一点。

进阶讨论（显示/隐藏）»

一些软件允许执行快速、极低分辨率的多 TI ASL 序列来估算团注到达时间。如果可用，这可用于为每位患者选择最佳的 TI。
