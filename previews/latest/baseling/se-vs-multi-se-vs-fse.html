自旋回波序列的变体

自旋回波、多回波自旋回波和快速自旋回波之间有什么区别？ 
单回波自旋回波
自旋回波 (SE) 脉冲序列最简单的形式包括一个 90° 脉冲、一个 180° 脉冲，然后产生一个回波。第一个射频脉冲的中心到自旋回波峰值之间的时间称为 回波时间 (TE)。  然后序列在时间 TR 处重复，即 重复时间。
传统（单回波）SE 脉冲序列
正如在 之前的问答 中所描述的，180° 脉冲能够重聚那些因恒定磁场畸变和不均匀性而导致相位分散的静止自旋。该脉冲无法校正由原子/分子层面的随机过程引起的 T1 或真实 T2 效应。它也无法校正那些移动、流动、扩散或发生化学交换的自旋的相移。但是，如果一个自旋仅仅是因为位于含铁血黄素团块附近或主磁场中任何原因造成的缺陷附近而去相位，那么 180° 脉冲将对此进行校正！
自旋回波的形成。

左侧的动画（修改自 维基百科）展示了自旋回波形成的步骤：1) 通过 90° 脉冲将自旋倾倒至横向平面；2) 这些自旋因局部磁场不均匀性而去相位；3) 通过 180° 脉冲将整个自旋集合像煎饼一样在横向平面内翻转；以及 4) 在时间 TE = 2 x t 处自旋重聚形成自旋回波。 

多回波自旋回波
只要 T2 弛豫尚未完全破坏 MR 信号，就有可能用额外的 180° 脉冲刺激系统并产生额外的回波。由于 T2 衰减，每个回波的幅度逐渐减小。
双回波 SE 序列
直到过去十年，像上图所示的双回波 SE 序列还是一种标准技术。第一个回波（短 TE）提供“自旋密度”对比，而第二个回波（长 TE）提供 T2 加权对比。如今，对于大多数应用，T2-FLAIR 已在很大程度上取代了自旋密度加权的 SE 图像，因此多回波 SE 仅偶尔使用。

快速（涡轮）自旋回波 (FSE) 

多回波 SE 应与 FSE 区分开来，在 FSE 中，多个 180° 脉冲和回波也跟随每个 90° 脉冲。简化的 FSE 脉冲-回波时序图如下所示。
快速（涡轮）自旋回波示意图
表面上看，这似乎与多回波 SE 相似，尽管你可能会注意到，与预期的多回波 SE 不同，回波的幅度并未随着 TE 的增加而均匀减小。回波看起来不同的原因是，在 FSE 中，每个 180° 脉冲都施加了不同的相位编码梯度（图中未显示）。在单回波和多回波 SE 成像中，相位编码梯度在每个 TR 间隔内仅开启一次。  根据回波数量的不同，在 FSE 成像中，相位编码在每个 TR 间隔内可能被施加 8、16、32 次或更多次。  

如果你现在不理解这一点，请不要担心，因为我们尚未涉及相位编码、k 空间或图像生成的部分。后续将有多个完整的问答章节专门讨论这些主题以及 FSE 成像。
