预测化学位移

如何预测分子中氢的化学位移？  
化学位移源于分子中 ¹H 核受到的不同电子屏蔽。屏蔽较弱的核具有较大的化学位移（δ 值较高）。屏蔽程度可以通过观察邻近分子基团的电负性来估算，特别是靠近氧 (O) 原子、双键和芳香环的情况。这些概念在下图中进行了说明，并在后续章节中有更完整的解释。
四甲基硅烷 (TMS), δ = 0
四甲基硅烷 (TMS)
四甲基硅烷 (TMS) 作为 ¹H NMR 波谱的参考标准，被任意指定为 δ = 0。选择 TMS 担任此角色是因为其 ¹H 核受到电子云的高度屏蔽。中心的 Si 和 C 原子电负性低，且分子对称。几乎所有其他有机分子的 ¹H 核受到的屏蔽都比 TMS 弱，因此具有更大的化学位移。 
脂肪族质子, δ = 1 − 2
大多数脂肪族质子，例如脂肪（甘油三酯）中包含的那些，共价键合在相对较长（16-20 个原子）的碳原子链上。碳的电负性很弱，共价键合的 -(CH2)- 基团拥有的电子云能有效地屏蔽质子免受外场影响。因此，大多数脂肪族脂质的化学位移与 四甲基硅烷 (TMS) 参考值相差不远，通常在 δ = 1.0-1.5 范围内。
水质子, δ = 4.65
由于水中高电负性的氧原子导致的 ¹H 核去屏蔽。

与脂肪和简单烷烃的 ¹H 核不同，水质子键合在 高电负性的氧原子上。电子云被拉向氧原子，导致 ¹H 质子被 去屏蔽。与深嵌在长共价含碳链中的脂肪质子不同，水质子更“暴露”于外加磁场。因此，它们感受到更高的局部磁场，比脂肪族质子共振得更快。水的确切化学位移会随温度、pH 值、氢键以及溶解溶质和气体的存在而变化。
芳香族质子, δ = 6 − 8
芳香族（酚类）质子，例如在胆固醇和其他含苯环化合物中发现的那些，具有更大的化学位移，范围在 6-8 ppm。这种大化学位移的原因与芳香环周围 离域 (π) 电子 （“环电流”）的循环有关。这些电子循环感生的局部磁场增强了环平面内苄基质子所感受到的外加磁场，导致它们感受到比预期更高的局部磁场。 
由于 π 电子循环，苄基氢感受到的增强局部磁场
羧酸和醛类, δ = 9 − 12
羰基 (C=O) 具有磁各向异性且电负性强。它们会强烈去屏蔽位于其平面内的邻近 ¹H 核。这解释了羧酸 (R-COOH) 和醛类 (R-COH) 相对较大的 δ 值。 
金属蛋白（大 ±δ's）

对于非常靠近顺磁性离子（如脱氧血红蛋白等金属蛋白中的 Fe+3）的 ¹H 核，可以观察到非常大的化学位移（低于 −50 到超过 +100 ppm）。这是由于所谓的 超精细位移 机制，该机制源于质子与金属中心附近未配对电子之间的标量和偶极相互作用。根据几何结构，位移可能为正或负。这些大位移峰通常非常小，因为绝大多数质子并不靠近顺磁中心，其 δ 值仍显示在“正常”范围（0 - 12 ppm）。

经验丰富的化学家能够通过考虑诸如 sp 杂化、靠近电负性原子（O、N、卤素）或磁各向异性基团（酚基、双键、羰基）等因素，将大多数有机分子的 ¹H δ 值预测到约 0.3 ppm 以内。数千种分子的化学位移已被测量，取代基表（例如 Curphy-Morrison 表）广泛可用。最复杂和准确的化学位移预测来自使用现代量子化学软件程序（例如 Gaussian、GAMESS），这些程序考虑了三维构型、磁各向异性、孤对电子相互作用和空间压缩效应。

理论上，一个完全去屏蔽（“裸露”）的质子的化学位移约为 +40 ppm，这个值在所谓的“超强酸”（如 HF-SbF5）中近似。当分子含有金属中心时，可能会出现更大（正和负）的化学位移。

前述金属蛋白中描述的离域电子与质子之间的 超精细位移 源于通过化学键的标量（费米）接触贡献以及具有角度和 1/r³ 距离依赖性的偶极（赝接触，“通过空间”）相互作用。两种贡献都与 S(S+1) 成正比，其中 S 是自旋态。因此，在氧合血红蛋白（S=0）中看不到此效应，但在脱氧血红蛋白（S=1/2）中存在，并且在高铁血红蛋白（S=5/2）中程度更大。

例如，弱抗磁性的氧合血红蛋白，其几乎所有共振峰都在 0 到 +10 ppm 范围内，这是大多数不含金属的有机分子的典型范围。脱氧血红蛋白的谱图仍以这些核心共振峰为主，但也显示出许多额外的从 +10 到 +20 以及低至 −20 的负位移小峰，所有这些都源于靠近血红素中心的质子。已确定 δ = +75 和 +63 ppm 处的两个非常微小的宽峰源于近端组氨酸残基的超精细位移 NH 质子。这些峰已在实验中用于监测 O2 结合。具有比脱氧血红蛋白更强顺磁中心的高铁血红蛋白，在“标准”范围外显示出更多的共振峰，最高位移出现在 δ = +87。

¹H 核附近存在顺磁中心也可能导致其化学位移仅从预期位置偏移几个 ppm。例如，已记录到从乳房植入物泄漏的降解硅胶碎片具有 δ 值在 0 到 −3 范围内（而非预期的 0 到 +0.5 范围）。这种意外的负位移被归因于与组织中铁和其他顺磁性离子的聚集。

需要注意的是，并非所有顺磁性离子都会引起超精细位移。例如，Gd+3 具有非常短的电子自旋弛豫时间（T1e），其功能更像是谱线“展宽剂”而非谱线“位移剂”。
