概述

  1. 一名精神状态改变的患者从急诊室送来接受急诊磁共振检查，但没有家属、既往影像学资料或病史可用。为确保检查的合理安全性，你应该怎么做？
    1. 致电急诊科医生，询问在缺乏病史的情况下是否有合理的替代磁共振检查方案，如果没有，则按医嘱进行扫描。
    2. 使用手持式铁磁性金属探测器排除危险的植入物或异物
    3. 进行头颈部、胸部及腹部/盆腔X线检查作为筛查程序
    4. 拒绝进行磁共振检查，直到能够获得完整的病史。

    在完全缺乏病史的情况下，磁共振检查前的最低筛查应包括头颈部、胸部及腹部/盆腔X线检查，这将识别出绝大多数潜在危险的植入物和异物。因此c)是正确的。虽然询问急诊科医生是否可以进行替代影像学检查是合理的，但仅执行磁共振检查医嘱而未进行进一步安全核查是不够的，因此a)是错误的。手持式金属探测器不一定能识别深部体内的物体或由非铁磁性金属构成的物体，因此b)是错误的。拒绝检查可能会使患者失去重要且紧急的诊断机会，从而危及健康，因此d)也是错误的。 链接至问答讨论

  2. 以下哪项被认为是“有源”植入物？
    1. 植入式输液泵
    2. 磁性义齿
    3. 带可编程阀门的脑脊液分流管
    4. 肺动脉导管

    有源植入物包含其自身的电源，因此a)植入式输液泵是唯一正确的答案，因为它有电池。磁性义齿和可编程脑脊液分流阀含有永磁体，但没有内在电源。肺动脉导管可能包含导线和电子传感器，但由外部供电，因此被认为是“无源”装置。 链接至问答讨论

  3. 以下哪项是目前不被接受的用于描述装置在磁共振环境中安全性的术语？
    1. MR 不安全
    2. MR 条件性安全
    3. MR 兼容
    4. MR 安全

    尽管在2005年之前使用过，但根据ASTM标准F2503-20，MR兼容(c)已不再是可接受的术语。然而，许多在2005年之前制造和测试的装置可能仍带有旧标签。 链接至问答讨论

  4. 关于装置的安全性标签，以下哪项陈述是正确的？
    1. 如果大型硅胶植入物内部深处含有一个微小的非铁磁性夹子，则不能评定为MR安全。
    2. 一个MR条件性安全的装置，只要遵循大部分指定条件即可安全扫描。
    3. 任何含有永磁体的植入物都是MR不安全的。
    4. 如果植入物在严格的ASTM测试下没有发生移动或加热，则可评定为MR安全。

    要被认为是MR安全的，装置不得含有任何金属成分。硅胶植入物中的微小螺丝使其成为MR条件性安全，因此a)是正确的。MR条件性安全的装置必须满足所有指定的条件，因此b)是错误的。内部含有永磁体并不一定使装置成为MR不安全（尽管可能是）。最后，如果植入物含有少量金属，即使在实验室测试下没有移动或加热，也不能被认为是MR安全的，只能是MR条件性安全。因此d)也是错误的。 链接至问答讨论

  5. 关于磁共振中的抛射物事件，以下哪项是正确的？
    1. 自磁共振诞生以来，全球至少有十几人死于抛射物事故。
    2. 所有报告的死亡案例均涉及铁磁性氧气罐。
    3. 大多数报告的抛射物事件涉及小物体。
    4. 由非铁磁性材料制成的物体仍可能成为危险的抛射物。

    磁共振中仅有两例有记录的抛射物致伤死亡报告——2001年纽约一名儿童和2018年孟买一名男子。两人均被铁磁性氧气罐击中。因此a)是错误的，b)是正确的。大多数报告的抛射物事件涉及大型物体（例如轮椅、推车、输液架）被无意带入磁共振室，因此c)是错误的。完全由非铁磁性材料制成的物体只会被磁场微弱吸引，不会构成危险，因此d)是错误的。 链接至问答讨论

  6. 关于磁共振环境中金属探测器的以下哪项陈述是错误的？
    1. 磁共振中的金属探测器与用于机场安检的探测器非常相似。
    2. 联合委员会现在要求所有新建和翻新的磁共振室配备铁磁性金属探测器。
    3. 磁共振金属探测器无法可靠地检测深部植入的铁磁性材料。
    4. 在使用磁共振金属探测器的场所，报警疲劳可能是一个问题。

    答案a)是错误的，而其他陈述是正确的。机场探测器设计用于检测任何金属的存在，而磁共振探测器则设计为仅在存在铁磁性金属时报警。 链接至问答讨论

  7. 关于孕妇的磁共振成像，哪项陈述是错误的？
    1. 由于对胎儿存在未知风险，只有在母亲或胎儿的生命受到直接威胁时才应进行磁共振检查。
    2. 应尽可能避免使用钆对比剂。
    3. 首选低比吸收率序列，以避免加热胎儿组织。
    4. 首选静音梯度序列，以避免在孕早期后使胎儿暴露于高声噪音。

    尽管磁共振对发育中胎儿的风险极低，但可能并非为零，因此建议在与患者讨论后审慎使用磁共振。大多数有效的磁共振检查指征是严重的，但并非直接危及生命（例如母体中风、胎儿异常），因此选项a)是错误的。其他选项是正确的。 链接至问答讨论

  8. 以下哪项常见的术后物品应作为潜在的磁共振安全危害进行进一步审查：
    1. 皮肤缝合钉
    2. 静脉导管和管路
    3. 心电图电极片
    4. 扁条引流管

    心电图电极片，答案c)，最应引起关注，因为那些未认证可用于磁共振的电极片含有金属，可能导致皮肤烧伤。皮肤缝合钉由非铁磁性金属制成，没有造成损伤或移位的风险。静脉导管和管路是安全的（前提是没有连接金属夹）。最后，扁条引流管不含金属，也是MR安全的。 链接至问答讨论

  9. 对于哪些患者，磁共振检查期间不需要进行体温监测？
    1. 健康青少年
    2. 健康婴儿
    3. 麻醉患者
    4. 老年患者

    其他方面健康的青少年和非老年成人接受常规磁共振检查不太可能出现热力学不稳定性，不需要进行体温监测。因此答案a)是正确的。其他列出的患者类型存在风险，应进行监测。 链接至问答讨论

  10. 磁共振检查期间测量体温最可靠的装置是
    1. 直肠光纤探头
    2. 食管光纤探头
    3. Foley导尿管热敏电阻
    4. 置于腋窝的液晶皮肤探测器

    最佳答案是b)，一种带有荧光或砷化镓探测器及光纤电缆的探头，置于食管远端三分之一处。 链接至问答讨论

  11. 在以下哪个眼眶位置存在小块金属不作为磁共振检查的禁忌证？
    1. 漂浮在玻璃体液中
    2. 在肌锥内间隙
    3. 在泪腺中
    4. 在结膜上

    泪腺，答案c)，距离重要视觉结构足够远，即使发生磁场诱导位移也不会造成损伤。 链接至问答讨论

  12. 考虑到在磁共振检查前使用眼眶CT筛查金属异物，以下哪项是错误的？
    1. 应对所有机械师、管道工、铸造厂工人或任何有类似职业性金属碎片暴露史的人常规进行眼眶CT筛查。
    2. 眼眶平片是CT可接受的替代方法。
    3. 应采用低剂量CT技术。
    4. CT可能检测到小到在磁共振中无临床意义的金属碎片。

    在我看来，仅仅因为工作性质而可能接触金属，如果没有已知的损伤史，不足以证明进行昂贵的CT筛查是合理的。严重眼损伤的实际风险极为罕见，40年来报告的病例不到半打。因此答案a)是错误的。其他陈述都是正确的。 链接至问答讨论

  13. 关于子弹和弹片，哪项陈述是正确的？
    1. 手持式铁磁性金属探测器可以可靠地确定子弹是由铅还是钢制成。
    2. 双能CT可以可靠地确定子弹是由铅还是钢制成。
    3. 过去25年在美国，钢被步枪子弹的销售或持有是非法的。
    4. X光片上看到的子弹碎片变形或碎片轨迹提示抛射物不是由钢制成的。

    只有答案d)是正确的，但此类发现并非决定性，只是对成分有强烈提示作用。其他选项都是错误的。如果子弹触发手持式探测器的警报，则它是铁磁性的，但阴性结果不确定。同样，尽管双能CT和其他技术有希望，但技术问题仍然存在。最后，钢被子弹在美国对于手枪是非法的，但对于步枪则不然。 链接至问答讨论

  14. 在评估一名带有假定铁磁性子弹或弹片的患者时，哪项因素是评估进行磁共振检查风险最重要的因素？
    1. 碎片的位置
    2. 碎片的形状
    3. 植入后的时间长度
    4. 是否是在军事环境中获得的

    就像房地产一样，一切都是关于位置、位置、位置。选项a)是最佳选择。如果异物嵌入如大脑这样的软组织中，即使有瘢痕形成，当置于磁场中时仍可能发生明显移动并造成显著的局部组织损伤。相反，嵌入肌肉、骨骼或肌腱等致密组织中的碎片则不太可能移动。形状很重要，因为长而参差不齐的碎片比小而圆的碎片更危险。并且损伤后时间越长，就有越多时间形成限制其在磁场中移动的瘢痕/纤维化。但位置在相对重要性上超过形状和时间。在军事环境中获得会增加碎片为铁磁性的可能性，但问题已假定你已确定其为此种成分。 链接至问答讨论

静态磁场

  1. 关于铁磁性，以下哪项陈述是错误的？
    1. 根据定义，所有铁磁性材料必须至少含有一些铁。
    2. 当磁畴中的电子自旋排列对齐时，就会产生铁磁性。
    3. 随着铁磁性物质被磁化，磁畴的大小和形状发生变化。
    4. 硬铁磁性材料在从外部磁场中移除后仍能保持显著的磁化强度。

    尽管名称如此，铁磁性材料并不需要含有铁；它们只需要形成具有被磁化能力的磁畴。镍、钴、铬、锰以及几种稀土元素及其合金都是铁磁性的。因此a)是错误的。陈述b)、c)和d)是正确的。 链接至问答讨论

  2. 以下哪种金属不被认为是铁磁性的？
    1. 镍
    2. 钴
    3. 铬
    4. 镁

    锰是铁磁性的，但镁(d)不是。 链接至问答讨论

  3. 关于不锈钢，以下哪项是错误的？
    1. 400系列不锈钢是铁磁性的，而300系列是非磁性的。
    2. 奥氏体不锈钢是铁磁性的，而马氏体不锈钢是非磁性的。
    3. 自1990年以来，美国食品药品监督管理局要求所有钢制植入物必须由非铁磁性不锈钢制成。
    4. 即使是非铁磁性不锈钢，如果在机械加工过程中弯曲或拉伸，也可能产生铁磁性。

    答案a)是正确的，但b)是错误的，因为名称被颠倒了：马氏体（=400系列）钢是铁磁性的，而奥氏体（=300系列）钢是非磁性的。c)和d)都是正确的。 链接至问答讨论

  4. 关于磁饱和，哪项陈述是错误的？
    1. 几乎所有铁合金的饱和范围都在1-2特斯拉。
    2. 所有铁磁性材料在3.0T磁场中都会饱和。
    3. 不可能制造出磁场强度大于2.5特斯拉的永久铁磁体。
    4. 永磁体的最大强度与其饱和点无关。

    选项d)是错误的。饱和点限制了永磁体或铁芯电磁铁能达到的最大强度，在大多数情况下远低于2特斯拉。答案a)-c)是正确的 链接至问答讨论

  5. 由以下哪种材料制成的植入物可能因其铁磁性而构成安全风险：

    2. 镍钛诺（镍钛合金）
    3. 硅钢
    4. 金

  6. 以下哪种稀土元素通常用于制造室温永磁体？
    1. 钕
    2. 磁动力
    3. 钬
    4. 镝

    掺杂铁和硼的钕(a)通常用作室温永磁体的基础，钐钴也是如此。钬和镝在20 ºK以下具有高铁磁性，但在较高温度下会失去此特性。选项b)磁动力不是一种元素！ 链接至问答讨论

  7. 以下列表中哪种金属或合金在梯度回波成像上预计会产生最大的磁敏感性伪影？
    1. 铅
    2. 铜
    3. 铝
    4. 镍钛诺（镍钛合金）

    镍钛诺，答案d)，由于其镍含量而具有最大的磁化率，因此会产生最大的伪影。 链接至问答讨论

  8. 关于铁磁性材料的磁滞回线，以下哪项是正确的？
    1. 材料在首次暴露于外部磁场后，其曲线会发生变化。
    2. 矫顽力是外部磁场移除后保留的磁化强度
    3. 反转磁场方向不能使磁化方向反转。
    4. 硬铁磁性材料的磁滞回线具有小的矫顽力。

    只有a)是正确的。“原始”材料（即从未暴露于强外部磁场的材料）开始时内部磁化强度为零或极小，随着施加外部磁场，该磁化强度增长到最大值（饱和值）。当该外部磁场移除时，磁化强度通常不会回到零，而是保持在一个正值——剩磁，而不是如b)中错误陈述的“矫顽力”。反转外部磁场可以反转磁化强度，只要其大小大于矫顽力值（答案c是错误的）。硬铁磁性材料难以退磁，具有非常宽的磁滞回线，因此具有大的矫顽力（答案d是错误的）。 链接至问答讨论

  9. 关于退磁场，哪项陈述是错误的？
    1. 抗磁性和顺磁性材料的退磁场非常小，实际上可以忽略不计。
    2. 退磁场(D)指向与磁化强度(M)相反的方向
    3. 退磁场(D)指向与外部磁场(Bo)相反的方向
    4. 退磁场(D)增加了物体内部的净磁场。

    答案d是错误的。退磁场通过对抗M和B，降低了物体内部的净磁场。 链接至问答讨论

  10. 关于退磁场的影响，哪项陈述是错误的？
    1. 退磁场增加了弱铁磁性物体的表观磁化率。
    2. 由强铁磁性材料制成的物体的表观磁化率主要取决于其形状，而非其具体成分。
    3. 铁磁性材料中的退磁场降低了实现磁饱和所需的外部磁场强度。
    4. 退磁因子是无量纲数，范围在0到1之间。

    除a)外，所有陈述都是正确的。退磁场降低了弱铁磁性物体的表观磁化率。 链接至问答讨论

  11. 一名眼部嵌有钢珠的人接近3.0T磁共振扫描仪。假设制造该钢珠的低碳钢饱和值Bsat = 1.5T，并且由于其球形，在每个方向上的退磁因子(N)为1/3，计算钢珠达到磁饱和时的外围外部磁场(Bext)为多少？
    1. 0.5 T
    2. 1.0 T
    3. 1.5 T
    4. 3.0 T

    产生磁饱和的外部磁场由方程Bext= N • Bsat给出。对于钢珠的情况，Bext= 1/3 • 1.5T = 0.5T (答案a) 链接至问答讨论

  12. 以下哪种形状的材料沿主磁场方向具有最大的退磁因子(N)？
    1. 球体
    2. 面向主磁场的平板
    3. 平行于主磁场的平板
    4. 沿主磁场方向指向的细长圆柱体

    垂直于主磁场方向（en face）的平板，其虚拟磁极非常接近，因此会产生强烈的退磁效应。因此N将接近1.0，这是任何形状物体可能达到的最大值，所以答案b)是正确的。 链接至问答讨论

  13. 对金属物体所受磁力的数学分析通常使用基于以下哪种形状的模型？
    1. 球体

  14. 将一个未饱和的金属物体移近圆柱孔式磁共振扫描仪时，其平移力最大处是
    1. 刚进入磁共振扫描室门口时
    2. 在磁体孔洞中心的扫描等中心点
    3. 在扫描等中心点的孔洞边缘
    4. 刚好在磁体孔洞开口边缘的内侧

    平移力与局部磁场(B)乘以其空间变化率(dB/dz)成正比，这个组合量称为空间梯度积。空间梯度积在磁体孔洞开口边缘附近最强，使其成为平移力最大的位置（答案d是正确的）。然而，在磁体等中心点，dB/dz ≈ 0，因此一旦物体到达磁体中心，出乎意料地没有平移力。 链接至问答讨论

  15. 将一个未饱和的金属物体移近圆柱孔式磁共振扫描仪时，其扭矩最大处是
    1. 刚进入磁共振扫描室门口时
    2. 在磁体孔洞中心的扫描等中心点
    3. 在扫描等中心点的孔洞边缘
    4. 刚好在磁体孔洞开口边缘的内侧

    这是一个有点误导性的问题。答案实际上是d)，位于空间梯度或空间梯度积最大值位置稍后的位置。对于未饱和物体，扭矩与磁场(B)的平方成正比。因此，扭矩在B最大处最大。大多数人认为这是在扫描仪的中心，但对于圆柱形磁体，局部场在孔洞开口边缘内侧沿扫描仪内壁处可能高出10-20%。因此正确答案是d)，位于空间梯度或空间梯度积最大值位置稍后，即磁场(B)最强的位置。 链接至问答讨论

  16. 未饱和金属物体所受的力最小处是哪里？
    1. 仅精确地在扫描等中心点
    2. 在磁体孔洞内静态场均匀的绝大多数位置
    3. 刚好在扫描仪孔洞开口边缘的内侧
    4. 距扫描仪孔洞开口一米处

    正确答案是b。当空间梯度(dB/dz)最小时，平移力最小。由于现代扫描仪具有高度均匀性，磁体主孔内的大多数位置dB/dz = 0，因此不产生平移力。 链接至问答讨论

  17. 一把钢制扳手被无意中带入装有自屏蔽1.5 T扫描仪的房间。关于扳手所受的磁扭矩，哪项陈述是正确的？
    1. 当扳手垂直竖立并垂直于主磁场时，扭矩最大。
    2. 当扳手与磁场方向成45º角倾斜时，扭矩最大。
    3. 当扳手转向与磁场平行时，扭矩最大。
    4. 扳手的确切形状或位置对扭矩没有影响，只有其质量有影响。

    答案b)是正确的。未饱和细长物体的扭矩取决于sin 2θ，其中θ是与外部磁场形成的角度。当θ = 45º时，该值最大。 链接至问答讨论

  18. 当金属物体被外部磁场(B)磁饱和时，以下哪项是错误的？
    1. 位移力与B无关。
    2. 位移力与dB/dz无关。
    3. 扭矩与B无关。
    4. 扭矩与物体相对于B的倾角(θ)无关。

  19. 空间梯度积定义为空间各点（静态场强）乘以（静态场的空间梯度）。空间梯度积的单位是
    1. 特斯拉
    2. 特斯拉/米
    3. 特斯拉/米²
    4. 特斯拉²/米

    静态场(T)乘以空间梯度(T/m)得出空间梯度积的单位为T²/m，因此答案d)是正确的。 链接至问答讨论

  20. 对于典型的圆柱形磁共振扫描仪，空间梯度积的最大值位于
    1. 磁体孔洞中心的扫描等中心点
    2. 扫描等中心点的孔洞壁处
    3. 孔洞开口的中心
    4. 孔洞开口处的壁沿

  21. 一个小的金属物体正在使用ASTM方法在扫描仪孔洞开口边缘通过细线悬挂进行平移力测试。悬挂物使细线偏离垂直线40º。以下哪项结论是不正确的？
    1. ASTM会声明磁力带来的风险不大于地球重力带来的风险。
    2. ASTM会宣布该物体为MR条件性安全。
    3. ASTM会宣布该物体为MR不安全。
    4. ASTM会宣布该物体为MR安全。

    由于物体偏转角未超过45º，偏转力小于装置自身的重量。因此根据ASTM标准，施加磁感应偏转力带来的任何风险都不大于正常日常活动在地球引力场中带来的任何风险。因此答案a)是正确的。MR安全、不安全、条件性安全的定义基于多种因素（即，加热、扭矩），超出了此简单位移测试所能提供的信息。 链接至问答讨论

  22. 关于洛伦兹力，以下哪项是正确的？
    1. 它是带电粒子在电场中运动时所受的力。
    2. 它导致心电图出现T波变化
    3. 它导致7T扫描仪中出现磁光幻视和味觉干扰。
    4. 它导致镰状红细胞在磁场中堆积。

    只有b)是正确的。这描述了磁流体动力学效应，是由于洛伦兹力导致降主动脉中正负离子位移引起的心电图记录电压变化。其他选项都是错误的。洛伦兹力是由粒子在磁场（而非电场）中运动产生的。磁光幻视和味觉干扰是由法拉第-楞次定律感应的电流引起的。镰状红细胞的堆积是一种磁敏感性诱导力。 链接至问答讨论

  23. 关于磁共振相关的头晕和眩晕，以下哪项是错误的？

  24. 关于磁光幻视，哪项陈述是错误的？
