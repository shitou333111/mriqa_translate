波的性质

哪些性质定义了电磁波？
描述电磁波形状的周期性振荡函数在其数学表述中，有三个基本性质清晰可见：
S(t) = A sin(ωot − ϕ)
其中 S(t) 是作为时间函数的信号， A 是 振幅， ωo 是 角频率，而 ϕ 是 瞬时相位。  
频率 (f 和 ω)
由于部分读者可能不熟悉角频率，在此插入一个简短的 YouTube 视频和解释或许是合适的。
常规频率或线性频率 (f)，有时也用希腊符号 "nu" (ν) 表示，用于统计给定时间内完整的振荡或旋转次数。因此其单位是每秒周数 (cps)，也称为 赫兹 (Hz)。MRI 中使用的频率通常在 10 到 300 MHz ("兆赫" = 百万赫兹) 之间。

角频率 (ω)，也称为径向频率或圆周频率，测量的是单位时间内的角位移。因此其单位是度（或弧度）每秒。 由于绕圆一周是 2π 弧度 = 360°，所以 1 弧度 ≈ 57.3°。 角频率（以弧度计）的大小是常规频率（以 Hz 计）的 2π 倍：                 ω = 2πf
速度 (v)
在真空中，电磁波以光速传播，通常用"c"表示。这个速度是一个普遍的物理常数，精确地等于 299,792,458 m/s。（这个数字是精确的，因为根据国际协议，米的定义与光速相关）。

在物质中，速度会减慢，这是因为振荡的电场使束缚电荷（电子和离子）极化，在整个材料中产生感应电偶极子。这些偶极子会再次辐射出与入射波相位不同的次级场。 入射场和再辐射场的叠加导致相位滞后和相速度降低，因此波前在材料中的传播速度比在真空中慢。（这种现象是物理学中所谓折射率的基础）。

材料的电磁响应特性由它的 相对（电）介电常数 (εr) 和它的 相对（磁）导率 (μr) 共同表征。在非磁性、非色散介质中，麦克斯韦方程预测电磁波以相速度 v = c (εrμr)−½ 传播。对于大多数生物组织，μr ≈ 1，因此 εr 成分占主导。人体组织是中等介电性的（εr ≈ 60−80），因此在不考虑导电损耗的情况下，射频波在体内的速度仅为光速的 11−13% 左右。然而，在射频频率（10-300 MHz）下的真实组织中，介电常数是频率相关且呈复数的，并且导电损耗可能很显著。因此，波会受到强烈衰减和畸变，其相速度可能高达 0.2c（取决于确切的频率和组织类型）。

波长 (λ)
波长 (λ) 定义为具有 速度 (v) 和 频率 (f) 的正弦波上对应点（例如，波峰或波谷）之间的距离，并通过以下公式关联：λ = v / f.  由于介电常数和电导率效应的影响，射频波在人体组织中的波长，与速度一样，大约只有其在真空中波长的 10-20%。
在 3.0 T 及更高场强下，射频波长变得与人体横截面尺寸相当。其影响包括：1）B1 磁场不均匀性显著增加；2）出现具有相长/相消干涉的驻波效应；以及 3）介电阴影。 
组织中射频波长随场强的变化关系。
相位 (ϕ)
瞬时相位的概念及其与频率的关系可能是另一个令人困惑的话题。瞬时相位 (ϕ) 表示周期性波形相对于其原点的角度偏移。例如，一个 ϕ = 0 的正弦波在 t = 0 时应为零值。  上图中的正弦波经过了相移，其值不为零。相移以弧度（或度）来度量。  在此定义下，正弦波和余弦波彼此相位相差 90° (π/2 弧度)。

经过一段时间 Δt 后，两个初始相位同步但角频率相差 Δω 弧度/秒的正弦波将产生一个由下式给出的总相位差 (ΔΦ)： ΔΦ = Δω × Δt。或者用微积分更普遍地表达为： 
Φ = ∫ ω(t) dt
其中 Φ 是在一小段时间增量 (Δt≅dt) 内累积的总相位变化，而 ω(t) 是可能随时间变化的频率。总累积相位变化 (Φ) 可以看作是频率-时间曲线下的面积。当我们后续在问答环节讨论梯度或流动引起的相位变化时，这个事实将证明是有用的。 
如果你仍然不太理解为什么（频率 x 时间）= 相位，请思考一下测量的单位。频率通常以赫兹（Hz）或每秒周数来测量。时间以秒为单位测量。 所以 频率 x 时间 = (周数/秒) x 秒 = # 周数。因此，两个频率相差 200 Hz 的正弦波，彼此之间每秒会逐渐产生 200 个周期的相位差。
要理解磁共振图像中频率和相位编码的更复杂方面，有必要回顾当两个正弦波相加时会发生什么。  如果两个正弦波具有相同频率但不同相位，它们的总和是另一个具有相同基频但不同振幅和相位的正弦波。  当两个波完全同相时，它们的信号会相互增强。  当它们稍微不同相时，总体信号会减弱，这称为相消干涉。  当它们之间的相位差达到 180° 时，两个波恰好相互抵消。 
两个频率相同但相位不同的正弦波的叠加。当 Δφ → 180°（反相条件）时，两个波发生相消干涉，产生的净信号几乎为零。（由宾夕法尼亚州立大学声学研究生课程的 Dan Russell 博士提供）。
