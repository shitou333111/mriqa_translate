True FISP

什么是 True FISP，它为何比常规 FISP 更“真实”？  
True FISP 是西门子公司对一种稳态相干脉冲序列的商品名，该序列在所有三个轴上均使用平衡梯度。“平衡”意味着在一个 TR 间隔内，由梯度引起的净失相为零。相反，FISP 和 PSIF 序列中的部分梯度是不平衡的，这使得可以分别记录稳态自由进动信号的 FID 和回波分量。然而，在 True FISP 中，平衡梯度将这两个分量在 TR 间隔的正中心重聚为一个单一的回波。 其他厂商也有类似的序列。GE 称其为 FIESTA（采用稳态采集的快速成像），飞利浦则称其为 Balanced-FFE（平衡式快速场回波）。 
True FISP。注意沿所有 3 个轴的对称（平衡）梯度以及回波在序列中间形成。这就是它更“真实”的原因吗？
True FISP 带状伪影。
尽管 True FISP 中回波形成的原理早已为人所知，但由于严格的技术要求，其广泛的临床应用进展缓慢。True FISP 序列对磁场均匀性以及梯度切换和成形的控制水平要求极高。 如果体素内失相超过 ±180º，重聚机制就会失效，表现为带状伪影。在过去十年中，现代扫描仪已克服了这些限制，使得 True FISP 在大多数中高场系统中成为一种可行且有用的序列。

这是一个有趣的结果。可以看到信号强度与比值 1 ÷ (T1/T2) = T2/T1 直接相关。对于大多数实体组织，T2/T1 比值 很小。例外是 脂肪 (0.30) 和液体 (0.70)。 因此，True FISP 序列倾向于突出显示诸如脑脊液和血液等液体，使其成为心血管 MRI、MR 脑池/脊髓造影、MR 尿路造影和 MR 小肠造影的理想选择。
对于大多数需要明亮 T2/T1 信号的 True FISP 应用，TR 选择尽可能短，TE = TR/2，翻转角 (α) 设置为中到高范围 (50°-80°)。低翻转角提供自旋密度加权对比，但很少使用。 同样，通常也避免使用长的 TR 值。
