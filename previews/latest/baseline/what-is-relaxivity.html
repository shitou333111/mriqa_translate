弛豫率

对比剂的弛豫率是什么意思？如何测量？  
MR 对比剂的弛豫率反映了溶液的弛豫速率如何随浓度 [C] 变化。由于对比剂可能分别影响两种弛豫速率（1/T1 和 1/T2），因此存在两个对应的弛豫率，分别表示为 r1 和 r2。 根据定义
 1/ΔT1 = r1 • [C]           以及         1/ΔT2 = r2 • [C]  
由于 ΔT1 和 ΔT2 的单位是秒，而 [C] 的单位是毫摩尔每升，因此 r1 和 r2 的单位是升/毫摩尔-秒。
溶液中对比剂的弛豫速率是通过绘制不同浓度下弛豫速率的变化（1/ΔT1 和 1/ΔT2）来获得的。直线的斜率代表 r1 和 r2.
弛豫率取决于温度、场强以及溶解对比剂的物质。对于临床使用的对比剂，通常引用其在 1.5T 场强下、于体温（37°C）血浆中测得的 r1 和 r2 值。

左侧的表格说明并非所有钆对比剂都相同，其 r1 和 r2 值存在显著差异。弛豫率较高的对比剂通常具有更高的分子量、更高的蛋白结合率，或两者兼有。除肝脏特异性对比剂 Eovist® 外， MultiHance® 在所有通用细胞外对比剂中具有最高的 T1 弛豫率，这使得其在标准剂量下，在 T1 加权图像上检测强化效果最优。MultiHance® 还具有较高的 T2 弛豫率，这意味着它作为灌注研究的磁敏感对比剂也表现良好。

进阶讨论（显示/隐藏）»

此处提供的弛豫率简化处理被称为“快速交换”模型，该模型假设所有对 MR 信号有贡献的质子，在时间尺度上（小于或等于介质的 T1 弛豫时间）都能平等且不受限制地接触到顺磁性中心。显然，在存在多个隔室且交换速率不同（甚至正反向速率也不同）的生物组织中，情况并非如此。尽管如此，这里的简化处理在很大程度上解释了涉及对比剂在组织和溶液中行为的基本现象。
