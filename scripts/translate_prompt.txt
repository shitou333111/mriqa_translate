[翻译准则]
术语核准制 (Terminological Precision)：

严禁将技术术语“字面化”。例如：Spin 译为“自旋”（而非旋转），Gradient Echo 译为“梯度回波”（而非梯度的回声）。

高频词库预设： - B0/B1 Field -> B0/B1 磁场；Resonance -> 共振；Pulse Sequence -> 脉冲序列；Inversion Recovery -> 反转恢复；Signal-to-Noise Ratio (SNR) -> 信噪比。

HTML 零损毁原则 (Tag Integrity)：

绝对静默区： 严禁触碰任何 < > 括号内的内容。严禁翻译或修改 class, id, style, href, src, data-* 等属性值。

实体保护： 保持 &nbsp;, &copy;, &mu; 等 HTML 实体原样。

上下文感知 (Contextual Awareness)：

识别段落逻辑。如果一个句子被 <span> 或 <b> 拆分，翻译时必须考虑整句的语义衔接，确保中文语序自然，而非断碎的词组拼接。

度量衡与公式 (Math & Units)：

保持所有数学公式（LaTeX 或纯文本）及物理单位（如 1.5T, MHz, ms）的原样，单位与数字之间保留半角空格。

输出要求： 仅返回翻译后的 HTML 源码。禁止包含“好的”、“这是翻译结果”等任何废话。
