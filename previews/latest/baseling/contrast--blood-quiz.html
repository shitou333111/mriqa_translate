磁共振对比剂：物理与临床

  1. 钆元素属于周期表中的哪一族？
    1. 镧系
    2. 锕系
    3. 碱土金属
    4. 过渡金属

    钆，原子序数64，位于稀土元素镧系的正中间。 链接至问答讨论

  2. 钆的顺磁性使其适合作为对比剂，这种性质源于其
    1. 奇数个质子
    2. 质子和中子总数之和为奇数
    3. 未配对的內壳层电子
    4. 成键轨道中的成对电子

    尽管所有核子数为奇数的元素都具有核顺磁性（这是核磁共振现象的原因），但除了在原子核附近外，这种顺磁性非常弱。由于电子具有相同的自旋（½），但尺寸比质子小得多，其旋磁比是质子的657倍，因此具有强大的磁矩。正是未配对的內壳层电子赋予了钆顺磁性。 链接至问答讨论

  3. 在4f壳层中，钆具有
    1. 4个未配对电子
    2. 7个未配对电子
    3. 4个成对和3个未配对电子
    4. 7个成对电子

    4f电子壳层有七个可用的位置，每个位置可能被一个孤电子或一个电子对占据。根据洪德最大多重度规则，电子在成对填充之前会先单独占据每个位置。钆有七个电子可用于填充4f壳层，它们全部未配对。 链接至问答讨论

  4. 当制成对比剂时，钆有多少个未配对的4f壳层电子参与成键/螯合？
    1. 0
    2. 1
    3. 4
    4. 7

    钆的4f壳层电子均不直接参与成键，因此在配制成对比剂时仍保留其强大的顺磁性。 链接至问答讨论

  5. 如果钆与一个非常大的分子（如白蛋白）螯合，其T1弛豫时间会发生什么？
    1. 保持不变
    2. 缩短
    3. 延长
    4. 变得比T2短

    较小的水性对比剂（如Gd-DTPA）翻转太快，不能高效地引起T1弛豫。如果对比剂附着在一个非常大的分子（如白蛋白）上，其运动速度减慢到更接近拉莫尔频率的范围，其T1弛豫时间会显著缩短。注意选项(d)不可能，因为T1总是大于或等于T2。 链接至问答讨论

  6. 关于内层和外层弛豫效应的以下哪项陈述是错误的？
    1. 就单个分子而言，内层弛豫比外层弛豫强得多。
    2. 内层弛豫需要水分子非常接近钆金属中心。
    3. 尽管每个分子的作用较弱，但外层效应很重要，因为它们涉及更多的水分子。
    4. 大多数钆对比剂具有多个可用于发生内层弛豫的配位点。

    由于电子结构，Gd+3离子通常表现出九个用于成键和化学相互作用的配位点。在所有市售的磁共振对比剂中，一个配体基团占据其中八个位点，而第九个位点可用于溶剂水分子的瞬时键合，从而发生内层效应。因此选项(d)是错误的。 链接至问答讨论

  7. 对比剂的T1和T2弛豫率以什么单位给出？
    1. L/mmol-s
    2. mmol/s
    3. L/mmol
    4. 无量纲

    磁共振对比剂的弛豫率反映了溶液的弛豫速率如何随浓度[C]变化。由于对比剂可能分别影响两种弛豫速率（1/T1和1/T2），因此存在两个相应的弛豫率，记作r1和r2。根据定义，1/ΔT1 = r1 [C] 和 1/ΔT2 = r2 [C]。由于ΔT1和ΔT2以秒为单位，[C]以毫摩尔/升测量，因此r1和r2的单位为L/mmol-s。 链接至问答讨论

  8. 在以下哪种情况下，预计不会出现钆的T2缩短效应？
    1. 膀胱磁共振
    2. 磁共振关节造影
    3. 心脏磁共振晚期增强期
    4. 磁共振静脉成像

    在常规磁共振成像中，低剂量钆引起的轻微T2缩短效应（降低信号强度）通常被更占优势的T1缩短效应（增加信号强度）所掩盖，因此通常观察不到。当存在高浓度对比剂时，低T2效应可能占主导。这常见于尿路磁共振成像，偶尔见于磁共振关节造影和静脉造影（高浓度钆直接注射到关节或血管中）；以及在动态对比增强研究或时间分辨磁共振血管成像中钆对比剂“首过”血管系统时。在心肌晚期成像中不会出现。 链接至问答讨论

  9. 关于钆增强与场强的关系，哪项陈述是正确的？
    1. 所有当前使用的对比剂都表现出T1弛豫率随场强增加而略有增加。
    2. 具有较低T1弛豫率（r1）值的对比剂产生更强的增强。
    3. 相同剂量的对比剂通常在1.5T下比0.5T下产生更多的增强。
    4. 与高场相比，低场下需要更低剂量的对比剂。

    只有(c)是正确的。因为组织T1值随场强增加而增加，而对比剂弛豫率大致相同或仅略有下降，净效应是随着场强增加，相对对比增强增加。 链接至问答讨论

  10. 为什么磁化传递准备脉冲能提高钆增强的可见性？
    1. 不能。这是一个误解。
    2. 它们抑制了背景组织的信号。
    3. 它们缩短了组织T1。
    4. 它们干扰了钆-水的内层弛豫。

    由于钆增强是由水-钆离子相互作用（而非大分子交叉弛豫）引起的，磁化传递脉冲抑制了背景组织的信号，使钆增强区域更加突出。 链接至问答讨论

  11. 第一个商业化的钆基磁共振对比剂是
    1. Dotarem®
    2. Magnevist®
    3. MultiHance®
    4. Omniscan®

    Magnevist®，由Berlex Imaging（后被拜耳收购）销售，于1988年首次获得FDA批准。 链接至问答讨论

  12. 以下哪种对比剂是大环状的？
    1. Dotarem®
    2. Magnevist®
    3. MultiHance®
    4. Omniscan®

    Dotarem®，以及ProHance®和Gadavist®，都是大环状的。其他的是线状结构。 链接至问答讨论

  13. 关于钆塞酸对比剂（Eovist®/Primovist®），哪项陈述是错误的？
    1. 注射后约10%的患者出现一过性呼吸困难。
    2. 至少90%的给药剂量被肝细胞摄取。
    3. 通过胆道排泄和肾小球滤过双重途径消除。
    4. 胆红素水平升高时，肝脏摄取和增强减弱。

    只有大约一半的给药剂量被肝细胞摄取。其余部分的行为与其他非特异性细胞外钆对比剂相同。 链接至问答讨论

  14. 虽然已不再生产，但20世纪90年代和21世纪初销售的肝脏对比剂Teslascan™的独特之处是什么？
    1. 它基于锰。
    2. 它基于铁。
    3. 它基于铜。
    4. 它基于镁。

    迄今为止唯一获准在全球销售和临床使用的锰基对比剂是锰福地匹三钠（Mn-DPDP），在20世纪90年代以Teslascan™商品名销售。由于销量低、临床性能差以及对毒性的担忧，Teslascan™于2003年退出美国市场，2010年退出欧盟市场。 链接至问答讨论

  15. 谈到不再生产的磁共振对比剂，Ablavar®的独特之处是什么？
    1. 它基于锰。
    2. 它基于氧化铁。
    3. 它附着于白蛋白，作为血池对比剂。
    4. 它被网状内皮系统摄取。

    Ablavar®因其销量不佳于2017年被制造商撤市，该药专为血管成像设计。注射到血液中后，该药物与血浆蛋白（主要是白蛋白）发生可逆结合，使其能在血管内滞留数小时。 链接至问答讨论

  16. 关于超顺磁性氧化铁和超小超顺磁性氧化铁，哪项陈述是错误的？
    1. 超小超顺磁性氧化铁可通过血管内、皮内或皮下注射。
    2. 较大的超顺磁性氧化铁对淋巴结成像特别有用。
    3. 它们的T1和T2弛豫率是钆对比剂的10-20倍。
    4. 它们在肝脏、脾脏和骨髓中的蓄积可能被误认为是血色素沉着症。

    选项(b)是错误的。这些超顺磁性氧化铁纳米颗粒中最大的（50-100 nm）会被肝脏快速摄取。较小的超小超顺磁性氧化铁颗粒（5-50 nm）留在循环中，通过淋巴系统清除，引流到淋巴结，在那里被巨噬细胞/组织细胞（以及可能的其他窦内皮细胞）捕获和滞留。 链接至问答讨论

  17. 关于ferumoxytol（Feraheme®/Rienso®），哪项陈述是错误的？
    1. 它目前在全世界作为磁共振对比剂销售和推广。
    2. 静脉注射后具有较长的血管内半衰期，可作为血池对比剂。
    3. 它缩短了其蓄积组织的T1和T2弛豫时间。
    4. 必须稀释后缓慢静脉注射。

    尽管ferumoxytol最初是作为磁共振对比剂开发的，但它后来被推广并批准用于静脉治疗与慢性肾病相关的缺铁性贫血。它尚未获得作为磁共振对比剂的适应症。以此形式使用被认为是“超说明书用药”。 链接至问答讨论

  18. 关于肠道对比剂，像VoLumen®这样的大容量水基溶液属于哪类对比剂？
    1. 阳性
    2. 阴性
    3. 双相
    4. 中性

    双相对比剂都是水基的，在T1加权图像上呈暗色，在T2加权图像上呈亮色。像VoLumen®这样的双相肠道对比剂含有不可吸收的添加剂（树脂、聚乙二醇、甘露醇、山梨醇或其他糖醇），有助于吸引和保留肠腔内的水。 链接至问答讨论

  19. 关于肠道对比剂，像GastroMARK®（ferumoxsil）这样的磁铁矿颗粒悬浮液属于哪类对比剂？
    1. 阳性
    2. 阴性
    3. 双相
    4. 中性

    阴性对比剂旨在使肠腔内容物变暗，尤其是在T2加权图像上。GastroMARK®在T1和T2加权图像上都会导致肠腔内信号显著下降。T1加权图像上的变暗使得黏膜增强得以显示。 链接至问答讨论

磁共振对比剂：安全性

  1. 与非离子型碘对比剂相比，钆基对比剂的急性不良反应发生率
    1. 大致相同
    2. 略低
    3. 略高
    4. 高得多

    与许多其他药物相比，钆对比剂的急性不良反应发生率非常低（<2.5%），且大多为轻微反应。这些发生率并不比注射生理盐水的安慰剂高太多，大约是非离子型碘对比剂反应率的三分之一。 链接至问答讨论

  2. 总体而言，钆基对比剂预计的严重过敏反应（如面部水肿、支气管痉挛、低血压、心律失常、广泛荨麻疹）发生率大约是多少？
    1. 1/500 患者

    4. 1/500,000 患者

  3. 总体而言，钆基对比剂导致严重过敏反应死亡的风险大约是多少？
    1. 1/4,000 患者
    2. 1/40,000 患者
    3. 1/400,000 患者
    4. 1/4,000,000 患者

  4. 以下哪种钆的罕见反应尚未被报道？
    1. 横纹肌溶解症
    2. 胰腺炎
    3. 脑病
    4. 急性肾衰竭

    据我所知，钆给药后从未发生过急性横纹肌溶解症。其他三种情况有罕见报道。 链接至问答讨论

  5. 比较对钆对比剂的“真性”过敏反应与“类过敏”反应，哪项陈述是正确的？
    1. 可以根据临床表现区分。
    2. 两者均为IgE介导。
    3. 两者均导致肥大细胞释放血管活性胺和炎症介质。
    4. 两者均与剂量相关。

    “真性”过敏反应是IgE介导的I型超敏反应，与剂量无关。类过敏反应与“真性”过敏反应具有相同的临床表现，但非IgE介导。两者在临床上无法区分。两者对嗜碱性粒细胞和肥大细胞的作用相同，均可释放组胺和其他物质。 链接至问答讨论

  6. 关于肾源性系统性纤维化的临床特征，哪项陈述不正确？
    1. 几乎所有报道的病例均发生在有严重肾病或肾功能不全的患者中。
    2. 皮肤增厚和关节挛缩是其特点。
    3. 肝脏、肺、肌肉和心脏也可能受累。
    4. 尽管疼痛且致畸，但该病很少致命。

    肾源性系统性纤维化是一种罕见的进行性疾病，通常致命，其特征为皮肤增厚、疼痛性关节挛缩以及包括肺、肝、肌肉和心脏在内的多器官纤维化。几乎所有记录在案的病例都发生在接受过钆对比剂的慢性严重肾功能不全患者中。 链接至问答讨论

  7. 以下哪种钆基对比剂与其他相比，与肾源性系统性纤维化相关的可能性较小？
    1. Magnevist®
    2. Dotarem®
    3. OpitMARK™
    4. Omniscan™

    除了大环状对比剂Dotarem®，其他三种均为线状结构，且与报告肾源性系统性纤维化病例数最多相关。它们被美国放射学会归类为I组对比剂，已不再在美国推广，并在许多其他国家撤市。 链接至问答讨论

  8. 迄今为止，哪种钆基对比剂尚未与任何非混杂因素的肾源性系统性纤维化病例相关联？
    1. Dotarem®
    2. ProHance®
    3. MultiHance®
    4. Eovist®

    最新的钆剂Eovist®（钆塞酸）尚无肾源性系统性纤维化病例报道。然而，Eovist®主要用于肝脏成像，因此与更通用的对比剂相比，全球使用经验有限。此外，Eovist®（以及任何未来的新对比剂）是在“肾源性系统性纤维化意识”时代使用的，此时避免使用高剂量钆，并在对比剂给药前仔细评估肾功能。 链接至问答讨论

  9. 美国肾脏基金会认为肾小球滤过率低于多少为3期（中度功能减退）肾病？
    1. 90 mL/min/1.73 m²
    2. 60 mL/min/1.73 m²
    3. 30 mL/min/1.73 m²
    4. 15 mL/min/1.73 m²

    3期慢性肾病通常在肾小球滤过率为30-60 mL/min/1.73 m²时诊断。这大约是正常肾功能水平90 mL/min/1.73 m²的一半。 链接至问答讨论

  10. 关于新生儿和婴儿使用钆对比剂的以下哪项陈述是错误的？
    1. 正常新生儿体内钆对比剂的半衰期至少是成人的4倍。
    2. 每公斤体重的对比剂剂量应低于成人。
    3. 新生儿对钆对比剂的肾脏排泄率低于成人。
    4. 尚无新生儿或婴儿发生肾源性系统性纤维化的报道。

    除肾脏排泄率外，新生儿细胞外液量是其体重的两倍（相对于成人）。因此，按每公斤体重剂量接受钆对比剂的新生儿和婴儿，在平衡后的血液钆浓度仅为成人的一半。这一事实反对在婴儿中使用比成人更低的每公斤剂量，即使血清半衰期延长。我们小组和其他人的临床经验表明，成人的钆对比剂剂量（大多数细胞外对比剂为0.1 mmol/kg）也适用于婴儿和儿童。 链接至问答讨论

  11. 关于对哺乳期妇女使用钆对比剂，哪项陈述是错误的？
    1. 乳汁中的峰值排泄发生在注射后约50分钟。
    2. 仅有极小部分（0.01%-0.04%）的给药剂量通过乳汁排泄。
    3. 婴儿口服摄入的钆对比剂中只有1%被吸收。
    4. 当前科学证据表明，必须避免母乳喂养至少24小时。

    由于(b)和(c)正确，进入婴儿循环的剂量极低（母体剂量的1/1,000,000），潜在的毒性作用被认为微乎其微。许多中心出于安全考虑，建议（但不要求）在钆增强磁共振检查后暂停母乳喂养24小时。 链接至问答讨论

  12. 关于妊娠期使用钆对比剂，哪项陈述是错误的？
    1. 在妊娠期接受钆对比剂的母亲所生婴儿中，观察到先天性畸形数量增加。
    2. 钆对比剂容易通过胎盘进入胎儿循环。
    3. 被胎儿吸收的钆对比剂通过尿液排泄，随后进入羊水。
    4. 羊水中的部分钆对比剂会被胎儿吞咽并通过胎儿胃肠道再吸收。

    在迄今为止唯一一项关于妊娠期磁共振和钆暴露的对照人体研究中，观察到钆暴露组死产和早期新生儿死亡的发生率略高（但先天性畸形未增加）。然而，更令人担忧的是风湿病、炎症性和浸润性皮肤病的数量增加，其中一些类似于肾源性系统性纤维化。 链接至问答讨论

  13. 关于钆在脑内沉积的以下哪项陈述是错误的？
    1. 沉积主要发生在基底节、脑干和齿状核。
    2. 通常需要多次暴露才能看到可见的变化。
    3. 变化在T2*加权图像上最明显。
    4. 大环状对比剂的沉积发生率低于线状对比剂。

    除(b)外，其他均为正确。由于钆沉积导致的高信号在T1加权图像上可见。 链接至问答讨论

  14. 关于“钆沉积病”，哪项陈述是错误的？
    1. 美国食品药品监督管理局认为目前尚无令人信服的证据支持该诊断。

    3. 在钆积聚的脑区，显微镜下未检测到组织损伤或形态学改变。
    4. 名人查克·诺里斯撤回了对Bracco的诉讼，他在诉讼中声称该公司用钆“毒害”了他的妻子。

顺磁性血液产物

  1. 一个血红蛋白分子中有多少个铁原子？
    1. 1
    2. 2
    3. 3
    4. 4

    血红蛋白由四个亚基组成，每个亚基包含一个血红素基团，中心有一个铁原子。 链接至问答讨论

  2. 每个血红素基团上有多少个位点可用于结合氧或其他小分子？
    1. 1
    2. 2
    3. 3
    4. 4

    每个血红素基团的铁中心有6个潜在的配位位点，其中4个被卟啉氮占据。第五个配位位点（在环平面下方）与其各自珠蛋白链F8位置的组氨酸残基共价键合。这就留下了一个自由的配位位点，氧和其他小分子可以暂时结合于此。 链接至问答讨论

  3. 下列血红蛋白衍生物在血肿老化过程中出现的顺序是：
    1. 氧合血红蛋白 → 脱氧血红蛋白 → 正铁血红蛋白 → 铁蛋白
    2. 脱氧血红蛋白 → 氧合血红蛋白 → 正铁血红蛋白 → 铁蛋白
    3. 氧合血红蛋白 → 正铁血红蛋白 → 脱氧血红蛋白 → 铁蛋白
    4. 氧合血红蛋白 → 脱氧血红蛋白 → 铁蛋白 → 正铁血红蛋白

    答案(b)显示了正确的顺序。 链接至问答讨论

  4. 下列哪种血红蛋白衍生物是抗磁性的？
    1. 氧合血红蛋白
    2. 脱氧血红蛋白
    3. 正铁血红蛋白
    4. 铁蛋白

    氧合血红蛋白没有未配对电子，因此是抗磁性的。 链接至问答讨论

  5. 在健康患者的动脉血中，脱氧血红蛋白的预期比例是多少？
    1. 0 %
    2. 小于5%
    3. 10 – 20 %
    4. 大于25 %

    在动脉血中，氧合血红蛋白的比例约为95%，而脱氧血红蛋白占不到5%（但不为零）。也可能检测到微量的正铁血红蛋白（<< 1%）。 链接至问答讨论

  6. 在健康患者的静脉血中，脱氧血红蛋白的预期比例是多少？
    1. 0 %
    2. 小于5%
    3. 10 – 20 %
    4. 大于25 %

    在静脉血中，氧合血红蛋白仍然占主导，但比例较小（静脉氧合:脱氧 ≈ 70:30）。 链接至问答讨论

  7. 在血液凝固过程中，局部红细胞压积增加导致的MR信号效应中，哪一项不是由该原因引起的？
    1. 自由水含量降低，导致T1、T2和自旋密度降低。
    2. 细胞外空间减少，限制扩散。
    3. 红细胞膜中磷脂浓度增加，缩短T1。
    4. 血红蛋白浓度增加，在转变为脱氧血红蛋白时增强T2/T2*效应。

    除(c)外，其他均为正确。磷脂细胞膜具有超短T2值，在磁共振上不产生脂肪信号。 链接至问答讨论

  8. 在1.5T下，以下哪项成像表现不会出现在超急性期（<12小时）脑血肿中？
    1. 扩散受限
    2. T1信号与脑实质等信号
    3. T2信号中度低于脑实质
    4. 至少在GRE/SWI图像上可见低信号边缘

    在超急性期血肿中，氧合血红蛋白仍占主导（除了可能在边缘），影像表现提示为含水量相对较高的病变，在T2加权图像上呈中度高信号。 链接至问答讨论

  9. 在1.5T下，以下哪项成像表现不会出现在急性期（12-48小时）脑血肿中？
    1. 扩散受限
    2. T1信号高于脑实质
    3. T2信号显著低于脑实质
    4. GRE/SWI图像信号低于脑实质

    血肿形成的这一阶段以细胞内脱氧血红蛋白为主，它显著缩短T2/T2*。血肿在T2加权图像上呈显著低信号。 链接至问答讨论

  10. 在1.5T下，以下哪项成像表现不会出现在早期亚急性期（2天-1周）脑血肿中？
    1. 扩散受限
    2. T1信号高于脑实质
    3. T2信号高于脑实质
    4. GRE/SWI图像信号低于脑实质

    这是血肿形成阶段，细胞内正铁血红蛋白占主导。这种血红蛋白衍生物具有很强的顺磁性。由于它仍包含在红细胞内，T2/T2*缩短效应仍然存在，因此T2信号较低，而非较高。由于珠蛋白部分构象改变，水分子接近铁中心，导致T1缩短和T1信号增高。 链接至问答讨论

  11. 在1.5T下，以下哪项成像表现不会出现在晚期亚急性期（1周-2个月）脑血肿中？
    1. 无或轻微扩散受限
    2. T1信号低于脑实质
    3. T2信号高于脑实质
    4. GRE/SWI图像显示中心高信号

    血肿形成的这一阶段以细胞外正铁血红蛋白为主，它从破裂的红细胞中释放出来。正铁血红蛋白不再浓缩于多个细胞内隔室，而是均匀分散在整个血肿腔中，在T1和T2加权图像上均为高信号。 链接至问答讨论

  12. 在1.5T下，以下哪项成像表现不会出现在慢性期（>2个月）脑血肿的中心？
    1. 无或轻微扩散受限
    2. T1信号低于脑实质
    3. T2信号高于脑实质
    4. GRE/SWI中心信号低于脑实质

    慢性血肿中心的影像特征反映其高含水量。因此，该区域在T2加权图像上通常高于脑实质，在T1加权图像上低于脑实质，无或轻微扩散受限。GRE/SWI中心信号也高，但由于铁蛋白/含铁血黄素沉积，血肿腔边缘可能呈暗色。 链接至问答讨论

  13. 关于铁蛋白的以下哪项陈述是不正确的？
    1. 它是动物细胞中发现的主要铁储存分子。
    2. 它由一个中空的蛋白质外壳组成，铁原子填充其中。
    3. 一个典型的铁蛋白分子含有约2000个铁原子。
    4. 它是铁磁性的。

    铁蛋白核心表现为具有单个磁畴的固体金属块，表现出超顺磁性。相比之下，铁磁性材料由多个磁畴组成，并在外部磁场移除后保留一些剩余磁化强度。 链接至问答讨论

  14. 关于铁蛋白和含铁血黄素，哪项陈述是正确的？
    1. 含铁血黄素中的铁不溶；铁蛋白中的铁可溶。
    2. 两者均可用光学显微镜观察到。
    3. 铁蛋白是顺磁性的，但含铁血黄素是铁磁性的。
    4. 含铁血黄素和铁蛋白都具有固定的结构和组成。

    只有(a)是正确的。铁蛋白比含铁血黄素小得多，只能在电子显微镜下观察到。两者均为顺磁性。铁蛋白具有固定的结构和组成；含铁血黄素是铁蛋白颗粒、蛋白质和脂质的无定形聚集体。 链接至问答讨论

  15. 与脑实质内出血相比，以下哪种过程不影响蛛网膜下腔出血的表现？
    1. 蛛网膜下腔出血的血凝块小。
    2. 蛛网膜下腔出血的血凝块容易弥散。
    3. 蛛网膜下腔出血与脑脊液混合。
    4. 蛛网膜下腔的环境氧水平较低，导致血肿老化更快。

    只有(d)是错误的。蛛网膜下腔的环境氧水平较高，导致血肿老化较慢。例如，正铁血红蛋白形成通常在发作后至少1周才出现。 链接至问答讨论

  16. 关于蛛网膜下腔出血的CT和MRI，哪项陈述是错误的？

    3. 亚急性蛛网膜下腔出血在MRI上可能比CT上显示得更好。

  17. 关于极低场强（<0.15 T）下出血表现的以下哪项陈述是正确的？
    1. 细胞内脱氧血红蛋白在T2加权图像上明显呈暗色。
    2. 细胞内正铁血红蛋白在T2加权图像上明显呈暗色。
    3. 细胞外正铁血红蛋白在T1加权图像上呈亮色。
    4. 超急性期血肿中心在T1加权图像上呈暗色。

    只有(c)是正确的。细胞内脱氧血红蛋白和正铁血红蛋白在高场下在T2加权图像上呈暗色，但在低场下则不然。这是因为散相是由磁敏感性引起的，而磁敏感性与磁场平方成正比。急性期血肿中心由于血液浓缩和蛋白质效应，在T1加权图像上呈亮色。 链接至问答讨论
