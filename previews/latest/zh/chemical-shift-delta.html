化学位移

什么是化学位移？  
化学位移 指的是处于不同分子环境中的两个其他方面完全相同的原子核，其共振频率之间的微小差异。之所以产生这种频率差异，是因为特定原子核的共振频率并非由外部施加的磁场强度 (Bo) 决定，而是由原子核在原子层面所感受到的局部磁场 (Bloc) 决定。

因此，并非所有原子核都以完全相同的频率共振。每个原子核的确切频率既取决于其所处的特定分子，也取决于其在该分子上的具体位置。对于 ¹H 原子核而言，化学位移相对较小（在 1.5T 场强下小于几百赫兹），但仍然可以检测到。化学位移可以提供关于被成像组织分子组成的重要信息，并构成了 磁共振波谱 (MRS) 的基础。 
当置于外部磁场 (Bo) 中时，原子核周围电子云的响应会产生一个感应磁场 (Bind)，该磁场通常与施加的磁场方向相反。这被称为“屏蔽”。原子核的共振频率取决于 Bloc = Bo − Bind。
化学位移源于原子/分子层面的 抗磁化率 效应。磁化率是物质对外部磁场的响应，可能由几种不同的机制产生（在 之前的问答 中已有描述）。理解此现象最简单的方法是想象围绕原子核运动的电子产生了一个“感应”磁场 (Bind)，该磁场与 Bo 方向相反。因此，电子云充当了一个“屏蔽层”，保护原子核免受 Bo 的全部作用力。屏蔽因子 通常用符号 σ 表示，其中 Bind = σBo。因此，原子核所感受到的局部有效磁场是 Bloc，其中
Bloc = Bo − Bind = Bo (1 − σ)
较早的文献有时也将 σ 称为“化学位移”。这种术语造成了一些混淆，因为 Bind 和 Bloc 无法直接测量。实际上，化学家测量的是待测样品与外部参考标准之间 NMR 频率的差异。2001 年，国际纯粹与应用化学联合会 (IUPAC) 建议放弃使用 σ ，转而采用 δ 标度，并将 化学位移 (δ) 定义为
δ = (fsamp − fref ) / fref
其中 fsamp 是所研究核种的 NMR 频率，fref 是参考化合物的 NMR 频率。

频率差 (fsamp − fref ) 通常在几百到几千赫兹的量级，而参考频率 (fref ) 则以兆赫兹 (MHz) 为单位测量。因此，化学位移 (δ) 是一个很小的数值，以 百万分之一 (ppm) 为单位表示。后缀 ppm 可与 x10 −6 互换，正如符号 % 可与 x0.01 或 x10−2 互换一样。

化学位移 (δ) 是一个无量纲常数，对于每种类型的原子核（¹H、³¹P 等）和化学构型（—CH3、—OH、--PO4 等）都是特定的。化学位移值表很容易获取（现已嵌入计算机/智能手机应用程序中），使化学家能够在 MRS 实验中快速识别检测到的原子核。

使用 ppm 单位可以独立于场强来描述化学位移。考虑两个化学位移相差 1.0 ppm 的原子核。在 1.5T 场强下（拉莫尔参考频率约为 64 MHz），这两个原子核的绝对频率差将为 64 MHz x 1 ppm = 64 Hz。在 3.0T 场强下（拉莫尔频率为 128 MHz），两者的频率差将为 128 MHz x 1 ppm = 128 Hz。
