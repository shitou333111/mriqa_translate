接收带宽

什么是带宽？ 
带宽 (BW) 是指电子信号在 传输或接收过程中所涉及的频率范围（以 Hz 为单位）。 在 MRI 中，该术语可用于指与射频激励相关的频率（发射带宽, tBW）或信号接收相关的频率（接收带宽, rBW）。未特别说明时，通用术语“带宽”通常指接收带宽，这也是本页的主题。关于射频带宽的概念将在后续的问答中讨论。
回想一下，图像的一个维度通常是通过在该方向上施加一个空间变化的梯度（Gf）来进行频率编码的。如相邻图像所示，该梯度会轻微改变进动频率。在此示例中，图像最左侧的体素共振频率为 64,000,000 Hz，而最右侧的体素共振频率为 64,050,000 Hz。因此，总接收带宽 rBW 为 50,000 Hz。该值是一个操作员可选的参数，由技师在扫描开始前选择。总接收带宽的可用值范围约为 5-100 kHz，50 kHz 是典型值。

这个总带宽被平均分配给沿频率编码方向的像素。反过来，每个像素的宽度（Δw）由另外两个操作员选择的参数决定：频率编码方向上的视野  (FOVf) 和 频率编码步数 (Nf)。 
每像素带宽 = 总带宽 ÷ Nf
不同 MRI 制造商定义接收带宽的方法略有不同。GE Healthcare 使用整个图像的总带宽，而西门子和佳能使用每像素带宽 (Px)。在上面的例子中，无论选择的空间分辨率如何，GE 都会报告 rBW = 50 kHz。西门子和佳能则会按像素计算带宽。因此，对于 Nf = 256，他们会报告带宽为 50,000/256 或 195 Hz/Px。 

飞利浦有一种略显晦涩的带宽设定方式——以像素为单位的“脂肪/水位移”。在 1.5T 场强下，脂肪和水质子的共振频率相差约 220 Hz，导致它们在图像中重聚的位置略有不同。这被称为化学位移伪影，将在未来的问答中更详细地解释。继续上面的例子，在 1.5T 下，195 Hz/像素的带宽，飞利浦会报告为 220/195 = 1.1 像素 (Px)。在 3.0T 下，化学位移差约为 440 Hz，因此飞利浦在此场强下报告的带宽将为 2.2 像素。 
在“基本原理”层面，接收带宽等同于 MR 信号的数字化速率。驻留时间 (td) 是数字化采样点之间的时间间隔。这又由采样时间 (ts) 和所测量的复采样点数 (ns) 定义。 

继续上面的例子，如果在 5.12 ms 内采集了 256 个采样点，则驻留时间 (td) 为 5.12 ms/256 = 20 μs。这将导致总接收带宽为 1/20μs = 50,000 Hz。

进阶讨论 (显示/隐藏)»

你可能想知道，为什么我们定义的 BW=1/td 与采样理论中的奈奎斯特准则不冲突。奈奎斯特准则要求一个正弦波每个周期至少采样两次才能确定其真实频率。因此，似乎正确的公式应该是 BW=1/2td。

答案是，在 MRI 中我们以复模式采样，为每个点收集实部和虚部数据。如果我们只采样单个实信号（就像在一些较旧的 NMR 谱仪和非常古老的 MRI 扫描仪中所做的那样），那么真正的带宽将仅为 1/2td。我们将能够检测到 −1/2td 到 +1/2td 范围内的信号，但无法区分该范围内的正负频率。我们实际能表示的带宽仅为 0 到 1/2td。

同时收集实部和虚部数据使我们能够区分 +1/2td 和 −1/2td。因此，完全可用的带宽为 BW = +1/2td − (−1/2td) = 1/td。最大可检测频率仍然是 1/2td，但我们现在可以区分该范围内的正负频率。

上述讨论中忽略的另一个主题是频率过采样的概念。为简化起见，我没有严格区分 Nf（频率编码方向上显示的像素数）和 ns（实际采样的信号中的复数据点数）。现代扫描仪上可用的高数字化速率使得 MR 信号的采样速率可以远高于所需显示分辨率的要求。通常，ns 可能超过 Nf 两倍或更多。因此，即使频率编码方向上的基本显示分辨率可能仍仅为 Nf = 256，采样 512 或 1024 个 MR 信号点是可能的（并且很常见）。这被称为频率过采样，对于消除频率编码方向上的混叠（卷折）伪影非常有用。常规的频率过采样是为什么我们通常只在相位编码方向看到这种伪影，而在频率编码方向看不到的原因。更多内容将在后续问答中详述！
