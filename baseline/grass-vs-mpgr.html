GRASS 与 MPGR

单层 GRASS 与多层 GRASS (MPGR) 有何区别？  
单层模式（简单 GRASS 或 FISP）

当采用非常短的 TR 值时，通常一次只能获取单层 GRASS/FISP 图像。因此，为了覆盖大面积区域，必须逐一采集一系列独立的 GRASS/FISP 图像。这种技术被称为 顺序多层采集。

在顺序多层模式下，每个层面都是独立采集的，其信号特性是 TR、TE 和 α 的函数，正如 之前的问答 中针对理想化 GRASS/FISP 序列所描述的那样。此外，就流动效应而言，每个层面都可以被视为一个“流入层面”。因此，当使用这种采集模式时，血液或脑脊液的流入增强效应达到最大。
多层模式（多层 FISP，MPGR）

当采用较长的 TR 值时，可以在单个 TR 间隔内同时采集多个层面。这种方法被称为 层面交错 或 层面复用采集。

然而，在层面复用模式下，会出现显著不同的对比度行为。因为必须在每个 TR 间隔内选择多个层面，所以必须为每个成像层面开启层面选择梯度。这些层面选择梯度会破坏横向相干性，并将 GRASS/FISP 序列转变为显示破坏性梯度回波对比度。同样在此模式下，由于只有首尾层面是真正的“流入层面”，流入增强效应不那么明显。
2D 与 3D 实现方式
GRASS/FISP 以及大多数其他脉冲序列，都可以在 2D 或 3D 采集模式下运行。图像对比度的许多方面会因选择 2D 或 3D 而有所不同。
一个立即显而易见的效果是血管的表现。在 2D 单层模式下，GRASS/FISP 序列通常显示出流入每个层面的血液和脊髓液的显著流动相关增强。这种流入效应在 2D 多层模式下有所减弱，在 3D 采集中则进一步降低。 

整个 3D 容积内不同部分的整体图像对比度也可能有所不同。这是由于难以在整个容积内获得均匀的射频翻转角。随着翻转角的变化，图像对比度也会变化。
