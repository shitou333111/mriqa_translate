MR信号

  1. 单个射频脉冲产生的MR信号称为
    1. 自由感应衰减
    2. 自旋回波
    3. 梯度回波
    4. 受激回波

    自由感应衰减由单个射频脉冲产生，此时被倾倒的核磁化矢量围绕Bo进动，并通过感应（法拉第-楞次定律）在接收线圈中产生电流。 链接至问答讨论

  2. 关于自由感应衰减，以下哪项是正确的？
    1. 它以共振频率无限振荡
    2. 它以时间常数T2*衰减
    3. 它以时间常数T1增加幅度
    4. 由完美的180°脉冲产生

    自由感应衰减是以共振频率进行的阻尼正弦振荡，以时间常数T2*衰减。 链接至问答讨论

  3. 如果在射频脉冲之后立即开启磁场梯度，MR信号会发生什么？
    1. 变成自由感应衰减
    2. 保持不变
    3. 迅速消失
    4. 被重聚焦成梯度回波

    初始的MR信号已经等同于自由感应衰减（选项a是错误的）。在自由感应衰减期间开启梯度会导致其衰减速度比仅由T2*过程引起的更快（选项c是正确的）。需要后续施加一个极性相反的梯度才能产生梯度回波。 链接至问答讨论

  4. 梯度回波的另一个名称是
    1. 场回波
    2. 射频回波
    3. 自旋回波
    4. 受激回波

    较少使用的术语“场回波”是正确的答案。一些制造商，包括飞利浦和佳能，使用FE或FFE（快速场回波）来命名其梯度回波序列。另一个同义词是梯度召回回波。 链接至问答讨论

  5. 如果在自由感应衰减自发衰减后开启梯度，会发生什么？
    1. 产生梯度召回回波。
    2. 产生自旋回波。
    3. 部分再生自由感应衰减。
    4. 什么也不会发生。

    一旦自由感应衰减通过T2*过程自发衰减，就无法通过梯度作用恢复任何信号。 链接至问答讨论

  6. 简单梯度回波成像序列中施加的第一个梯度瓣称为
    1. 读出梯度。
    2. 散相梯度。
    3. 重聚焦梯度。
    4. 双极梯度。

    梯度回波序列的第一步是使自由感应衰减散相，因此该瓣被称为“散相梯度”。散相梯度导致除正常T2*过程之外的加速散相，使自由感应衰减“压制/混乱”。 链接至问答讨论

  7. 重聚焦梯度重聚焦由以下因素散相的自旋
    1. 仅散相梯度
    2. 仅T2
    3. 仅T2*
    4. 散相梯度、T2和T2*

    重聚焦梯度仅重聚焦由散相梯度本身引起的自旋混乱。T2和T2*过程不受影响，并导致持续的信号损失 链接至问答讨论

  8. 自旋回波最早由谁描述？
    1. 费利克斯·布洛赫
    2. 爱德华·珀塞尔
    3. 伊西多·拉比
    4. 欧文·哈恩

    欧文·哈恩于1949年在读研究生时发现了自旋回波。 链接至问答讨论

  9. 单个自旋回波由以下产生
    1. 单个射频脉冲
    2. 单个射频脉冲加梯度反转
    3. 两个射频脉冲
    4. 三个射频脉冲

    单个射频脉冲产生自由感应衰减，但两个连续的射频脉冲产生自旋回波。 链接至问答讨论

  10. 在自旋回波脉冲序列中，第一个射频脉冲中点与自旋回波峰值之间的时间称为
    1. TR
    2. TE
    3. TE/2
    4. TI

    第一个射频脉冲中点与自旋回波峰值之间的时间称为回波时间（TE）。 链接至问答讨论

  11. 在自旋回波脉冲序列中，第一个和第二个射频脉冲之间的时间是
    1. TR
    2. TE
    3. TE/2
    4. TI

    自旋回波出现在时间TE，它是脉冲间隔的两倍。所以正确答案是TE/2 链接至问答讨论

  12. 用于成像的自旋回波通常使用90°脉冲后跟180°脉冲产生。如果将第二个脉冲改为90°，会发生什么？
    1. 不会产生回波。
    2. 在时间2×TE处产生一个正常大小的回波
    3. 在时间TE处产生一个大小减半的回波
    4. 在时间2×TE处产生一个大小减半的回波

    欧文·哈恩最初使用的是90°-90°射频对，由此产生的回波通常称为哈恩回波。任意两个射频脉冲在时间TE处都会形成哈恩回波，其幅度取决于所选择的角度。90°-90°对的最大自旋回波强度仅为90°-180°对产生的一半。 链接至问答讨论

  13. 以下哪对自旋回波脉冲预计会产生最大的信号？
    1. 90°-90°
    2. 90°-180°
    3. 180°-90°
    4. 180°-180°

    90°-180°对最常用的原因是这种组合产生最大可能的回波信号。90°-90°对的最大自旋回波强度仅为90°-180°对产生的一半。一般来说，如果第一个射频脉冲的翻转角为α1，第二个为α2，则哈恩回波的最大信号强度比常规（90°-180°）回波小一个因子(sin α1)•(sin² α2/2)。 链接至问答讨论

  14. 一系列3个紧密间隔的连续射频脉冲可以产生多少个回波？
    1. 3
    2. 4
    3. 5
    4. 6

    三个射频脉冲最多可以产生5个自旋回波：3个主回波，1个次级回波和1个受激回波。 链接至问答讨论

  15. 在这些回波中，有多少被认为是“受激”回波？
    1. 1
    2. 2
    3. 没有
    4. 全部都是

    3个射频脉冲只产生1个受激回波。 链接至问答讨论

  16. 置于静磁场中的样品产生初始纵向磁化强度Mo。然后对其施加一系列等间隔为TR的射频脉冲。如果TR>>T1，关于每个后续射频脉冲之前的纵向磁化强度Mz，以下哪项正确？
    1. 无法预测。
    2. 与Mo相同。
    3. 大于Mo。
    4. 小于Mo。

    如果TR足够长（通常要求TR > 4-5×T1），纵向磁化强度将有时间完全恢复，并在下一个射频脉冲时恢复到其完整的初始磁化强度Mo。 链接至问答讨论

  17. 置于静磁场中的样品产生初始纵向磁化强度Mo。然后对其施加一系列等间隔为TR的射频脉冲。如果T2* < TR < T1，关于每个后续射频脉冲之前的纵向磁化强度Mz，以下哪项正确？
    1. 无法预测。
    2. 与Mo相同。
    3. 大于Mo。
    4. 小于Mo。

    在这种情况下，TR现在与T1相等或更短。第二个及后续脉冲发生在纵向磁化强度恢复到其初始值（Mo）之前。经过几次脉冲后，将建立一个新的稳态纵向磁化强度（Mss），其中Mss < Mo。 链接至问答讨论

  18. 上述情况（问题17）被称为
    1. 部分饱和
    2. 完全饱和
    3. 稳态自由进动
    4. 不完全磁平衡

    这种现象称为部分饱和，意味着自旋系统尚未完全弛豫回其平衡状态。 链接至问答讨论

  19. 置于静磁场中的样品产生初始纵向磁化强度Mo。然后对其施加一系列等间隔为TR的射频脉冲。如果T2* < TR < T1，关于连续射频脉冲之间的中点处的横向磁化强度Mxy，以下哪项正确？
    1. 无法预测。
    2. 与Mo相同。
    3. 小于Mo但不为零。
    4. 为零。

    如问题#17所述，样品部分饱和，达到一个新的非零稳态纵向磁化强度Mss < Mo。但由于TR > T2*，在每对射频脉冲之间的中点处，横向磁化强度完全衰减，因此答案为0（d）。 链接至问答讨论

  20. 与问题#19相同的情景，但这里TRMxy，以下哪项正确？
    1. 无法预测。
    2. 与Mo相同。
    3. 小于Mo但不为零。
    4. 为零。

    在这种情况下，TR非常短，横向磁化强度Mxy在射频脉冲之间没有完全衰减，达到一个小于Mo的非零值。 链接至问答讨论

  21. 上述情况被称为
    1. 部分饱和
    2. 完全饱和
    3. 稳态自由进动
    4. 完全磁平衡

    如果TR变得甚至更短（≤T2*），每个射频脉冲后的自由感应衰减在自旋回波/受激回波成分开始形成之前不会完全消失。自由感应衰减和自旋回波/受激回波的尾部合并在一起。在这种情况下，横向信号从未完全消失，其结果称为稳态自由进动。 链接至问答讨论

  22. 如果样品的初始纵向磁化强度为Mo，那么在90°脉冲之后立即的纵向磁化强度是多少？
    1. Mo
    2. −Mo
    3. ½ Mo
    4. 零

    在理想的90°脉冲之后，纵向磁化强度应为零。 链接至问答讨论

  23. 如果样品的初始纵向磁化强度为Mo，那么在90°脉冲之后立即的横向磁化强度是多少？
    1. Mo
    2. −Mo
    3. ½ Mo
    4. 零

    在理想的90°脉冲之后，所有纵向磁化强度都应转换为横向。 链接至问答讨论

  24. 如果样品的初始纵向磁化强度为Mo，那么在45°脉冲之后立即的纵向（Mz）和横向（Mxy）磁化强度分别是多少？
    1. 约Mo的90%
    2. 约Mo的70%
    3. 约Mo的50%
    4. 约Mo的10%

    从三角恒等式可知，当初始与z轴对齐的磁化矢量翻转角度α时，翻转后的横向和纵向磁化强度分量分别为[Mo sin α]和[Mo cos α]。对于α=45°，sin 45° = cos 45° = 1/√2 ≈ 0.707。因此正确答案是约Mo的70%（b）。 链接至问答讨论

  25. 在没有横向相干性的梯度回波序列中，使组织MR信号最大化的翻转角称为
    1. 哈恩角
    2. 布洛赫角
    3. 珀塞尔角
    4. 恩斯特角

    以理查德·R·恩斯特命名，他因这一发现于1991年获得诺贝尔化学奖。 链接至问答讨论

  26. 恩斯特角（最佳翻转角）何时近似等于90°？
    1. 当TR >> T1时
    2. 当TR ≈ T1时
    3. 当TR << T1时
    4. 从不

    恩斯特角（αE）可由公式αE = arccos (e−TR/T1)计算。仅在TR >> T1时达到最大值90°。 链接至问答讨论

自旋回波与反转恢复

  1. 在自旋回波脉冲序列中，180°脉冲重聚焦由以下因素散相的自旋
    1. 磁场不均匀性
    2. 扩散
    3. T2过程
    4. 以上所有

    180°脉冲可以重聚焦因恒定场畸变和不均匀性而导致相位散乱的静止自旋。该脉冲不能纠正由原子/分子水平随机过程引起的T1或真实T2效应。它也不能纠正运动、流动、扩散或发生化学交换的自旋的相位偏移。 链接至问答讨论

  2. 比较多回波自旋回波和快速（涡轮）自旋回波，以下哪项陈述是错误的？
    1. 多回波自旋回波和快速自旋回波在每个TR间隔内都使用多个180°脉冲。
    2. 多回波自旋回波在每个TR间隔内仅使用一个相位编码梯度。
    3. 快速自旋回波在每个TR间隔内多次施加相位编码梯度。
    4. 在多回波自旋回波和快速自旋回波成像中，回波幅度随连续的180°脉冲而递减。

    在快速自旋回波中，每个180°脉冲都施加不同的相位编码梯度，因此回波幅度可能根据相位编码步骤而变化。在多回波自旋回波中，相位编码梯度在每个TR间隔内仅开启一次，因此后续回波的信号强度会降低。 链接至问答讨论

  3. 以下哪组自旋回波参数选择会产生质子密度加权图像？
    1. TR = 500, TE = 10
    2. TR = 500, TE = 100
    3. TR = 5000, TE = 10
    4. TR = 5000, TE = 100

    答案c)，长TR和短TE用于产生质子密度加权图像。 链接至问答讨论

  4. 使用短TR和长TE的自旋回波序列产生
    1. T1加权图像
    2. T2加权图像
    3. 质子密度加权图像
    4. 噪声大的低对比度图像

    正确答案是d)。因此这种组合很少使用。 链接至问答讨论

  5. 具有长T1和长T2值的组织或病变通常在
    1. T1加权自旋回波图像上呈高信号
    2. T2加权自旋回波图像上呈高信号
    3. T1和T2加权自旋回波图像上均呈高信号
    4. T1和T2加权自旋回波图像上均呈低信号

    此类病变通常在T2加权图像上呈高信号，在T1加权图像上呈低信号。 链接至问答讨论

  6. 通过交替180°脉冲传输的相位，CPMG（Carr-Purcell-Meiboom-Gill）对自旋回波脉冲序列的改进
    1. 减少了由重复的不完美180°脉冲引起的累积相位误差。
    2. 减少了由B1不均匀效应引起的伪影。
    3. 改善了T2值的估计
    4. 以上都不是
    5. 以上所有

    该技术于1958年开发，至今仍用于多回波自旋回波序列，原因包括上述所有列举的方面。 链接至问答讨论

  7. 在反转恢复自旋回波脉冲序列中，反转时间被定义为
    1. 第一个180°脉冲与第一个90°脉冲之间的时间
    2. 第一个180°脉冲与回波之间的时间
    3. 每个TR间隔内两个180°脉冲之间的时间
    4. 第一个90°脉冲与第二个180°脉冲之间的时间

    答案a)是TI的定义。 链接至问答讨论

  8. 反转恢复技术的优点包括
    1. 更短的扫描时间
    2. 减少流动相关伪影
    3. 可叠加的T1和T2对比度
    4. 更低的比吸收率

    在常规自旋回波成像中，T1和T2效应通常是竞争的，意味着↑T1通常导致信号强度降低，而↑T2导致强度增加。在反转恢复成像中，通过适当选择TR，可以分离T1和T2效应并使其叠加。 链接至问答讨论

  9. 在1.5 T下获得的幅度重建反转恢复图像上，哪个反转时间可用于抑制脂肪？
    1. TI = 170
    2. TI = 450
    3. TI = 980
    4. TI = 2000

    对于大多数组织，零点通常出现在TI ≈ 0.69 × T1时。对于脂肪，在1.5 T下T1值约为250，对应的TI约为170 ms。 链接至问答讨论

  10. 关于相位敏感反转恢复，以下哪项陈述是错误的？
    1. 无论TI如何，短T1组织总是比长T1组织信号更亮。
    2. 背景空气总是显示为黑色。
    3. 通过调整TI来选择性抑制组织对于相位敏感反转恢复是不可能的，而幅度重建反转恢复则可以。
    4. 它对于评估儿童髓鞘形成和心肌对比增强很有用。

    相位敏感反转恢复保留了180°（反转）脉冲后纵向磁化强度的极性信息，更负的值显示为更暗。背景空气通常呈现为中灰色，而不是黑色。短T1组织总是比长T1组织亮，但在短TI时两者都可能比空气暗。常见应用在儿科和心脏成像。 链接至问答讨论

  11. 关于给定组织在幅度重建反转恢复中的零点值，哪项陈述是错误的？
    1. TInull随场强增加而增加。
    2. TInull随T1值增加而增加
    3. 对于长TR，TInull ≈ 0.69 × T2。
    4. TInull取决于信号是由常规自旋回波还是快速自旋回波序列产生。

    对于TR>>T1，TInull = T1 • (ln 2) ≈ 0.69 × T1（而非0.69 × T2，因此答案c是错误的）。由于组织T1随场强增加而增加，TInull也随之增加。TInull取决于快速自旋回波序列中最后一个回波的回波时间。 链接至问答讨论

  12. 关于短TI反转恢复序列，以下哪项陈述是错误的？
    1. 它是一种非常有用的脂肪饱和技术。
    2. 它是一种非常有用的检测钆增强的技术。
    3. 它具有叠加的T1和T2效应。
    4. 在均匀性相对较差的低场永磁体中尤其有用。

    STIR不能用作钆增强后的脂肪抑制技术。STIR并非特异性抑制脂肪；它抑制T1值在脂肪范围内的组织（200-300 ms），因此钆增强也会被抑制。 链接至问答讨论

  13. 以下哪组脉冲序列参数可用于创建T1-FLAIR脑图像？
    1. TR = 2500, TE = 10, TI = 900
    2. TR = 2500, TE = 50, TI = 180
    3. TR = 4000, TE = 80, TI = 1800
    4. TR = 8000, TE = 100, TI = 2500

    答案(a)正确。答案(b)可视为STIR图像，而(c)和(d)则是T2-FLAIR图像。 链接至问答讨论

  14. 用于脑成像的双反转恢复序列利用两个不同的TI值，通常用于抑制
    1. 脂肪和脑脊液
    2. 脑脊液和白质
    3. 脑脊液和灰质
    4. 灰质和白质

    双反转恢复脑成像对于检测白质和皮层病变特别有用。通常第一个180°脉冲抑制脑脊液，第二个抑制白质。 链接至问答讨论

梯度回波与磁敏感性

  1. 以下哪项不是梯度回波成像相对于自旋回波成像的优点？
    1. 更短的TE
    2. 更短的TR

  11. 关于ADC图，以下哪项陈述是正确的？

  23. 关于体素内不相干运动的以下哪项陈述是正确的？

    2. 体素内不相干运动描述了由扩散和微循环灌注引起的信号损失。
    3. 体素内不相干运动灌注效应减轻了仅由扩散引起的信号损失。
    4. Le Bihan的体素内不相干运动模型允许估计毛细血管血流量。

  24. 关于扩散峰度的以下哪项陈述是错误的？

    2. 当使用长回波时间时，峰度效应更明显。

脂肪-水

  1. 体内对磁共振信号贡献最大的主要脂肪类型是

    2. 游离脂肪酸
    3. 胆固醇
    4. 磷脂

  2. 在给定场强下，脂肪和水中¹H质子的核进动频率有何不同？
    1. 脂肪和水质子以相同频率进动
    2. 脂肪质子比水质子进动更快。
    3. 脂肪质子比水质子进动更慢。
    4. 脂肪质子根据其在甘油三酯侧链上的位置可能进动更快或更慢。

    甘油三酯的H质子位于长碳链中，被电子云屏蔽，免受外加磁场全部作用的影响。相比之下，水中的电负性氧原子将保护性电子云从H原子核上拉开，使其暴露于外加磁场的全部作用。因此，水质子比脂肪质子共振更快。 链接至问答讨论

  3. 3.5 ppm的化学位移写成十进制形式是
    1. 0.00000035
    2. 0.0000035
    3. 0.000035
    4. 0.00035

  4. 如果水-脂肪化学位移为3.5 ppm，计算在200 MHz运行的7T扫描仪上的脂肪-水频率差。

    3. 700 Hz

  5. 如果水-脂肪化学位移为3.5 ppm，计算在150 kHz运行的便携式0.064T扫描仪上的脂肪-水频率差。
    1. 0.5 Hz
    2. 5 Hz
    3. 50 Hz
    4. 500 Hz

  6. 在1.5 T下，水和脂肪约每2.2毫秒进出相位一次。在3.0 T下，它们进出相位的速度是多少？

    2. 每2.2毫秒
    3. 每3.3毫秒
    4. 每4.4毫秒

    该相位循环的周期为1/Δf，其中Δf是自旋之间的频率偏移。因此，如果场强加倍，频率偏移加倍，相位循环周期减半，因此(a) 1.1毫秒是正确的答案。 链接至问答讨论

  7. 在同相位图像上信号强度中等，在反相位图像上信号低的肾上腺肿块最有可能是

  9. 关于CHESS/脂肪饱和脉冲，以下哪项陈述不正确？

  10. 以下哪种翻转角模式不能定义为用于水激发的复合二项式脉冲？

    2. 1:1:1
    3. 1:2:1
    4. 1:3:3:1

  13. 比较SPAIR和SPIR的以下哪项陈述是不正确的？

    2. SPAIR使用180°反转脉冲，而SPIR使用100°-120°范围内的脉冲。
    3. SPAIR向组织中沉积的射频能量比SPIR少。
    4. SPAIR的反转时间比SPIR长。

  14. 关于绝热射频脉冲的以下哪项陈述是错误的？
