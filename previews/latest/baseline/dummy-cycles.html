预备循环

什么是预备循环？为什么需要它们？  
在之前的问答中介绍了磁饱和的概念。简单回顾一下，在施加任何射频脉冲之前，纵向净磁化矢量 (Mz) 处于最大值 (Mo)，而磁化矢量的横向分量 (Mxy) 为零。 一旦开始射频激发，纵向磁化矢量开始下降，直至达到一个稳态值 Mss，该值小于 Mo。在一些快速的梯度回波序列中，会形成相干的横向磁化矢量，并同样达到一个非零的稳态。 
部分饱和与稳态磁化 (Mo) 的形成
[左图] TR >> T1，因此在第二个射频脉冲之前，Mz 完全恢复至 Mo。
[右图] TR 较短，在施加下一个射频脉冲前，Mz 未能完全恢复。经过几个脉冲后，达到一个新的稳态纵向磁化 (Mss)，且 Mss < Mo。
为了让这些稳态磁化得以建立，扫描仪会执行数个不记录信号的序列循环。只有当系统接近平衡状态时，才开始数据采集。这些循环被称为预备循环、禁用采集或丢弃采集。预备循环的确切数量取决于 TR 和所选的序列。其数量可能从几个到超过 50 个不等（对于具有短 TR 的稳态梯度回波序列）。
