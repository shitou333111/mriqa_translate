一般线性模型 (GLM)

我实在不理解 GLM 是如何工作的。你能更完整地解释一下吗？ 
请不要气馁。对于缺乏统计学或矩阵代数背景的人来说，一般线性模型 (GLM) 可能令人生畏，但我会尝试在此让它变得更易于理解。这是一项值得付出的努力，因为在过去的 25 年里，GLM 一直是分析基于任务的 fMRI 实验最广泛使用的技术，并且是厂商为其临床 fMRI 软件包提供的默认方法。
可以将 GLM 视为一个更熟悉的统计技术的扩展：线性回归。线性回归，有时也称为趋势线分析，是一种用于为一组实验数据计算“最佳拟合”直线的方法。线性回归软件在流行的图形和电子表格软件包（如 Microsoft Excel）中广泛可用。


简化的线性回归示例，改编自
Hansen 等人 (2004) 的数据
作为线性回归的一个应用，让我们研究一下视觉皮层中一个体素的平均 fMRI 信号 (Y) 如何响应视网膜光照水平 (X) 的增加。将我们的数据绘制在图表上（左图），我们可以看出 X 和 Y 之间存在相当线性的关系。线性回归软件为此数据计算“最佳拟合”直线。对于每个数据点 (xi, yi)，其关系可以写成 yi = xiβ + εi, 其中 β 是计算出的直线斜率，而 εi  是计算出的直线与相应数据点之间的误差​（或残差）距离。请注意，xi 值由实验设计决定，yi 值是测量得到的，而β 和 εi's 则由软件计算得出。
线性回归的一个必要假设是误差 (εi) 是随机且独立的，遵循均值为零的高斯分布。“最佳拟合”直线被定义为使点与所选直线之间的误差平方和 (ε1² + ε2² +  ε3² + …) 最小化的那条直线。请注意，尽管计算出的斜率 (β) 对于每个数据点(xi, yi) 是相同的，但个体误差 (εi) 会有所不同。
我们简单的线性回归示例实际上产生了n 个独立的方程，每个方程的形式为  yi = xiβ + εi，其中 n 是每个时间间隔 i 采集的数据点数量。矩阵代数允许我们将这组方程压缩成更易于管理的形式。我们通过将 Y、X 和 ε 定义为“列向量”来实现这一点，每个向量包含一系列单独的测量值或计算值。具体来说，

fMRI 实验的一般线性模型 (GLM) 表示保留了与我们简单线性回归示例相同的基本形式(Y = Xβ + ε)。用文字表述，GLM 表明 Y（单个体素随时间变化的测量 fMRI 信号）可以表示为一个或多个实验设计变量 (X) 的总和，每个变量乘以一个权重因子 (β)，再加上随机误差 (ε)。

在 GLM 中，Y ​和 ε 仍然作为单列向量，分别包含单个体素在连续时间点 (i = 1 到 n) 的 fMRI 信号数据 (yi) 或误差估计 (εi)。然而，实验设计矩阵 (X) 通常要复杂得多，可能包含 5-10 列，而不仅仅是一列。X 的每一新列将由研究者构建，以反映被认为会影响实验结果的特定因素（“回归因子”）。
基本回归因子 （也称为关注回归因子）是一组理想化的预测，描述了如果感兴趣的体素因任务或刺激而被激活，其血流动力学响应函数 (HRF) 应该呈现的样子。例如，考虑一个简单的事件相关 fMRI 实验，受试者被给予三个任务（A、B 和 C）——例如，以随机顺序呈现三种不同频率的声音。如果任务 A 的刺激在上图红色箭头所示的时间点施加，那么听觉皮层中一个感受性体素的预测响应可能是一系列略有延迟、短时程的 HRF，如图所示。与任务 A 的此预测响应模式对应时间点的数值 (X i,1) 随后将被转移到 GLM 设计矩阵的第一列并垂直列出（如下图所示）。对任务 B 和 C 的预测响应也将以类似方式创建，并用于填充设计矩阵 (X) 的第二列和第三列。
我们设计矩阵示例的最后 7 列被认为是协变量或干扰回归因子——这些是干扰分析（例如头动或信号漂移）但本身并不特别感兴趣的实验因素。填充这些列的数据可以是经验确定的（例如，实验期间获得的头部平移和旋转的实际测量值），也可以是建模的（例如，通过线性趋势或振荡基函数）。
由于设计矩阵 (X) 现在包含多列，权重参数 (β) 将不再是简单线性回归示例中的单个数字。在完整的GLM 实验中拟合数据意味着 β 现在将变成一个列向量，其值为 (β1, β2, …, βp)，每个值反映了每个设计因子 (Xi) 对 fMRI 信号的相对贡献。因此，幅度向量 (β) 必须具有与设计矩阵 (X) 的列数相同的行数。​​

单个体素的一般线性模型（GLM）示意图，其时间序列 Y 由设计矩阵 X 预测，该矩阵包含10个效应（三个感兴趣的回归量 – 例如，任务A、B、C – 以及七个 nuisance 回归量 – 例如，六个运动参数和一个线性漂移）。计算得出的与每个回归量相对应的加权因子（β1 − β10）置于幅度向量 β 中，而列向量 ε 则包含与每个时间点 i 相对应的模型计算误差项（εi）。（来自 Monti, 2011，遵循 CC BY 许可）
基本的GLM分析固有两大假设，这两项假设都可能受到合理质疑。首先，GLM是一种单变量方法，它在逐个体素的基础上计算统计量，并假设每个体素的信号相互独立。其次，该模型假设误差是随机且独立的，遵循均值为零的高斯分布。

在fMRI数据被采集并存储于Y向量中、实验设计因子被指定于X矩阵之后，随后使用经典或贝叶斯方法计算β 和ε向量的分量。然后可以对这些结果进行统计检验，测试关于各个回归量或其组合的各种假设。最终输出通常是叠加在解剖图像或模板上的统计参数图（SPM）或后验概率图（PPM）。

血液动力学响应函数（HRF）的建模
HRF模型
对于给定的fMRI实验，究竟应该如何确定HRF的形状？术语“规范”HRF 指的是研究者在数据采集之前选择的默认形状。选择包括简单的伽马函数、两个伽马函数的差值（以允许刺激后 undershoot）、傅里叶基组、由“合理”HRF形状组成的基组以及有限脉冲响应函数。为了提高准确性，可以将HRF的一阶和二阶时间导数（通常分别称为“时间”导数和“弥散”导数）添加到模型中。使用这些导数相当于对HRF函数进行二阶泰勒级数近似。包含一阶导数可以调整峰值响应的时间，而二阶导数则允许HRF的宽度变化。﻿

为了创建回归量，将所选HRF的形状与实验设计进行卷积（“乘以”）。事件之间的非线性相互作用可能会扭曲预期的HRF（例如，两个刺激在非常短的时间框架内施加，总响应可能小于各自响应之和），也可以使用Volterra核或其他方法进行建模。​​
统计显著性的检验
﻿﻿大多数fMRI实验的最终目的是确定大脑如何响应各种刺激和条件。对于基于任务的fMRI研究，从GLM计算出的幅度向量β提供了实验设计中每个分量（Xi）的相对权重（“重要性”）的估计值（βi）。

我们从最简单的fMRI实验之一开始：手指敲击。在此实验中，受试者在敲击手指和休息之间交替循环，同时记录全脑的BOLD响应。在此示例中，设计矩阵可能只包含一个基本的回归量，表示为：X1 = “手指敲击”。数据采集后，GLM分析产生每个体素对手指敲击的幅度响应估计值（β1）。运动皮层附近的体素将表现出强烈的响应，其β1值相对较大，而远处的体素可能仅包含静息态“噪声”。那么，我们可以使用什么标准来确定何时认为一个体素被真正激活了呢？ 
检验统计显著性的形式化/经典方法涉及构建一个零假设（Ho），用语言表述可能是：“与休息相比，手指敲击期间体素内没有净BOLD信号效应。”用数学术语可表示为：
​Ho:  β1 = 0.
﻿通常，零假设是以一种否定/反常的方式构建的——即，给定事件或行为没有产生fMRI效应。然而，在现实中，我们暗自希望感到惊讶，并发现一个不太可能偶然发生的结果，从而可以拒绝零假设。
我们通常所说的“统计显著”是指拒绝零假设的I型（假阳性）错误小于某个任意设定的概率水平（p值），例如p ≤ 0.05。换句话说，只有当事件偶然发生的概率小于5%时，我们才可能称结果为“显著”。 
t分布与z分布的比较。随着自由度（df）增加，两者趋同。
GLM计算既产生β1的估计值，也产生该估计值的标准差（SD）。SD取决于均方误差之和以及设计矩阵（X）的结构和大小。为了检验零假设，我们构造检验统计量T = β1/SD。假设GLM的假设成立（即误差是独立的且服从均值为零的高斯分布），则统计量T服从学生t分布，据此可以确定p值的百分位数。t分布的确切形状取决于自由度（df），其定义为分析的时间点数减去独立回归量的数量。t分布与高斯分布或标准正态（z-）分布密切相关，随着df → ∞，前者收敛于后者。在观测值超过100个的实验中，通常报告Z值和百分位数而非T值，因为两者非常接近。
多个回归量：对比和F检验
上述手指敲击示例仅包含一个感兴趣的协变量（X1 = “手指敲击”）及其对应的估计幅度（β1）。然而，大多数有趣的fMRI研究包含多个感兴趣的回归量。让我们考虑一个更复杂的fMRI实验，其中受试者交替看到红色、蓝色和绿色的光，单独或组合呈现。设计的三个基本回归量可能是：X1 = “红光开启”，X2 = “蓝光开启”，以及X3 = “绿光开启”。数据采集后，GLM将生成三个幅度估计值（β1，β2和 β3），对应于可以检验统计显著性的三种条件。
多个设计变量允许研究组合效应。例如，我们可能想知道一个体素是否被红光和蓝光同等激活（β1= β2），或等价地（β1 − β2 = 0）。或者，红光和蓝光照射的平均效果是否与单独绿光照射的效果相同（½ [β1 + β2 ] = β3），或等价地（½β1 + ½β2 − β3 = 0）。

数据的自相关性
GLM的基本信条之一——逐点误差的统计独立性——在fMRI实验中几乎总是被违反的。功能MRI研究不是随机的数据集合，而是时间序列，高值之后更可能出现高值，低值之后更可能出现低值。准周期性噪声源（如心脏和呼吸搏动）也促成了这一现象。

幸运的是，序列相关性的存在并不影响加权参数（βi)的估计。然而，它确实会导致误差方差被低估，因为真正独立观测的数量（“有效”自由度，df）将低于实际进行的观测数量。因此，t-或F-统计量（其分布形状取决于df）将被人为放大，可能导致假阳性率升高和错误的推论。

控制自相关最常用的方法被称为预白化。术语“预白化”指的是移除时间序列数据中的序列相关性，使其残差具有更均匀的频率分布，类似于“白噪声”。预白化通过迭代过程执行。首先生成一个初始GLM解，并将残差的自相关结构拟合为一阶或二阶自回归模型[AR(1)或AR(2)]。然后通过移除估计的协方差结构来调整原始数据，并对“白化”后的数据执行第二次GLM。
多受试者分析
到目前为止，我们只讨论了单个受试者数据的GLM分析。为了使fMRI结论更具普适性，通常将个体结果汇总到多个受试者或组中。然而，在进行统计分析之前，必须将多个受试者的数据变形（warp）到一个共同的解剖空间（如Talairach或MNI模板）。这个过程称为归一化，已在之前的问答中描述过。
最早的组分析方法之一是将所有受试者的数据变形到归一化空间，然后将它们连接成一个巨大的时间序列（就好像数据都来自一个“超级”受试者）。这种所谓的固定效应分析已被基本放弃，因为它不能充分解释个体差异，也不能干净地用于进行总体推断。
多受试者fMRI数据本质上是层次化的，至少需要考虑两个层次：个体和群体。因此，混合效应分析更为合适，其中误差在受试者内部和受试者之间都被测量和考虑。
﻿实现这一点的一种方法是两水平GLM分析。在第一水平，对每个受试者分别执行标准GLM。然而，在第二水平，仅传递βi值（而不是完整的fMRI响应数据），它们成为因变量。第二水平的推论可能很困难，因为βi只是估计值，其值被第一水平分析的方差“污染”。存在几种处理此问题的方法，其中最流行的方法之一是使用期望最大化算法的限制性最大似然技术。
