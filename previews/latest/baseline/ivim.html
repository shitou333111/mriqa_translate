IVIM 成像

什么是体素内不相干运动 (IVIM)？这与扩散相同吗？ 
现代扩散成像之父 Denis Le Bihan 在 20 世纪 80 年代创造了 体素内不相干运动 (IVIM) 这一术语，用以指代在 MR 实验过程中水分子在体素内的微观平移。当在 MR 信号演化期间施加梯度时，IVIM 会导致自旋失相和信号丢失。
如果梯度相对较强，IVIM 引起的信号丢失主要归因于扩散——即水分子在细胞内及周围的布朗运动。然而，当使用较弱的梯度时，第二种 IVIM 机制也会导致信号丢失——毛细血管网络中的血液微循环。自 Le Bihan 的开创性论文发表后的 25 年里，微观的灌注效应在很大程度上被忽视了，但最近在如今被称为IVIM 成像的领域中迎来了复兴。
如之前的问答所述，现代成像系统测量每个体素在有和无扩散敏感梯度情况下的 MR 信号强度 (S)。通常只使用两个 b 值：非常低的（例如，b= 0-50 s/mm² ）和高的（例如 800-1000 s/mm²）。 根据这两个测量值，可以计算出表观扩散系数 (ADC)，这是一个有用但粗略的指标，基于一个单指数模型，该模型将 MR 信号强度 (S) 相对于基线 (So) 表示为  

S = Soe−b•ADC       或       ln(S) = ln(So) − b•ADC

在此模型下，信号以 ln (信号) 对 b 绘制的半对数图应是一条斜率为 −ADC 的直线（下图中的红色实线）。

DWI 中预期的 MR 信号偏离单指数模型的情况。在低 b 值时，必须考虑微观灌注引起的 IVIM 效应（紫色），而在高 b 值时，必须考虑峰度效应（绿色）。
然而，当使用大量不同的 b 值进行仔细的信号测量时，会观察到与这种简化的单指数模型存在相当大的偏差。在低 b 值下（即 ≤ 300 s/mm²），信号从 So 下降的速度比预期的要快，这是由于微观灌注导致的 IVIM 损失增加。在较大的 b 值下（≥ 1000−1500 s/mm²），信号通常比粗略的 ADC 线预期的要大，这种效应是由于扩散概率分布的非高斯形状（峰度）造成的，将在下一篇问答中进一步讨论。
为了解释这种在低和中 b 值下的偏差，Le Bihan 提出了一个双指数模型，同时考虑了扩散和灌注效应。  在此，他区分并命名了两个扩散系数：D（“纯”扩散）和D*（“伪”扩散），前者反映血管外间隙的扩散，后者反映由于在半随机排列的毛细血管中灌注引起的失相。血管密度由一个无量纲的灌注分数 (f)描述，即毛细血管占据的体素体积百分比。相反，(1−f) 反映了仅发生纯扩散的血管外间隙。根据曲线初始部分的陡峭程度（这又取决于毛细血管密度和灌注），D* 可能比纯D大 5-10 倍。这些效应使得双指数模型可以简化为： 

S/So = fe−bD* + (1−f)e−bD

通过在多个 b 值下采集 DWI 图像并将数据拟合到上述方程，可以估计 f、D 和 D*，并为每个参数创建图。通常获取 6 到 10 个数据集，b 值范围在 0 到 1000 s/mm² 之间，其中至少一半的测量值在小于 250 s/mm² 的条件下进行。

然而，在估计 D* 时存在相当大的误差，我个人认为这些估计不可靠，在不同中心和不同研究之间变化太大。不仅 D*，所有 IVIM 参数估计都受到运动、噪声以及分泌性生理过程（如腺体和肾小管流动）的影响，这些过程很难与毛细血管灌注区分开来。因此，我目前对 IVIM 成像在我们未来的成像技术库中将发挥多大作用仍持一定的怀疑态度。
