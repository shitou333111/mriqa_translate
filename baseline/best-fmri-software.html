fMRI 处理软件

处理 fMRI 数据的最佳软件是什么？  
大多数主要的 MR 制造商都提供基本的集成式 fMRI 处理软件（例如，GE 的 BrainWave，Philips 的 I View BOLD）作为可选购买项。对于大多数用于临床皮层定位的简单任务态 fMRI 研究来说，这些软件可能已经足够，但更复杂的实验可能需要第三方解决方案。许多第三方软件是免费且开源的。有些提供了用于实验设计、处理和数据分析的强大工具集，而另一些则更为“基础”。我不会在此评判哪个是最好的，但我确实是伴随着 SPM 成长起来的，对它最为熟悉。以下按字母顺序总结了最广泛使用的独立 fMRI 软件。
AFNI (Analysis of Functional Neuroimaging) 是一套广泛的、基于 C 语言的免费程序集，用于处理、分析和显示 fMRI 数据。最初由威斯康星医学院的 Robert Cox 在 20 世纪 90 年代中期开发，AFNI 现由位于马里兰州贝塞斯达的国家心理健康研究所 (NIMH) 托管。附加工具包括用于颅骨剥离的程序和用于基于皮层表面的 fMRI 分析的 SUMA。可在 Unix、SGI、Solaris、Linux 和 Mac OS X 上运行。  可通过此链接获取。
Brain Connectivity Toolbox 包含超过 120 个脑连接和统计函数，主要为复杂网络（图论）分析而设计，最广泛用于分析静息态 fMRI 研究。这个免费网站是多名研究者的成果，主要是印第安纳大学的 Olaf Sporns 和剑桥大学的 Mikail Rubinov。需要 MATLAB，尽管部分函数可以在 Octave 中运行。可通过此链接获取。 
BrainSuite 是一套开源软件工具集，主要用于自动化脑部 MRI 处理。它是由加州大学洛杉矶分校的 David Shattuck 和南加州大学的 Richard Leahy 的研究小组合作制作和发布的。其主要功能是提取和参数化大脑皮层的内外表面，并分析扩散 MRI 数据。适用于 64 位 Mac OS X、Windows 和 Linux。可通过此链接获取。
BrainVISA，来自法国第 49 号联合研究所 (IFR)，是一个用于海量数据分析的免费神经影像软件平台。可在 Windows、Unix/MacOS 和 Linux 系统上运行，并与 FreeSurfer 接口。可定制模块包括用于脑分割、皮层沟回识别与形态测量、交互式 3D 可视化以及皮层表面参数化与分析的工具箱。可通过此链接获取。
Brain Voyager 是一套全面的商业程序套件，用于功能成像分析和可视化，不仅包括 fMRI，还包括 DTI、EEG 和 MEG。还提供用于 TMS 神经导航、fMRI 生物反馈和神经网络模拟的附加模块。该公司总部位于荷兰，首席执行官 Rainer Goebel 最初在 20 世纪 90 年代中期开发了这个平台。它可在所有主要计算机平台上运行，包括 Windows、Linux 和 Mac OS X。它拥有优秀的在线手册，解释了软件的操作，并且对于 fMRI 处理总体上非常有教育意义。可通过此链接获取。 
FreeSurfer 是一个用于处理和分析脑部 MR 图像的开源软件套件。它由哈佛大学/MGH Martinos 中心计算神经影像实验室的 Bruce Fischl 及其团队开发。该软件功能全面，包括用于颅骨剥离、图像配准、分割、皮层拓扑、fMRI 分析和纤维束成像的模块。  也可以进行 PET 数据分析。该软件真正的优势在于其皮层和皮层下分析工具，用于测量、膨胀或展平皮层表面并在其上执行多被试 fMRI 统计。FreeSurfer 可以与 FSL 和 MNI 的脑成像软件工具箱集成（尽管有些笨拙）。它可在 Linux、Mac OS X 和 Windows（通过 VirtualBox）上运行。提供广泛的 Wiki 和在线教程。这是链接。
FSL (FMRIB Software Library) 是牛津大学脑功能磁共振成像 (FMRIB) 实验室提供的一套免费应用程序。该软件的首次发布于 2000 年，此后 FSL 一直呈指数级增长。它主要为 Mac OS X 和 Linux 设计，但可以在 Windows 的虚拟机环境中使用。提供数十个作为独立程序运行的工具集。最广泛使用的模块包括 FEAT（用于任务态 fMRI 的基于模型的分析）、MELODIC（用于静息态 fMRI 的基于 ICA 的无模型分析）、BET（用于脑提取）、FAST（用于组织分割）和 MCFLIRT（用于运动校正）。还提供用于扩散纤维束成像和分析，以及基于 ASL 的 fMRI 和灌注测量的工具。提供许多第三方插件。FSL 可以通过此链接访问。
MIALAB (Medical Image Analysis Lab)，由新墨西哥大学的 Vince Calhoun 领导，开发了一套基于 MATLAB 的免费脑成像分析软件。他们小组的独立成分分析工具箱 (GIFT) 是使用最广泛的。这是链接。 
REST (RESting-state fMRI data analysis Toolkit) 是一组基于 MATLAB 和 SPM8 的应用程序，用于评估静息态 fMRI 数据。它由北京师范大学的一个团队构思和设计。该软件提供与 SPM、AFNI 和 FSL 的数据交换/共享。其流程和灵活模块可以执行全方位的预处理步骤（例如，层间时间校正、运动校正、去趋势），但其真正的优势在于分析 RS-fMRI 数据（例如，基于相关性的功能连接、低频波动幅度分析、格兰杰因果分析）。主要软件应用程序可通过此链接获取。 REST 也嵌入在一个更全面的网站中，该网站链接到其他 RS 和连接性应用程序，即The R-fMRI Network (rfmr.org)。
SPM (Statistical Parametric Mapping) 由 Karl Friston 在 20 世纪 80 年代末创建，用于在体素水平分析 PET 图像。在五年内，SPM 成为分析 PET 数据以及（当时尚处于起步阶段的）fMRI 技术的主导方法。如今，SPM 已发布第 12 版，由伦敦大学学院威康信托神经影像中心的成员和合作者持续开发。SPM 是免费的，只要安装了核心 MATLAB，即可在 Windows、Linux 和 Mac OS X 下运行。SPM 几乎具备 fMRI 处理、分析和显示所需的所有功能。与大多数其他 fMRI 软件一样，SPM 采用广义线性模型 (GLM) 进行其主要分析，因为 Friston 是该技术的早期倡导者之一。SPM 网站可通过此链接找到。

进阶讨论 (显示/隐藏)»

上述讨论未包括专门的 fMRI 范式设计工具。这些程序帮助研究者规划 fMRI 实验所需的事件时序和数据采集。然后，它们向各种视听设备和 MRI 扫描仪发送适当的信号，以触发刺激呈现和记录被试反应。

所有主要的 MR 供应商都提供基本的范式设计工具，作为其标准集成 fMRI 软件包的一部分。此外，上述几个第三方 fMRI 分析软件包也包含设计模块。以下是几个主要功能为 fMRI 设计的常用工具包列表：

E-Prime® 是来自宾夕法尼亚州匹兹堡 Psychology Software Tools, Inc. 的商业应用程序套件。该软件广泛用于非 MRI 心理学和神经生理学实验，但也有扩展功能允许其用于 fMRI。可以通过拖放式图形用户界面创建广泛的常规范式，但更复杂的实验需要使用其专有的脚本语言（幸运的是，该语言几乎与 Visual Basic for Applications 相同）。 更多信息可通过此链接获取。

nordicAktiva 是来自挪威公司 NordicNeuroLab 的商业化临床刺激和工作流软件解决方案。他们不仅提供设计软件，还提供用于 fMRI 实验的完整视觉、听觉和反应硬件包。我们历史上曾使用此系统进行临床语言皮层定位研究。然而，编写复杂的新范式有一定难度，因为它需要直接编辑 XML 代码。 更多信息可通过此链接获取。

Presentation 来自加利福尼亚的 Neurobehavioral Systems, Inc.，可能是神经科学中最易使用的刺激呈现和实验控制程序，具有可定制的图形用户界面。它能够同时且独立地呈现广泛的刺激，包括图像、2D 和 3D 图形、音频和视频。主要限制是该程序仅在 Windows 下运行。 更多信息可通过此链接获取。

上述软件均为商业软件，单个用户许可证价格从几百到几千美元不等。也有一些免费的开源解决方案可用，包括诺丁汉大学的 PsychoPy (网站) 和 Psychtoolbox (网站)。PsychoPy 用 Python 编写，而 Psychtoolbox 在 Matlab 或 Octave 上运行。
