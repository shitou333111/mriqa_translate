超越 90° 的射频脉冲

我想我理解了 90° 脉冲的工作原理。但当扩展到 180°、270° 甚至 360° 时会发生什么？
射频脉冲使净磁化矢量从 0° 旋转至 360°。
从 90° 旋转至 180°
​
将 M 从 90° 旋转至 180° 需要持续向自旋系统输入射频能量。假设这是通过一个短时、理想的射频脉冲实现的，那么该旋转是相干的，自旋演化 保持幺正且可逆。与 90° 脉冲一样，不会发生布居数重新分布。 玻尔兹曼布居数在数量上得以保留，但在塞曼能级之间发生了交换。因此，射频脉冲重新定向了宏观磁化矢量，而没有改变熵或使塞曼能级达到平衡。 
在一个完美的 180° 脉冲之后，M 相对于 Bo 发生反转，其纵向 分量 (Mz) 现在等于 −Mo （大小相同，但符号与初始值 +Mo 相反）。  塞曼期望能量相应地由 −MoBo 升高至 +MoBo, 使自旋系统处于亚稳态的反转状态。这一变化反映了由幺正演化产生的布居数的相干重排，而非能量的不可逆吸收。没有熵产生。任何预先存在的横向相干性在旋转过程中得以保留。多余的塞曼能量仅在随后的纵向（T1）弛豫过程中释放。
​在一个短时、非饱和的 180° 射频脉冲中，所增加的射频能量可逆地存储为反转自旋布居数中增加的塞曼能量——即存储在塞曼库中——直到后来通过 T₁ 弛豫耗散到晶格中。
在 180° 时，自旋现在占据着稳态的能量本征态，但其比例使得它们在静磁场中的磁势能最大化而非最小化。这种反转分布代表了存储的能量，其形式不是振荡相干性，而是能量不利态的过量占据。
就像被强制指向错误方向的指南针针一样，M 的这种反转状态是亚稳态的，无法无限期持续。  可能发生两种情况：
  • T1 弛豫将导致能量从自旋系统流向环境（“晶格”），引起 M 纯粹沿 z 轴重新增长，直到其恢复到初始值 +Mo。
  • 持续的射频脉冲刺激将产生超过 180° 的进一步旋转，达到 270°、 360° 或更远。
180° 脉冲后，由于 T1 弛豫，M 沿 z 轴重新增长。
从 180° 旋转至 270°
回想一下，塞曼能量 指的是磁矩 (M) 与外磁场 (Bo) 之间的相互作用能。  由于 Bo 通常被指定沿 z 轴方向，其塞曼能量 (Ez) 可写为 Ez = − M • Bo  = − MoBo.  如图所示，磁化矢量在  0° /  90° / 180° /  270° / 360°  时的塞曼能量分别为  −MoBo / 0 / +MoBo ​/ 0 / −​MoBo。因此，0° 与 180° 之间的能量差为 +MoBo − (−MoBo ) = 2​MoBo。
​对于一个理想的共振脉冲，超过 180° 的持续旋转涉及射频场与自旋系统之间持续的相干能量交换。在 270° 时，横向磁化再次达到最大值（沿 −y 方向），塞曼能量为零。此状态是 90° 状态的相位反转类似物。从 180° 到 270° 的转变对应于塞曼能量减少了 MoBo。  由于在短脉冲期间没有发生弛豫，此能量无法作为热量耗散；相反，它被相干地返回到射频场，并最终返回到耦合的射频电路中。  耦合机制的详细信息可在此相关问答中找到。 
持续旋转使磁化矢量返回到 +z 方向。  在完成一个 360° 射频脉冲后，密度矩阵、磁化矢量和内能均与其初始值相同。在脉冲期间，能量在射频场与自旋系统之间相干交换，在 180° 时达到最大值，但在循环结束时，所有这些能量都返回到射频电路。没有横向相干性残留，在一个理想的 0°–360° 旋转中，没有净能量转移。
