预测 T1 与 T2 值

如何预测不同组织的 T1 和 T2 值？  
目前，尚无一个全面的理论能够可靠地预测不同组织的确切 T1 和 T2 值。  虽然已经发展出一些复杂的理论，足以解释简单溶液和大分子均质制剂的弛豫特性。  然而，组织是极其复杂的，包含多种不同的分子，并在微观和宏观层面都具有结构。  即使在组织中分析水也并不简单；  它可能以多种状态存在，从自由（未结合）状态到部分结构化状态，再到完全结合状态。
在做出关于交换速率、水组分等各项假设后，大多数组织的弛豫特性可以用 2 室或 3 室模型来解释。  其中最古老且最著名的模型之一是 Zimmerman 和 Brittin 于 1957 年首次提出的快速交换模型。  右侧图示展示了一个更现代的多室模型，它显示了影响 T1 和 T2 的不同因素。  需要记住的是，所有这些模型都只是对“现实”的粗略近似，并且可以构建出许多等效的模型。 
要预测不同组织的相对 T1 和 T2 值，并解释它们在磁共振成像中的一般表现，首先必须确定观察到的信号主要来自脂质质子还是水质子。  对于大多数组织，水质子产生了大部分（如果不是全部）磁共振信号。  事实上，T1 和 T2 值与组织的总体水含量有着最强且最简单的相关性。  通常，组织越“软”或“多汁”，其 T1 和 T2 值就越长。  例如，利用这一原理可以预测（并记住）肾髓质（尿液汇集处）的 T1 和 T2 值比肾皮质长；  脾脏（含血更多）的 T1 和 T2 值比肝脏长，等等。  尽管这个“软度”概念应被视为纯粹的记忆工具，并且很容易找到例外情况，但在过去几年里，它对我而言非常有用。  事实上，以这个粗略概念为起点，许多年前我能够证明（当然，经过严格的、双盲的放射学-病理学相关性研究）为什么有些脑膜瘤在磁共振图像上是暗的，而另一些是亮的。

虽然在大多数组织中，总体水含量是 T1 和 T2 的强预测因子，但在其他组织中，必须考虑脂肪族脂质质子对磁共振信号的贡献。这些非极性储存脂肪具有较短的 T1 值，但相对较长的 T2 值（由于它们大小适中，其运动接近拉莫尔频率，并且几乎没有静态贡献导致 T2 失相）。  例如，在脂肪组织中，几乎所有的磁共振信号都来自这类脂质质子。  在其他组织，如骨髓、肝脏和骨骼肌中，脂肪质子和水质子各自对总信号和净弛豫时间有显著贡献。在这里，仔细的测量实际上会揭示 T1 和 T2 的多组分贡献。

进阶讨论（显示/隐藏）»

如果我们仅仅知道所有正常和病理组织的 T1 和 T2 值，以及其他可测量参数（如自旋密度 [H]、磁化率、扩散等），我们是否能够基于独特的模式来诊断疾病？这就是“磁共振指纹识别”背后的理念，这是一项引人入胜的新进展，它将新颖的磁共振测量技术与模式识别算法和大型参数数据库相结合。详情请参阅 Ma 等人的参考文献。
