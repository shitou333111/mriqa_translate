频率编码

频率编码是如何工作的？  
空间位置的频率编码是通过使用辅助磁场梯度来实现的。梯度以一种可预测的模式轻微地扭曲主磁场，导致共振频率随位置变化。频率编码可用于定义以下两种位置：1) 层面内的位置，或 2) 层面间的位置。
用于层面内定位的频率编码
右图展示了如何使用频率编码来指定层面内的位置。这里，一个频率编码梯度 (Gf) 从图像左侧位置 x=0 开始，并沿水平轴线性增加。如果主（静态）磁场是 Bo，那么沿水平轴任意点 (x) 处的有效磁场 B(x) 由下式给出：
B(x) = Bo + xGf
根据拉莫尔方程 (f = γB)，我们可以很容易地证明，共振频率 f(x) 也沿频率编码轴随位置 (x) 线性变化： 
f(x) =  γB(x) = γBo + γx•Gf = fo + fg(x)
其中 fo 是主磁场的拉莫尔频率，fg(x) 是基于沿梯度位置的频率偏移。
沿水平 (x-) 轴施加的线性增加的频率编码梯度 (Gf)。像素 A、B 和 C 都以相同的较低频率共振；D、E 和 F 以相同的较高频率共振。
每个像素都有有限的宽度，因此实际上包含一个小的频率范围（称为每像素带宽），而不仅仅是单一频率。尽管如此，任何一列（如 ABC）中的信号都被分配到了相同的窄频带，并且这些频带与另一列（如 DEF）的信号频率不同。 
尽管频率编码是一大步，但我们仍然无法区分同一列内的像素（例如 A 与 B 与 C），因为它们的频率重叠。在第二个维度上的空间检测将需要另一种方法，通常是相位编码。
用于层面选择的频率编码
频率编码也可用于定义二维成像中整个层面的位置。基本原理与层面内频率编码相同，但现在梯度是沿层面选择方向施加的。整个层面的中心共振频率现在将随位置线性变化。可以通过施加一个专门调谐到该层面频率的射频脉冲来激励单个层面。此类射频脉冲的设计以及层面选择的更多细节将在后续的问答中找到。 
在 z 方向施加的磁场梯度导致共振频率在层面间变化数千赫兹。

进阶讨论（显示/隐藏）»

尽管频率编码用于定义二维磁共振成像中的层面位置，但三维磁共振成像中的层面选择通常使用相位编码来执行。
