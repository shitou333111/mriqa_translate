基础电磁学

  1. 当流经导线的电流方向反转时，导线周围的磁场
    1. 不变
    2. 增加
    3. 消失
    4. 方向反转

    安培定律指出，导线中的电流（i）会在导线周围感应出磁场（B）。如果电流方向反转，磁场方向也随之反转，因此d)正确。磁场的大小取决于电流的大小，因此b)和c)错误。链接至问答讨论

  2. 物质的体磁性质主要源于
    1. 质子
    2. 中子
    3. 电子
    4. 整个原子核

    固有电子自旋和电子轨道角动量的结合是物质体磁性质的主要原因。质子、中子和整个原子核具有自旋，但磁效应的大小相对较小，仅限于原子核附近区域。链接至问答讨论

  3. 如果导线中的电流加倍，感生磁场
    1. 加倍
    2. 变为四倍
    3. 保持不变
    4. 减半

    安培定律指出，导线中的电流（i）会在导线周围感应出与电流成正比的磁场（B）。如果电流加倍，B的大小也加倍。链接至问答讨论

  4. 导线周围磁力线的方向可以用______确定
    1. 右手定则
    2. 左手定则
    3. 法拉第定律
    4. 楞次定律

    弗莱明提出了右手定则：用右手握住载流导线，拇指指向电流方向，则手指弯曲的方向即为感生磁场的方向。链接至问答讨论

  5. 在外加静磁场中，静止导体上的感应电压
    1. 取决于导体与磁场的夹角
    2. 随时间增加
    3. 为零
    4. 取决于磁场的强度

    这是法拉第-楞次定律的一个例子，感应电压与磁场的变化率（dB/dt）成正比。本例中导体和磁场都是静态的，因此dB/dt = 0，感应电压为零。链接至问答讨论

  6. 关于电与磁的关系，以下哪项陈述是错误的？
    1. 导线中的恒定电流会在导线周围感应出恒定磁场。
    2. 导线中的变化电流会在导线周围感应出变化磁场。
    3. 恒定磁场会在附近的静止导线中感应出电压。
    4. 变化磁场会在附近的导线中感应出电压。

    安培定律指出，导线中的电流（i）会在导线周围感应出磁场（B）。如果电流恒定，磁场恒定；如果电流波动，磁场也随之波动。因此(a)和(b)正确。 法拉第-楞次定律指出，变化磁场可以在附近的导线中感应出电势（电压）。该电压（V）的大小与磁场的变化率（dB/dt）成正比。因此(d)正确。恒定磁场不随时间变化（dB/dt = 0），因此不会在附近导线中感应出电压。故(c)错误。 链接至问答讨论

  7. 关于特斯拉（T）的哪项问题是正确的？
    1. 它是cgs单位制中磁感应强度的官方单位。
    2. 1特斯拉 = 1,000高斯（G）
    3. 1 G = 1 mT
    4. 它是道路上最酷的汽车之一

    特斯拉是国际单位制（SI）（以前称为mks（米-千克-秒）制）中磁感应强度的单位。高斯是cgs（厘米-克-秒）制中的对应单位，因此a)错误。1特斯拉 = 10,000 G，且1 G = 0.1 mT，因此b)和c)都错误。剩下选项d)作为正确答案，这一点大家都知道！链接至问答讨论

  8. 关于磁场强度，哪项陈述是正确的？
    1. 地球磁场约为0.5 G。
    2. 用于拾起汽车的废金属场电磁铁比大多数MR扫描仪的主磁场强得多。
    3. 用于人体的研究型MR扫描仪现已存在场强超过20 T的系统。
    4. 更高场强的扫描仪具有更宽的孔径，以容纳额外的磁通线。

    赤道处的地球磁场约为0.5 G，因此a)是正确答案。废金属场电磁铁的场强通常约为1T，受限于钢的磁通密度，因此比大多数MR扫描仪弱得多，故b)错误。目前最大的人体扫描仪为11.7T，故c)错误。更高场强的扫描仪孔径更小，而非更大，故d)错误。链接至问答讨论

  9. 以下哪种材料是顺磁性的？
    1. 水
    2. 脂肪
    3. 骨骼
    4. 空气

    大多数生物组织（包括水、脂肪和骨骼）是弱抗磁性的。分子氧（O2）是顺磁性的，其效应超过了空气中其他成分N2和CO2的弱抗磁性。链接至问答讨论

  10. 被磁场弱斥的材料称为
    1. 顺磁性
    2. 抗磁性
    3. 超顺磁性
    4. 铁磁性

    抗磁性材料产生的内部极化（J）与外施磁场方向相反，因此b)正确。其他类别材料的极化方向与磁场相同，并被磁场吸引。链接至问答讨论

  11. 对于______材料，磁化率（χ）为负。
    1. 顺磁性
    2. 超顺磁性
    3. 抗磁性
    4. 铁磁性

    当内部极化（J）方向与主磁场（B）相反时，磁化率（χ）为负。根据定义，只有抗磁性材料具有负磁化率。链接至问答讨论

  12. 当电子自旋阵列通过量子交换相互作用连接起来时，铁磁性材料会形成磁________。
    1. 磁通线
    2. 磁极
    3. 矢量
    4. 磁畴

    交换相互作用是一种量子效应，其中未配对电子连接在一起形成单个磁畴，这些磁畴的行为类似于独立的小“磁铁”。链接至问答讨论

  13. 比较超顺磁性和铁磁性材料，哪项陈述是错误的？
    1. 铁磁性通常比超顺磁性更强。
    2. 移除磁化场后铁磁性仍然存在。
    3. 一旦移除外部场，超顺磁性仍然存在。
    4. 超顺磁性可被视为单畴粒子。

    铁磁性材料保留先前磁化的记忆（剩磁），而超顺磁性粒子是单畴的，但其磁化强度在移除磁化场后消失。链接至问答讨论

MR磁体

  1. 临床MR扫描仪最常见的配置是
    1. 开放式孔径超导
    2. 封闭式孔径超导
    3. 开放式孔径永磁
    4. 偶极电磁铁

    如今销售的MR扫描仪中超过90%是封闭式孔径（圆柱形）超导型。第二常见的是开放式孔径永磁型。 链接至问答讨论

  2. 美国食品药品监督管理局（FDA）允许用于成人的常规临床实践的最高场强是
    1. 3.0特斯拉
    2. 7.0特斯拉
    3. 8.0特斯拉
    4. 11.7特斯拉

    目前FDA对临床MR扫描仪的限制为成人及1个月以上儿童8.0 T；1个月以下儿童为4.0 T。商业制造的最高场强临床扫描仪为7.0 T，但存在高达11.7 T的人体研究型扫描仪。链接至问答讨论

  3. 第一家生产商用全身MRI扫描仪的公司是
    1. GE
    2. Fonar
    3. 西门子
    4. Technicare

    Raymond Damadian于1982年成立了第一家生产临床用MR扫描仪的公司FONAR。 链接至问答讨论

  4. 圆柱形封闭式孔径扫描仪中主磁场（Bo）的方向是
    1. 圆柱的纵向（沿主轴）
    2. 水平方向（横向于圆柱并与地板平行）
    3. 垂直方向（横向于圆柱并与地板垂直）
    4. 取决于开启的梯度，可以是任意角度

    圆柱形扫描仪的主磁场沿圆柱的长轴（螺线管）方向。答案d)明显错误，因为梯度不会改变（Bo）场的方向。 链接至问答讨论

  5. 以下哪项不是低场和中场（< 1.0 T）MR扫描仪的优势？
    1. 价格较低
    2. 杂散场较低
    3. 钆强化的检测能力提高
    4. 组织中的能量沉积较低

    在较低场强下，相对于组织的钆强化降低，这是一个缺点。 链接至问答讨论

  6. 以下哪项不是高场（≥ 1.0 T）MR扫描仪的优势？
    1. 更高的信噪比
    2. 更好地检测钙化和出血
    3. 金属植入物周围的伪影较小
    4. 更好的磁场均匀性

    金属物体周围的磁敏感性伪影在高场下比低场下更严重。 链接至问答讨论

  7. 以下给出了四种不同磁体的规格。哪一个具有最佳的均匀性？
    1. 在40 cm DSV上 <1 ppm
    2. 在20 cm DSV上 <1 ppm
    3. 在40 cm DSV上 >1 ppm
    4. 在20 cm DSV上 >1 ppm

    均匀性规格通常以在特定球体积（DSV）上的百万分之（ppm）磁场偏差来表示。因此，在最大DSV上具有最小ppm的磁体将是最均匀的——答案a)。链接至问答讨论

  8. 磁场均匀性差可能以下列方式影响图像质量
    1. 阴影伪影
    2. 空间畸变
    3. 脂肪抑制差
    4. 仅a)和b)
    5. 以上所有

    磁体均匀性差可能影响图像质量的多个方面，包括上述所有情况，加上模糊、强度损失、弯曲的层面轮廓和条纹伪影。 链接至问答讨论

  9. 关于被动匀场，以下哪项陈述是正确的？
    1. 其主要目的是校正由患者身体产生的场畸变。
    2. 铁磁材料不能用于被动匀场。
    3. 被动匀场受室温影响。
    4. 一旦场被校准并实现磁场均匀性，就可以移除被动匀场材料。

    被动匀场涉及放置铁磁元件（通常为板或棒）以校正空磁体（而非包含患者的磁体）的静态场不均匀性。因此a)和b)都是错误的。匀场元件由金属制成，因此匀场质量取决于扫描仪和室温，故答案c)正确。匀场元件在匀场完成后不会被移除，而是保留在磁体中，故d)错误。链接至问答讨论

  10. 关于超导性，以下哪项陈述是正确的？
    1. 如果温度足够低，所有元素都可以变得超导。
    2. 只有金属可以成为超导体。
    3. 超导线中心内部的磁场为零。
    4. 当温度低于转变温度（TC）时，导线的电阻线性降低到零。

    只有约一半的已知元素在低温下可以变得超导，因此a)错误。其中大多数是金属或金属合金，但一些非金属陶瓷也可能具有超导性。超导性的发生是瞬时的，一旦跨越转变温度，电阻立即降至零；因此答案d)错误。正确答案是c)，它描述了迈斯纳效应，即所有磁场从超导体内部完全排出，产生完美的抗磁性状态。 链接至问答讨论

  11. 几乎所有临床MR扫描仪主绕组中的超导成分是_____的合金。
    1. 铌（Nb）和钛（Ti）
    2. 铌（Nb）和铜（Cu）
    3. 镍（Ni）和钛（Ti）
    4. 镍（Ni）和铜（Cu）

    几乎所有临床超导扫描仪的主绕组都使用嵌入铜芯中的铌钛（NbTi）合金微丝。铜芯不是超导的，但其作用是支撑微丝，并在失超时用作电流分流器。 链接至问答讨论

  12. 典型超导MR扫描仪的低温恒温器包含以下所有，除了
    1. 液氦
    2. 液氮
    3. 主磁体绕组
    4. 超导匀场线圈

    虽然过去常见，但现代制冷系统已消除了对液氮的需求，因此当今MR扫描仪的低温恒温器中只存在液氦。 链接至问答讨论

  13. 液氦的温度约为
    1. 4 °K
    2. 0 °K
    3. −4 °K
    4. −400 °C

    用于冷却磁体主绕组的液氦温度仅比绝对零度高几度，约4 °K。这比维持NbTi合金丝超导性所需的转变温度低几度。等于或低于绝对零度（0 °K或−273.15 °C）的温度是不可能的，因此答案b)、c)和d)都是错误的。 链接至问答讨论

  14. MRI设施通常在门上张贴标志，写道：“警告！磁体始终处于开启状态。”此标志严格来说不适用于
    1. 电阻式磁体扫描仪
    2. 永磁体扫描仪
    3. 超导磁体扫描仪
    4. 该标志适用于所有类型的扫描仪，始终如此。

    只有FONAR仍然生产用于人体的电阻式（电磁铁型）MR扫描仪，在不使用时可以关闭。然而，所有永磁体和超导扫描仪始终保持全场强，因此该标志适用于几乎所有MRI设施。链接至问答讨论

  15. 按下MRI扫描仪房间门附近的“红色大按钮”
    1. 即使扫描正在进行，也立即开门
    2. 启动磁场的受控失超
    3. 关闭扫描仪和房间的所有电源
    4. 呼叫急救人员（911）并发出警报

    “红色大按钮”通常标记为“磁体停止”或“紧急停运”，按下时会启动磁场的受控失超。只有在危及生命的紧急情况下（例如患者被金属物体压在扫描仪中）才应使用。 链接至问答讨论

  16. 在磁体失超期间，为何应将患者和员工从扫描室撤离？
    1. 即使是少量气态氦也会导致眼睛灼伤和刺激。
    2. 可能发生窒息。
    3. 很可能发生严重冻伤。
    4. 释放的氦可能起火或爆炸。

    氦气无味无色，但可能充满扫描室导致窒息。幸运的是，失超管道通常会将几乎所有释放的氦气排出室外，因此据我所知，在医疗环境中从未发生过因失超导致的死亡。直接接触液氦会引起灼伤，但氦气雾不太可能造成眼部损伤或冻伤（尽管如果房间内气体浓度高，理论上可能发生）。链接至问答讨论

梯度

  1. 用于成像的磁场梯度通常以______为单位测量
    1. 毫特斯拉每米（mT/m）
    2. 高斯每秒（G/s）
    3. 特斯拉（T）
    4. 特斯拉每米每秒（T/m-s）

    磁场梯度定义为在特定距离上测量的磁场变化，因此唯一合适的单位是答案a)，毫特斯拉每米 = [磁场强度]/距离。梯度强度也以高斯/厘米（G/cm）报告。 链接至问答讨论

  2. 对于仰卧患者，在层面选择期间仅启用z-梯度来改变患者体内的磁场将产生
    1. 轴位层面
    2. 冠状位层面
    3. 矢状位层面
    4. 斜位层面

    施加单一梯度会导致质子频率沿梯度方向随位置变化。这种频率变化可用于空间编码。如果在层面选择和信号读出期间施加梯度，可以选择垂直于梯度方向的层面。 链接至问答讨论

  3. 在层面选择期间同时施加x-梯度和z-梯度会有什么效果？
    1. 图像将被扭曲。
    2. 会发生显著的层间串扰。
    3. 将产生一个斜位层面。
    4. 扫描仪将显示警告，提示不允许这种组合。

    在层面选择期间施加单一梯度会产生垂直于梯度轴的层面。同时施加两个（或三个）梯度会分别产生单斜（或双斜）层面。 链接至问答讨论

  4. 当y-梯度开启时，主（Bo）场的方向会发生什么变化？
    1. Bo场现在略微指向右侧。
    2. Bo场现在略微指向天花板。
    3. Bo场方向反转。
    4. Bo场保持指向其原始（z）方向。

    梯度不会产生使Bo左右或上下旋转/倾斜的横向分量。它们仅作用于产生Bo的z分量的面内“倾斜”。 链接至问答讨论

  5. 用于在圆柱形MR扫描仪中产生z-梯度的基本线圈配置称为
    1. 麦克斯韦对。
    2. 双鞍形。
    3. Golay。
    4. 指纹形。

    Z-梯度通常使用两个载有相反方向电流的环形线圈产生，这种配置称为麦克斯韦对。列出的其他线圈类型主要用于x-和y-梯度。 链接至问答讨论

  6. 关于涡电流，以下哪项陈述是错误的？
    1. 它们会产生多种图像伪影，包括重影和模糊。
    2. 它们是法拉第感应定律的一种表现形式。
    3. 它们尤其影响具有长TE的传统自旋回波序列。
    4. 它们会导致组织发热。

    涡电流由快速变化的磁场（尤其是成像梯度切换）产生，因此是法拉第定律的一种表现。它们通常出现在使用短TE的快速梯度回波或平面回波序列中；因此答案c)错误。它们普遍不受欢迎，会产生各种伪影和组织发热问题。 链接至问答讨论

  7. 关于有源屏蔽梯度，哪项陈述是正确的？
    1. 它们是减少超导系统中涡电流的最有效方法。
    2. 它们是减少超导系统中涡电流的最有效方法。

    被动屏蔽是一种减少杂散磁场的方法，因此a)在墙壁上衬铜以减少b)RF干扰是错误的。更高场强的安装更需要它，因此c)正确。现代扫描仪中的主动屏蔽技术减少了对被动方法的需求，故d)错误。 链接至问答讨论

    4. 铅

  14. MR扫描仪产生的杂散磁场
    1. 可以通过主动屏蔽消除。

  15. “5高斯线”是什么？
    1. 扫描仪内部x-和y-梯度强度相差小于5高斯（5 mT）的地方。
    2. MRI中心内，在此边界内信用卡会被消磁。
    3. 可能对佩戴某些起搏器的患者造成危险的杂散场线
    4. 扫描室内对患者安全但MR技师应避免跨越的杂散场线。

    5高斯线由美国食品药品监督管理局（FDA）设定为不应使不知情公众暴露的边界。该值基于这样一个事实：旧式起搏器中的干簧管暴露于此水平的杂散磁场下可能被触发，可能将患者的需求型起搏器转换为异步模式。应该认识到，它不仅仅是一条线，而是一个从扫描仪向外延伸的三维表面。因此，它可以延伸到扫描仪上方和下方的楼层以及侧面。 链接至问答讨论

  16. 关于ACR安全区1和2的陈述哪项是正确的？
    1. 区域1和2都位于5高斯线之外。
    2. 进入区域2之前需要进行患者安全筛查。
    3. 公众不应进入区域1；它仅供MR患者及其家属使用。
    4. 允许佩戴起搏器的患者进入区域2可能有风险。

    区域1对公众开放。进入通常从区域2开始受限，因为这里是进行安全筛查的地方。两者都位于5高斯线之外，对所有人都安全。 链接至问答讨论

  17. 关于ACR安全区3的陈述哪项是错误的？
    1. 除非患者已接受安全筛查，否则不应允许其进入区域3。
    2. 不得将铁磁物体带入该区域。
    3. MR操作员控制台位于该区域。
    4. 除非医务人员已接受MR安全培训，否则不应允许其进入该区域。

    区域3包括5高斯线以内的区域，因此所有患者和家属在进入前都需要接受筛查。区域3包括MR操作员控制台所在的位置。区域3的杂散场足够小，铁磁物体不会被推进扫描仪。然而，从区域3通常可以方便地直接进入扫描室（区域4），在那里可能发生飞铁危险。区域3不鼓励但也不禁止铁磁物体；它们当然不应被带到扫描室门附近。由于这些原因，所有医务人员在获准进入区域3之前都必须接受MR安全培训/教育。 链接至问答讨论

  18. 关于ACR安全区4的陈述哪项是正确的？
    1. 绝不允许陪同家属进入区域4。
    2. 区域4等同于装有MR扫描仪的房间。
    3. 区域4包括扫描仪、操作员控制台和设备室（梯度放大器所在处）。
    4. 区域3和区域4之间必须有一扇需要刷卡、钥匙或密码才能进入的门，并且除了运送患者时应保持关闭。

    区域4就是扫描室本身，因此b)正确而c)错误。如果家属经过适当筛查，可以被允许进入扫描室，因此a)错误。扫描室的门不上锁，在不进行扫描时常保持打开状态（尽管我们建议用带子拦住以防止意外进入）。铁磁材料不应带入区域4，因为被吸入扫描仪的风险很高。 链接至问答讨论

  19. 为什么距离MR扫描仪20米远的道路上的大型卡车可能会引起场地规划的关注？
    1. 它们的民用波段无线电以与MR信号相同的频率工作。
    2. 扫描仪磁场可能受到经过的卡车底盘中的致密铁的影响。
    3. 它们产生的物理振动会影响图像质量。
    4. 在此距离上，重型卡车交通根本无需担心。

    环境振动会显著影响扫描仪性能，在安装扫描仪之前，场地应进行振动测试。附近道路上重型卡车的频繁通过是可能的原因之一。其他振动源包括附近的空调设备、电机和建筑物电梯。来自民用波段无线电的RF干扰不应成为特殊问题，因为这些频率通常会被标准的RF屏蔽滤除。在20米的距离上，移动的金属不应引起静态场扰动；然而，如果卡车在10米内经过，这可能值得关注。 链接至问答讨论
