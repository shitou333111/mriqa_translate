MRI中的血流效应

  1. 在给定时间段内通过血管树某一点的血流容积（Q）以什么单位测量？
    1. cm/s
    2. cm/s²
    3. cm²/s
    4. cm³/s

    容积流量是单位时间的体积，以 cm³/s 为单位表示。 链接至问答讨论

  2. 通过一个直径为 2 cm、平均流速为 50 cm/s 的动脉，其近似容积流量（Q）是多少？
    1. 25 cm³/s
    2. 50 cm³/s
    3. 100 cm³/s
    4. 160 cm³/s

    血管的横截面积（A）为 πr² = π (1 cm)² = 3.14 cm²。所以容积流量（Q）= V x A = 50 cm/s x 3.14 cm² = 157 cm³/s。 链接至问答讨论

  3. 在血管的哪个部位血流速度最慢？
    1. 紧邻血管壁处
    2. 血管中心处
    3. 血管壁与中心之间的中点处
    4. 整个血管内的血流速度相同。

    在靠近血管壁处，流体剪切力和摩擦力最大，血流速度几乎为零。在管腔中心，血流速度最快。 链接至问答讨论

  4. 在心动周期中，哪条血管通常表现出明显的反向血流？
    1. 椎动脉
    2. 升主动脉胸部段
    3. 大脑中动脉
    4. 肾动脉

    在舒张期，随着冠状动脉充盈，胸主动脉有明显的反向血流。大脑和肾脏具有低血管阻力；因此，即使在舒张期，颈动脉和肾动脉血管也表现出较高的前向血流。 链接至问答讨论

  5. 哪条血管具有最高的峰值流速？
    1. 胸主动脉
    2. 腹主动脉
    3. 颈动脉
    4. 肾动脉

    升主动脉在主要血管中具有最高的平均峰值流速；典型值为 150-175 cm/s。远端主动脉和髂血管的血流减慢至 100-150 范围，而近端颈动脉、肱动脉和股浅动脉的峰值流速约为 80-110 cm/s。 链接至问答讨论

  6. 随着血管狭窄程度增加，峰值流速会发生什么变化？
    1. 它们降低。
    2. 它们增加。
    3. 它们保持不变。
    4. 它们先增加后降低。

    随着血管狭窄程度增加，其峰值流速最初增加（由于伯努利效应）。然而，在极高等级狭窄中，血管阻力变得非常高，以致于在达到完全闭塞前，流速和流量下降。 链接至问答讨论

  7. 在理想的层流中，血管横截面上速度的分布应为
    1. 线性
    2. 恒定不变
    3. 抛物线形
    4. 双曲线形

    层流是指流速在平行于血管壁的层流中呈可预测分布。理论上，在完全笔直、无分支、无非搏动性流动的血管中，其横截面上的速度分布应为抛物线形。 链接至问答讨论

  8. 在以下哪种解剖情况下，你不会预期出现涡流？
    1. 颈动脉狭窄远端
    2. 在近端（升）主动脉
    3. 在远端胸主动脉
    4. 在腹主动脉瘤内

    涡流是指从血管内中央流线分离出来的局部漩涡或停滞血流。这种涡流（也称为流动涡旋）经常发生在血管分叉处和狭窄区域远端。涡流在升主动脉中很突出，因为血液喷射流在每次心脏收缩时被排出并旋转。它们在远端胸主动脉较少见，因此正确答案是（c）。 链接至问答讨论

  9. 关于理想流体中的雷诺数（Re），哪项陈述是正确的？
    1. Re < 2000 的值预示着湍流。
    2. Re > 2500 的值预示着层流。
    3. Re 是一个速度，单位为 cm/s。
    4. Re 是一个无量纲参数。

    雷诺数（Re）是一个用于预测理想流体中流动特性的无量纲参数。（答案d正确）。Re 小于 2000 的值预示流动为层流，而大于 2500 的值通常表示流动将为湍流。 链接至问答讨论

  10. 在哪种情况下，你更可能看到湍流？
    1. 在小直径血管中
    2. 在血流速度快的血管中
    3. 在血液粘度高的血管中
    4. 在新近闭塞的血管残端中

    湍流的概率与雷诺数（Re）成正比，与血流速度和血管直径成正比，与血液粘度成反比。因此选项（b）正确。 链接至问答讨论

  11. 以下哪一项与流动相关的现象被认为是一种飞行时间（TOF）效应？
    1. 流入层面内的高血管内信号。
    2. 由于成像梯度平面内流动导致的信号丢失。
    3. 由于动脉瘤内湍流导致的信号丢失。
    4. 由于梯度矩归零导致血管内高信号。

    飞行时间（TOF）效应是指在给定脉冲序列期间，由于质子流入或流出成像容积而产生的信号变化。流入层面内的高血管内信号（选项a）是一种称为流动相关增强的 TOF 效应。其他选项都是自旋相位效应。 链接至问答讨论

  12. 飞行时间（TOF）效应的根本原因是：
    1. 流动自旋在进入成像层面时相位增加。
    2. 重复短 TR 脉冲对成像层面的部分饱和。
    3. 流入自旋的超极化。
    4. 90° 和 180° 脉冲对流动自旋的综合效应

    正确答案是（b）。流入成像层面的“新鲜”血液未受这些 RF 脉冲影响，因此处于完全磁化状态（未饱和，但未超极化）。当这团新鲜血液进入层面并受到第一组 RF 脉冲作用时，其信号明显强于层面内的组织（及其他血液）。 链接至问答讨论

  13. 关于高速信号丢失（“冲刷效应”），哪项陈述是错误的？
    1. 高速信号丢失被认为是一种飞行时间效应。
    2. 通常见于垂直于成像平面流动的动脉和较大静脉。
    3. 主要见于梯度回波成像序列。
    4. 它可能发生在成像层面的任何位置，不限于末端层面。

    答案（c）是错误的。高速信号丢失通常见于自旋回波序列，其中流动血液必须同时接收 90° 和 180° 脉冲才能产生信号。梯度回波序列使用单 RF 脉冲和梯度重聚。 链接至问答讨论

  14. 质子以恒定速度（v）在恒定梯度场强（G）中运动时间（t）所获得的总相位与以下哪个量成正比？
    1. t
    2. t²
    3. v²
    4. G²

    运动自旋所经历的相位偏移与其速度、所施加梯度的强度以及在该梯度中运动时间的平方成正比。 链接至问答讨论

  15. 一位放射科医生在解读疑似卒中患者的头部 MRI 时写道：“基底动脉可见流空效应。”这意味着什么？
    1. 基底动脉内无血流。
    2. 基底动脉内可能存在一些血流，但非常微弱。
    3. 基底动脉内含有流动的血液。
    4. 基底动脉内存在血栓。

    “流空效应”是一个广泛使用的术语，指包含快速流动血液的血管中信号较低，这是由于飞行时间和自旋相位效应的共同作用。尽管大多数专家熟悉这个术语，但它可能会在医疗团队中不太熟悉的成员中引起混淆，他们可能错误地将其解释为无血流。 链接至问答讨论

  16. 以下哪项陈述不是梯度回波序列可用于保持或增强流动血液信号的原因？
    1. 梯度的自旋重聚是层面选择性的。
    2. GRE 相对较短的 TE 可最大限度地减少 T2* 去相位。
    3. GRE 的顺序模式采集使每个层面都成为具有流动相关增强的“流入层面”。
    4. GRE 的中等到大翻转角可饱和静态组织并增强血液的 T2/T1 对比。

    除了（a）以外，其他都是正确的。在常规 SE 成像中，流动相关信号丢失的部分原因是自旋在 90° 和 180°（重聚焦）脉冲之间移出了层面。在 GRE 成像中，重聚焦是通过非层面选择性的梯度反转实现的。然而，这种非选择性重聚焦不会导致流入自旋的高信号；它只是防止它们经历 TOF 信号损失。 链接至问答讨论

  17. 以下哪项不是流动补偿（梯度矩归零）技术的局限性？
    1. 最小 TE 增加
    2. 能量沉积增加（SAR）
    3. 无法校正非恒定速度或加速度
    4. 梯度应力增加

    除了（b）外，其他都是正确的。没有使用额外的 RF 脉冲，只使用额外的梯度叶，因此 GMN 不会沉积额外能量。 链接至问答讨论

  18. 在哪种情况下最可能观察到重聚的错配血流信号？
    1. 在平行于相位编码轴的血管中。
    2. 在垂直于相位编码轴的血管中。
    3. 在 TE 值非常短的序列中。
    4. 在使用流动补偿（梯度矩归零）的序列中。

    血流错配最常见于在平面内、与频率和相位编码轴斜行的血管，使用长 TE 并采用梯度矩归零时。 因此（d）是正确答案。 链接至问答讨论

磁共振血管成像 - I

  1. 以下哪种方法被认为是“黑血”MRA技术？
    1. 飞行时间 MRA
    2. 钆增强 MRA
    3. SSFP MRA
    4. 磁敏感加权 MRA

    黑血 MRA 技术抑制流动血液的信号，同时保持周围静态组织的高信号，从而使血管呈现黑色。在列出的选项中，只有磁敏感加权 MRA 是一种黑血技术。 链接至问答讨论

  2. 以下哪种方法不是“亮血”MRA技术？
    1. 相位对比 MRA
    2. 双反转恢复 MRA
    3. 动脉自旋标记 MRA
    4. 费鲁莫西醇增强 MRA

    单次、双次甚至三次 180° 反转脉冲可用于根据血液和其他组织（如脂肪或心肌）各自的 T1 值来抑制其信号。黑血 IR 方法最常用于心脏和血管壁成像。 链接至问答讨论

  3. 与亮血 MRA 技术相比，以下哪项不是黑血 MRA 的优势？
    1. 较少高估狭窄程度
    2. 更好地勾画血管壁
    3. 对低流速状态更敏感
    4. 更少的搏动伪影

    与亮血技术相比，黑血 MRA 具有一些独特的优势。在亮血 MRA 上导致不必要信号丢失和狭窄程度高估的流动分离和湍流，在黑血 MRA 上反而能提高图像质量。由于管腔内信号被抑制，含有“黑血”的血管不会产生搏动（重影）伪影。此外，管腔内信号的缺失使得血管壁（或心腔）得以清晰勾画。由于这些原因，黑血技术主要用于心脏成像和血管壁疾病（动脉粥样硬化斑块和夹层）的评估。虽然黑血技术是成像高流量血管的绝佳方法，但对较慢的流速状态敏感性较低。“黑”血难以与邻近的低信号非血管结构（如钙化、皮质骨和空气）区分。 链接至问答讨论

  4. 在 2D-TOF MRA 中使用所谓的“流动饱和”脉冲的目的是什么？
    1. 减少静脉污染
    2. 最大化静态组织与血液之间的对比
    3. 最小化相位弥散引起的信号损失
    4. 增加流入自旋的信号

    这些饱和脉冲用于抑制通常会流入每个层面的静脉信号。 链接至问答讨论

  5. 在 2D-TOF-MRA 序列中使用中等到大翻转角（30°−60°）的目的是什么？
    1. 减少静脉污染
    2. 最大化静态组织与血液之间的对比
    3. 最小化相位弥散引起的信号损失
    4. 增加流入自旋的信号

    较大的翻转角使成像容积内的静态组织产生更多的磁饱和，从而最大限度地提高与流入血液的对比度。 链接至问答讨论

  6. 关于 TOF MRA 序列的最佳参数选择，以下哪项陈述是正确的？
    1. TR 应非常长，以确保最大程度的流入增强。
    2. TE 应非常长，以通过 T2 效应增加血管信号。
    3. 层面方向应垂直于血流方向。
    4. 翻转角（α）应非常小，以增加组织饱和。

    只有（c）是正确的。非常长的 TR 对改善流入增强的益处被背景组织信号增加的缺点所抵消。TE 应短以减少自旋相位损失。翻转角应为中到大，以减少静态组织的信号。 链接至问答讨论

  7. 平均速度为 50 cm/s 的动脉血液垂直流入一个 2 cm 厚的层面。当 TR 为多少时，流动相关增强达到最大？
    1. 10 ms
    2. 20 ms
    3. 30 ms
    4. 40 ms

    当流入的血液完全替换给定层面中现有的血液时，流动相关增强最大。当血流垂直于层面时，最大增强发生在 V ≥ d / TR，或等效地，TR ≥ d /V 时。对于 d = 2 cm 和 V = 50 cm/s，这意味着 TR ≥ 2/50 s = 0.04 s = 40 ms 链接至问答讨论

  8. 关于磁化传递增强 MRA，哪项陈述是错误的？
    1. 它对飞行时间和相位对比 MRA 均有效
    2. 它对非对比和对比增强 MRA 均有效
    3. 小血管细节得到改善
    4. 图像采集时间略有延长。

    MT-MRA 仅用于 TOF 和对比增强 MRA。它对 PC-MRA 没有益处。 链接至问答讨论

  9. 关于可变（倾斜）脉冲，哪项陈述是正确的？
    1. 它们在 2D 和 3D TOF MRA 中均有用。
    2. 它们的翻转角设计为沿血流方向线性减小。
    3. 它们旨在使层面深处的静态组织产生更大的饱和。
    4. 它们旨在增加层面深处流动血液的绝对信号。

    可变（倾斜）RF 脉冲解决了延伸到 3D-TOF 层面深处的血管 MRA 信号逐渐丢失的问题。（它们不用于 2D TOF）。翻转角设计为沿血流方向增加，使层面深处的静态组织产生更大的饱和。（答案c正确）。它们也会轻微抑制血液，但抑制程度不如静态组织。 链接至问答讨论

  10. MOTSA MRA 常见的一种伪影被称为
    1. 百叶窗伪影
    2. 阶梯状伪影
    3. 速度混叠伪影
    4. 振铃（Maki）伪影

    由于每个 MOTSA 层面块是在不同时间采集的，相邻层面块的位置和信号强度可能无法精确配准。这就产生了该技术特有的所谓“百叶窗”伪影。 链接至问答讨论

  11. 2D-TOF MRA 常见的一种伪影被称为
    1. 百叶窗伪影
    2. 阶梯状伪影
    3. 速度混叠伪影
    4. 振铃（Maki）伪影

    阶梯状伪影常见于 2D-TOF MRA，因为患者在层面间移动。 链接至问答讨论

  12. 在锁骨下动脉盗血综合征中，两条椎动脉之一的血流发生逆转。在颈部 2D-TOF MRA 上，这条血流逆转的椎动脉会呈现什么表现？
    1. 其表现将与对侧（正常血流）椎动脉相同。
    2. 它将显得比其对侧椎动脉反常地更亮。
    3. 它将显得比对侧椎动脉略暗。
    4. 它将无法显示，提示闭塞。

    2D-TOF 序列利用位于成像层面上游的流动饱和脉冲来消除静脉污染。在颈部，这些脉冲能有效消除颈静脉和其他深静脉的信号，但也会消除方向已逆转的动脉血流信号。因此（d）是正确答案。 链接至问答讨论

  13. 为什么对于脑出血患者，头部相位对比（PC）MRA 可能优于 TOF MRA？
    1. PC MRA 的空间分辨率高得多。
    2. PC MRA 已被证明在检测动脉瘤方面更准确。
    3. 血肿的 T1 透过效应不影响 PC MRA，但可能遮挡 TOF MRA。
    4. PC MRA 的采集时间比等效 TOF 技术短得多。

    TOF MR 血管图像使用最大强度投影（MIP）算法显示。除了明亮的血管外，任何其他具有高信号强度的物质都会“透过”并“污染” MIP 图像。由于基础的 TOF MRA 脉冲序列是 T1 加权的，短 T1 物质（血肿、钆和脂肪）是造成这种现象的主要原因。然而，PC MRA 依赖于运动引起的相位差，不受“透过”效应的影响。 链接至问答讨论

  14. 相位对比 MRA 的基本原理基于
    1. 双极流动编码梯度
    2. 单极流动编码梯度
    3. 激发之间的射频相位循环
    4. 沿相位编码轴使用斜率恒定的线性梯度

    PC MRA 利用双极梯度，沿一个或多个需要获取流动信息的方向施加。经受这种梯度对的静态自旋不会经历净相位偏移，但运动自旋将具有与其速度成正比的净相位偏移。 链接至问答讨论

  15. PC MRA 的 VENC（速度编码）参数的单位是
    1. 无量纲
    2. cm/s
    3. cm²/s
    4. cm/s²

    VENC 是一个速度，因此正确答案是（b），cm/s。 链接至问答讨论

  16. 如果 PC MRA 中的 VENC 设置为 60 cm/s，则调整双极梯度的强度和持续时间，使以 60 cm/s 流动的血液具有的相位偏移为
    1. +60º
    2. +90º
    3. +180º
    4. +360º

    对于给定的 VENC，调整双极梯度，使该速度水平的血流被赋予 +180º 的相位偏移。 链接至问答讨论

  17. 如果 VENC 设置为 50 cm/s，而实际前向血流速度为 75 cm/s，PC MRA 上显示的速度是多少？
    1. +50 cm/s
    2. +75 cm/s
    3. +25 cm/s
    4. −25 cm/s

    如果选择的 VENC 值为 50 cm/s，则调整双极梯度，使 +25 和 +50 cm/s 的血流分别被赋予 +90° 和 +180° 的相位。如果实际速度为 +75 cm/s，该血流将被映射到 +270°，该值无法与 −90° 的相位偏移区分。计算机不会将 +75-cm/s 的血流表示为其实际速度，而是将其指定为相反方向上 25 cm/s 的血流！ 链接至问答讨论

  18. 使用 VENC 设置正确的双极梯度的 PC MRA 确定三个主方向上的速度分别为 vx = −52 cm/s, vy = +24 cm/s, 和 vz = +72 cm/s。当显示为幅度（速率）图像时，每个体素的亮度与以下哪个值成正比？
    1. 148 cm/s
    2. 92 cm/s
    3. 72 cm/s
    4. 44 cm/s

    根据定义，速率 = √{v²x + v²y + v²z} = √{(−52)² + (+24)² + (+72)²} = √{8464} = 92 cm/s。 链接至问答讨论

  19. 关于 PC-MRA 速度测量与中大型血管多普勒测量相比的准确性，两种测量通常彼此的百分比差异在多少范围内？
    1. 小于 1%
    2. 约 5%
    3. 15% - 25%
    4. 大于 25%

    两种测量通常在彼此相差约 5% 的范围内。 链接至问答讨论

  20. 在以下哪种情况下，PC MRA 的速度测量最准确？
    1. 成像平面垂直于血流方向
    2. 像素大小接近血管直径
    3. 湍流
    4. 使用每个心动周期 8 帧对各大血管进行采样。

    当血管垂直于成像平面时，流速测量最准确。当像素大小超过血管直径的三分之一时，可能会发生显著的部分容积效应，导致流量和峰值速度的低估。当速度在大小、方向或两者上突然变化时（如在湍流中），流量测量的准确性下降。如果整个心动周期的数据采样率过低，可能无法测量到峰值流量。对于大血管成像，每个心动周期至少应采样 16 帧。 链接至问答讨论

磁共振血管成像 - II

  1. 关于门控 3D-FSE MRA 技术，哪项陈述是错误的？
    1. 它们可以门控到心电图或外周脉搏。
    2. 可获得舒张期和收缩期的配准图像。
    3. 将舒张期和收缩期图像相加得到血管造影图。
    4. 图像对比基于血液的长 T2。

    该序列门控到心电图或外周脉搏。可获得舒张期和收缩期的配准图像。在舒张期，动脉和静脉的信号均较高，反映了血液的长 T2。在收缩期，静脉信号保持较高，但动脉信号因流动相关信号损失而下降。从舒张期图像中减去（而非相加）收缩期图像可得到纯动脉图像。 链接至问答讨论

  2. 以下哪项不是 3D-FSE MRA 的缺点？
    1. 需要使用钆对比剂。
    2. 高估狭窄程度
    3. 血管搏动伪影
    4. 对心律失常敏感

    选项（a）是错误的。3D-FSE MRA 的主要优点是不需要对比剂，这一特性对于肾功能不全的患者（接受钆剂可能有肾源性系统性纤维化风险）可能很重要。 链接至问答讨论

  3. 平衡稳态自由进动（b-SSFP）MRA 中的血管内对比主要与血液的哪个特性相关？
    1. 高自旋密度
    2. 长 T1
    3. 长 T2
    4. 高 T2/T1 比

    b-SSFP MRA 利用了血液与大多数其他组织相比具有相对较高的 T2/T1 比这一差异。 链接至问答讨论

  4. b-SSFP MRA 序列通常不用于颅内成像的主要原因是什么？
    1. 其空间分辨率不足以分辨主要颅内血管。
    2. 颅内血管的信号被附近脑脊液的高信号所掩盖。
    3. 颅内血管通常表现出湍流，这会干扰 SSFP。
    4. 颅底的磁敏感伪影在大部分大脑上产生令人困扰的斑马条纹伪影。

    与血液一样，CSF 也具有高的 T2/T1 比，因此在 SSFP MRA 序列上会显得明亮。由于颅底动脉实际上“浸泡”在 CSF 中，这是这些序列不常用于颅内成像的主要原因。 链接至问答讨论

  5. 在流入增强、IR 准备的 3D-SSFP MRA 中使用的反转脉冲的目的包括以下所有，除了
    1. 抑制含水的背景组织信号。
    2. 抑制脂肪信号
    3. 抑制静脉血信号
    4. 超极化流入的动脉自旋。

    反转脉冲允许动脉流入因背景和静脉抑制而相对增强，而不是因为超极化。 链接至问答讨论

  6. 在动脉自旋标记（ASL）准备的 MRA 中，两个反转脉冲的功能是什么？
    1. 第一个抑制背景中的水，第二个抑制背景中的脂肪。
    2. 第一个非选择性地抑制背景，第二个抑制静脉信号。
    3. 第一个非选择性地标记流入的自旋，第二个选择性地抑制背景。
    4. 第一个非选择性地抑制背景，第二个选择性地标记流入的自旋。

    使用 ASL 的最简单的 MRA 形式称为流出法。该序列以两个 180°-RF 脉冲紧密相连开始。第一个 180°-脉冲是非选择性的，意味着它反转整个背景，无论位置如何。第二个 180°-脉冲是空间选择性的，用于恢复“新鲜”血液将要流入区域的磁化。在反转时间（TI）间隔期间，这些完全磁化的（“新鲜”）血液将进入感兴趣的解剖区域。然后使用 3D 平衡 SSFP 或 FSE 序列启动信号生成以产生 MRA。 链接至问答讨论

  7. 关于静息间隔单次激发（QISS）MRA，以下哪项陈述是错误的？
    1. 它不需要心电门控。
    2. 它不使用钆对比剂。
    3. 它主要用于四肢 MRA
    4. 未饱和的血液在静息间隔（QI）期间进入成像层面。

    QISS 是一种心电门控、非对比流入技术，与 2D 飞行时间（TOF）和流入增强 SSFP MRA 有相似之处。它专为外周 MRA 设计。新鲜（完全磁化/未饱和）血液在静息间隔期间进入层面，通过平衡 SSFP 序列产生动脉信号。 链接至问答讨论

  8. 关于对比增强（CE）-MRA，以下哪项陈述是错误的？
    1. 如果在注射后过早扫描，可能导致血管显示不充分。
    2. 注射后过晚扫描可能导致静脉污染。
    3. 如果在第一次采集中对团注的时机把握不当，可以等待 10 分钟后重复注射。
    4. 信号生成通常使用具有短 TR 和 TE 的 3D T1 加权扰相梯度回波序列进行。

    理想情况下，CE-MRA 应在对比剂到达血管时或接近其峰值浓度时进行。太早则看不到血管；太晚则可能出现静脉污染。你只有一次机会正确执行，并且不能重复该序列，直到一两天后钆从体内清除。 链接至问答讨论

  9. 什么是 MRA 的透视触发？
    1. 一种使用 X 射线透视信息来协助钆团注时间安排的方法。
    2. 一种估计小剂量（1-2 ml）钆测试团注到达时间的方法。
    3. 一种估计对比剂在整个成像容积中分布的 3D 相位对比方法。
    4. 一种在全部对比剂团注通过期间提供“类透视”MR 图像的方法，此时可开始 MRA 采集。

    在透视触发中，在获取感兴趣动脉的类透视图像（使用快速 2D 梯度回波技术）的同时，给予全部对比剂团注（约 20 mL）。技术人员随后可以启动 MRA 采集，或者可以自动完成触发。 链接至问答讨论

  10. 大多数 3D-CE-MRA 序列首选的 k 空间采样方法称为
    1. 椭圆中心顺序
    2. 线性顺序
    3. 顺序顺序
    4. 随机顺序

    椭圆中心顺序现在是大多数 CE-MRA 检查首选的 k 空间采样方法。线性方法可能仍用于较旧的设备和远端肢体的 MRA，其中对比剂团注到达时间可能延长或难以预测其确切时机。 链接至问答讨论

  11. 现代时间分辨 MRA 序列（如 TRICKS 和 TWIST）中使用哪种类型的 k 空间采样？
    1. 之字形
    2. 笛卡尔
    3. 径向
    4. 螺旋

    现代时间分辨 MRA 技术通常使用径向采样方案，以分段的圆形或椭圆形“圆柱体”获取 3D k 空间。 链接至问答讨论

  12. 哪项技术组件不是现代时间分辨 MRA 序列（如 TRICKS 和 TWIST）中使用的特征？
    1. 3D-SSFP 采集
    2. 相位和读出共轭对称
    3. 椭圆中心顺序
    4. 并行成像

    选项（a）是错误的。时间分辨 MRA 方法的核心是 3D-扰相 GRE 序列，具有薄层、非常短的 TR 和 TE、小翻转角、使用读出和相位共轭对称、并行成像采集以及层面方向的零插值填充。 链接至问答讨论

  13. 导致 CE-MRA 上锁骨下动脉“假性狭窄”伪影的原因是什么？
    1. 锁骨下静脉压迫锁骨下动脉
    2. 邻近锁骨下静脉中残留的钆引起的磁敏感效应
    3. 在其与腋动脉交界处的湍流
    4. 由于椎动脉逆行流入锁骨下动脉引起的反流

    在对比剂注射同侧的锁骨下动脉可能出现类似狭窄的局灶性不规则。这是由邻近锁骨下静脉中残留的钆引起的磁敏感（T2*）效应，左侧比右侧更常见。 链接至问答讨论

  14. CE-MRA 上振铃（Maki）伪影的原因是什么？
    1. Gibbs（截断）现象
    2. 在对比剂团注到达之前扫描了 k 空间的中心区域
    3. 在对比剂团注到达后过长时间扫描了 k 空间的中心区域
    4. 血管搏动

    如果在对比剂团注到达之前扫描 k 空间的中心区域，血管中部将显示为暗色，仅边缘有强化。过早扫描会产生这种最初由 Maki 等人描述的伪影。 链接至问答讨论

心脏MRI - 解剖

  1. 在短轴位上通常显示哪些心腔？
    1. RA 和 RV
    2. RA 和 LA
    3. LA 和 LV
    4. RV 和 LV

    短轴位通常用于显示左心室和右心室。 链接至问答讨论

  2. 在所谓的“拐杖糖”视图上最佳显示的结构是什么？
    1. 胸主动脉
    2. 回旋支动脉
    3. 左主干冠状动脉
    4. 右冠状动脉

    拐杖糖视图是一个斜矢状切面，展示了升主动脉、横主动脉、降主动脉及其近端分支。 链接至问答讨论

  3. 如果进行负荷试验，为什么可能需要在MRI前建立两条静脉通路？
    1. 第二条作为备用，以防第一条静脉通路失败。
    2. 检查期间所需的液体和药物总量超过单根普通孔径静脉通路的能力。
    3. 钆对比剂和腺苷不能通过同一条通路给药。
    4. 检查期间需要第二条静脉通路进行采血。

    钆和腺苷不能通过同一条管路给药，因此需要两条独立的静脉通路。如果使用瑞加德松或多巴酚丁胺，则可以选择使用单独的管路。 链接至问答讨论

  4. 在进行负荷 MRI 之前，应停用以下哪些食物和药物至少 12-24 小时？
    1. 硝酸甘油
    2. 钙通道阻滞剂
    3. 含咖啡因的饮料
    4. 奎尼丁

    应告知患者在扫描前 12-24 小时内不要摄入咖啡、茶、含咖啡因的苏打水、能量饮料和巧克力等兴奋剂。这些物质会干扰药物负荷试验的效果和检查结果的解读。 链接至问答讨论

  5. 患者被置于 MRI 中时观察到的最常见心电图变化是什么？
    1. R 波降低
    2. R 波倒置
    3. T 波抬高
    4. ST 段抬高

    尽管上述所有现象都可能发生，但 T 波抬高最为常见。这种抬高可能非常显著，以至于 T 波实际上变得比 QRS 波群更大。 链接至问答讨论

  6. 上述心电图变化的原因是什么？
    1. 胸廓血流感应出的电流
    2. 磁敏感效应
    3. 心脏副传导通路的刺激
    4. 心电图导联中感应出的电流

    这种磁场对心电图的影响并非源自心脏本身，而是代表了降胸主动脉内血液在磁场中流动所感应出的叠加电压。导电液体在磁场中流动时感应出电压的现象称为磁流体动力学（MHD）效应。 链接至问答讨论

  7. 心脏触发主要使用哪一心电图波？
    1. P 波
    2. Q 波
    3. R 波
    4. T 波

    心脏门控通常通过检测 R 波来进行，因为 R 波通常是心电图最突出的特征。R 波对应于心室收缩的开始。 链接至问答讨论

  8. R 波对应什么心脏事件？
    1. 心房收缩开始
    2. 心室收缩开始
    3. 心室收缩峰值
    4. 心室舒张开始

    R 波对应于心室收缩的开始。 链接至问答讨论

  9. 如果心率是 100 bpm，R-R 间隔的长度是多少毫秒？
    1. 60 ms
    2. 100 ms
    3. 600 ms
    4. 1000 ms

    在 1 分钟 = 60 秒内，有 100 次由 R 波标记的心跳。因此 R-R 间隔为 60s /100 = 0.6 s 或 600 ms。 链接至问答讨论

  10. 对于上述心率为 100 bpm 的患者，以下哪项 TR 值对于前瞻性门控序列是不可用的？
    1. 600 ms
    2. 900 ms
    3. 1200 ms
    4. 1800 ms

    在前瞻性门控序列中，重复时间（TR）不能自由设置，必须是平均 R-R 间隔的倍数。因此对于心率 = 100 的患者，TR 的选择只能是 600 ms、1200 ms、1800 ms 等。 链接至问答讨论

  11. 继续上述心率 = 100 bpm 的患者示例，如果选择门控因子为 4，TR 会是多少？
    1. 125 ms
    2. 600 ms
    3. 1200 ms
    4. 2400 ms

    选择的 R-R 倍数称为门控因子。因此，如果平均 R-R 间隔为 600 ms，相应的 TR 值为 600 x 4 = 2400 ms。 链接至问答讨论

  12. 在门控研究中，第一个 R 波与数据采集开始之间的时间称为
    1. 触发窗口
    2. 采集窗口
    3. 触发延迟
    4. 静息间隔

    触发延迟是第一个 R 波与数据采集开始之间的时间间隔。 链接至问答讨论

  13. 以下哪项不是回顾性门控相对于前瞻性门控的优势？

    2. 可以剔除心律失常期间采集的数据。
    3. 可以剔除患者运动期间采集的数据。
    4. 可以校正心电图中的磁流体动力学畸变。

    除了（d）之外，其他都是真实的优势。 链接至问答讨论

  14. 对于心脏成像，导航脉冲最常放置在
    1. 右侧膈肌穹窿上
    2. 左侧膈肌穹窿上
    3. 左心室上
    4. 胸壁上

  15. 关于心脏成像的双反转恢复序列，以下哪项陈述是正确的？
    1. 第一个 180° 脉冲是空间选择性的，而第二个 180° 脉冲是非选择性的。
    2. 血液抑制需要外部血液流入并置换层面内初始的血液。
    3. 血液和脂肪信号均被抑制。
    4. 当心率缓慢时，需要短的 TI 值。

    第一个反转脉冲是空间非选择性的，而第二个脉冲是空间选择性的，因此（a）错误。外部血液的流入是 DIR 正常运作的关键要求，因此（b）正确。DIR 不抑制脂肪，因此（c）错误。最后，当心率缓慢时，需要长（而非短）的 TI 值。 链接至问答讨论

  16. 与双反转恢复相比，三反转恢复在心脏成像中的优势是
    1. 抑制心包和纵隔脂肪
    2. 更短的成像时间
    3. 选择 TI 值的自由度更大
    4. 改善的 T1 对比

    使用第三个反转脉冲抑制脂肪是三反转恢复的主要优势。它也有局限性，包括更长的成像时间、更少的 TI 值选择以及 T1 对比被 T2 对比取代。 链接至问答讨论

    1. 15
    2. 17

    4. 21
