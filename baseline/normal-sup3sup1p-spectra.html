³¹P 谱峰

在 ³¹P 谱中能看到哪些峰，它们代表什么？
³¹P 核磁共振谱与 ¹H 谱看起来截然不同，如下方正常脑组织的谱图所示。
正常脑组织的 ³¹P 谱
          缩写: 
磷酸单酯 (PME)，磷酸乙醇胺 (PE)，磷酸胆碱 (PC)，无机磷酸盐 (Pi)，磷酸二酯 (PDE)，甘油磷酸乙醇胺 (GPE)，甘油磷酸胆碱 (GPC)，磷酸肌酸 (PCr)，三磷酸腺苷 (ATP)，烟酰胺腺嘌呤二核苷酸 (NAD)，尿苷二磷酸葡萄糖 (UDPG)
中央（通常也是最高的）峰来自能量代谢物磷酸肌酸 (PCr)。由于其突出性、生物学重要性以及相对稳定性，在活体研究中，PCr 被指定化学位移 (δ) = 0。PCr 右侧的代谢物具有负化学位移；左侧的则为正。³¹P 代谢物分布广泛，跨度范围约为 30 ppm。相比之下，临床 MRS 中使用的 ¹H 峰则集中在 0 到 5 ppm 的狭窄区间内。
PCr 及其右侧的所有代谢物对细胞氧化代谢都至关重要，可被视为“能量峰”。除了 PCr 之外，ATP（三磷酸腺苷）的三个磷酸基团产生的清晰共振峰也稳定可见，分别标记为α、β、 和γ。α-ATP 峰右侧肩部的小凹陷对应于与能量相关的二磷酸盐（NAD 和 UDPG）。
PCr 可逆地将其高能磷酸基团提供给二磷酸腺苷 (ADP)，生成 ATP 和肌酸 (Cr)。该反应由肌酸激酶催化，可写为：
PCr + ADP  ↔  ATP + Cr
当 ATP 分解供能时，它被重新转化为 ADP，并释放无机磷酸盐 (Pi)，反应如下：
ATP  →  ADP + Pi + 能量
通常在 δ = +5 ppm 附近可以看到一个小的 Pi 峰。（其确切位置取决于组织 pH 值，甚至可以通过测量 PCr 和 Pi 之间的化学位移差来计算该 pH 值。）ADP 的储存量太低，无法产生可检测的谱峰，但可以通过计算比值 [Pi]/[PCr] 来估算 ADP 的浓度。
除 Pi 外，PCr 左侧的所有峰均来源于与膜磷脂相关的代谢物。这些“膜峰”可分为两大类：磷酸单酯 (PME)和磷酸二酯 (PDE)。虽然不完全准确，但通常认为 PME 主要反映膜合成，而 PDE 反映膜分解。因此，PME/PDE 比值有时被用作净磷脂代谢的指标。
三磷酸腺苷 (ATP)
  • 主要的能量储存和转运化学物质
  • 来自 3 个磷酸基团 (α、 β、 γ) 的共振峰，位于 δ ≈ −7.5、 −16.0 和 −2.5 ppm
  • P-P 耦合常数 (J) 约为 16 Hz
  • α 和 γ 是双重峰，各自被 β 位上的 ³¹P 裂分
  • β 是 1:2:1 三重峰，被 α 和 γ 裂分
  • 在大脑中，高达 25% 的 ATP 信号来自其他核苷三磷酸 (NTP)，包括鸟苷、尿苷和胞苷三磷酸
  • ADP 和 AMP 的共振峰太小而无法检测
磷酸肌酸 (PCr)
  • 被称为“ATP 补充剂”，PCr 储存高能磷酸基团，用于将 ADP 转化为 ATP
  • 指定化学位移 δ = 0.0 ppm
  • 大多数谱图中最高的峰，尤其是骨骼肌
  • 肝脏或肾脏中不存在
无机磷酸盐（Pi）

HPO4−2

H2PO4−
  • 生物系统中³¹P的离子库
  • H2PO4:HPO4比值依赖于pH值；由于快速交换，仅观察到单个平均值
  • 可见子峰位于δ ≈ 4.8 和 5.2，代表细胞外和细胞内Pi
  • Pi和PCr之间的Δδ 可估算组织pH值
磷酸二酯（PDE）
甘油磷酸胆碱（GPC）
甘油磷酸乙醇胺（GPE）
  • δ = 2.5−3.5 ppm范围内的一组共振峰，包括交联膜磷脂的宽峰以及可移动形式的离散峰：
  • 甘油磷酸胆碱（GPC）位于δ = 3.0，是胞浆中胆碱储存的主要来源，也是重要的肾渗透物质
  • 甘油磷酸乙醇胺（GPE），一个较小的PDE峰，位于GPC左侧δ = 3.5，是一种膜分解产物
磷酸单酯（PME）
磷酸胆碱（PC）
磷酸乙醇胺（PE）
  • δ = 6−7 ppm范围内的一组共振峰，反映膜合成，在脑和肝脏谱线中尤为突出。通常可以识别出至少两个离散峰
  • 磷酸乙醇胺（PE）位于δ = 6.8 ppm，一种细胞内化合物，在未成熟脑中常为最高峰
  • 磷酸胆碱（PC）位于δ = 6.2 ppm，胆碱的主要前体和储存形式
二磷酸盐（NAD、UDPG）
烟酰胺腺嘌呤二核苷酸（NAD）
尿苷二磷酸葡萄糖（UDPG）
  • 烟酰胺腺嘌呤二核苷酸（NAD）是细胞呼吸的辅酶，通过交替氧化（NAD+）或还原（NADH）作为电子（氢）载体
  • 在高质量去耦谱线中，可见为α-ATP峰右侧肩部的一个小切迹，位于δ = − 8.2 ppm
  • 尿苷二磷酸葡萄糖（UDPG）是碳水化合物代谢的关键中间体，是糖原、多糖和鞘糖脂的前体
  • 尤其在肝脏和脑中可见一个小峰，位于NAD右侧，δ = − 9.7 ppm

高级讨论（显示/隐藏）»

³¹P与¹H一样，具有核自旋I = ½。³¹P和¹H都有两个量子化自旋态，因此被认为是偶极核。两者在整个表面具有均匀的电荷分布，呈球形。这意味着它们干扰探测电磁场时与方向无关，并具有强而锐利的MR信号。
