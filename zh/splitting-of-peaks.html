J 耦合

为什么有些谱峰会分裂成更小的峰，而有些则不会？ 
两个原子核之间的 J 耦合
通过电子媒介传递
谱峰分裂成双重峰、三重峰或更高阶多重峰，是由于同一分子上两个核自旋之间通过电子媒介相互作用的结果。这种现象被称为 J 耦合，其机制在 高级讨论 中有更完整的描述。 简而言之，一个原子核的自旋会使其价电子极化，这种极化通过化学键传递，从而影响远处原子核的自旋。

一个谱线要通过 J 耦合发生分裂，必须至少满足两个条件：1) 原子核之间必须相对靠近，通常相隔不超过 3-4 个化学键；2) 原子核必须在化学上可区分。后一个标准意味着，例如甲烷 (CH4) 中的 四个 ¹H 原子核将产生一条单一的（未分裂的）谱线，因为它们 在化学和磁学上是等同的。
​如果你上过大学水平的有机化学课程，你可能做过利用 J 耦合来确定简单化合物（如下面所示的氯乙烷）结构的作业题。（不过，就像我的大多数学生一样，你可能只是模糊地记得做过这件事！）

​氯乙烷 (CH3-CH2-Cl) 的 ¹H 原子核处于两种不同的分子环境中：甲基 (−CH3) 和亚甲基 (−CH2−)，其主共振峰分别在 δ ≈ 1.2 和 3.6 ppm 附近。如高分辨率实验室 NMR 谱图所示，两条主谱线因 J 耦合而分裂（分别成为三重峰和四重峰）。 

多重峰中各子峰之间的间距由 耦合常数 (J) 决定，对于氯乙烷，其值约为 7 Hz。 J 与磁场强度无关，因此以 Hz 而非 ppm 为单位报告。对于 有机分子中的 ¹H-¹H 耦合， J 通常位于 0−20 Hz 范围内。

一个给定的 ¹H 谱线将被分裂的次数可以通过 n+1 规则 来预测，该规则指出多重峰的数目比连接到相邻碳原子上的氢原子数 (n) 多一个。对于氯乙烷， −CH3 上的氢原子与相邻碳原子 (−CH2−) 上的 n = 2 个氢原子相互作用，因此分裂为三重峰 (2+1=3)。  相反，亚甲基 (−CH2−) 上的氢原子与甲基 (−CH3) 上的 n = 3 个氢原子相互作用，并分裂为四重峰 (3+1=4)。多重峰子峰的相对高度通常遵循 帕斯卡三角形 的二项式系数 (1:2:1, 1:3:3:1)，而 t每个峰簇下的相对面积 (3 和 2) 反映了每个化学基团中 ¹H 原子核的数量。 
脑部 MRS 显示乳酸 (Lac) 因 J 耦合诱导分裂为双重峰，而其他主要峰表现为单峰。
在脑部 MRS 中，产生三个最大峰 — N-乙酰天冬氨酸 (NAA)、胆碱 (Cho) 和肌酸 (Cr) — 的 ¹H 原子核没有最近邻的质子，不会经历 J 耦合，因此表现为 单峰。然而，当存在时，乳酸在 δ = 1.32 附近的主峰会分裂成双重峰，这很容易观察到。这导致了一种 错误的 观念，认为乳酸是谱图中唯一会分裂的峰。事实并非如此，因为许多代谢物（包括 NAA）都有较小的或次要的峰会被 J 耦合分裂（尽管这些峰可能由于 浓度低或与其他共振峰重叠而难以看到）。

关于 J 耦合的更详细描述

如上所述，J 耦合发生在一个原子核的自旋通过成键电子影响另一个原子核的自旋时。为了理解这是如何发生的，让我们考虑一个相对简单的分子，如 氟化氢 (HF)。以下分析并非严格的量子力学分析，我们也不会完全讨论能级跃迁，但希望它能提供一些关于该现象的见解。我们首先简要回顾自旋、轨道和成键作为背景。

首先你应该记得，氢和氟的两种常见同位素 (1H 和 19F) 的核自旋 (I) 都等于 ½。这意味着每个原子核有两个“可观测”的自旋状态，我们将其表示为自旋向上 (↑) 和自旋向下 (↓)。同样，由于所有电子的自旋 = ½，它们也可以描述为自旋向上 (↑) 或自旋向下 (↓)。

你无疑知道 ¹H 原子由一个质子和一个电子组成，因此有四种质子-电子自旋态组合是可能的 (↑↑, ↑↓ , ↓↑, ↓↓)。然而，可能有点令人惊讶的是，并非所有组合的可能性都相同，因为与它们相关的能量是不相等的。
质子和电子自旋在其波函数重叠处的超精细相互作用给予反平行构型非常轻微的能量优势，这一过程称为极化。
这背后的原因在于，质子和电子并非实心球，而更适合用波函数（Ψ）描述的概率分布来表示。这些波函数在空间中重叠，意味着即使在原子核内部也存在找到电子的微小可能性！在这个小的重叠区域内，发生所谓的超精细相互作用，该相互作用在能量上小程度地倾向于质子和电子自旋处于反平行构型。

类似的超精细相互作用也发生在F原子中，其电子和原子核波函数重叠。关于F，您还需要了解的一个特点是它有9个电子，除了2pz壳层中的一个未配对电子外，所有电子都成对占据轨道。这个半充满的壳层在空间上与H的1s壳层（其单个电子所在之处）重叠，并具有与之相当的能量水平。来自这些壳层的电子（每个原子各一个）主要负责形成HF分子的σ键。

在形成HF分子时，H和F各从其1s和2pz壳层提供一个电子（e−）以创建σ键。
将这些概念结合起来，我们开始理解电子在J耦合机制中的作用。下图显示了F处的自旋如何通过电子键受到H处自旋的调制，反之亦然。
两个原子核之间J耦合的强度取决于多个因素，包括原子核类型、它们之间的距离和取向。
  1. 原子核类型。上面的例子说明了异核耦合，其中涉及的原子核不同（H和F）。在临床MRS中，¹H原子核之间的同核耦合是常态。 
  2. 原子核之间的键数。J耦合效应随着距离增加而减弱，对于相隔超过3-4个键的原子核通常无法测量（除非其中一些键是双键、三键或芳香键）。
  3. 取向。化学键的性质（σ键、π键等）、键长以及它们之间的角度以复杂的方式影响J耦合。对于相邻碳原子上的¹H原子核，可以使用半经验Karplus方程将J耦合与分子的扭转角联系起来。
J耦合的命名法

在阅读波谱学文献时，您可能会遇到以下J耦合的命名法和缩写，因此我在此提供一份简要的翻译指南。耦合常数的一般形式为NJA-X，其中耦合的两个原子核为A和X，N = 它们之间的键数。两键耦合常数（2J）称为偕偶，而三键耦合常数（3J）称为邻偶。

因此，氟化氢分子上H和F之间的耦合应表示为1JH-F，而氯乙烷相邻碳上两个氢之间的耦合应表示为3JH-H。

应当注意，耦合是对称的，意味着如果原子核X以某种方式与原子核A耦合，那么原子核A同样也与X耦合。换句话说，NJA-X = NJX-A总是成立。

谱线分裂的模式
根据参与J耦合相互作用的化学上不同原子核的数量，谱线可能分裂为二重峰、三重峰和更高级别的多重峰。确切的外观取决于耦合常数（J）相对于原子核之间频率差（Δν）的大小。如果J相对于Δν较小，则观察到一级耦合。上图氯乙烷谱图所示即为一级耦合。
谱线分裂的程度等于n+1，其中n = J耦合的原子核数量。四重峰的形成由3个耦合自旋以各种组合方式展示，产生1:3:3:1的比例，这可以通过帕斯卡三角形预测。

存在比上述更复杂的耦合模式，特别是当耦合不同或为多重时，例如形成双二重峰。当J相对于原子核之间的化学位移频率差（Δν）较大时，会发生二级耦合，在最简单的情况下表现为屋顶现象（耦合的多重峰相互倾斜），直至极其复杂的谱图，只有经验丰富的化学家才能处理。
J耦合与场强
与共振频率和化学位移不同，J耦合相互作用独立于磁场强度（Bo）。这是因为自旋-自旋耦合的源头——原子核和电子的磁矩——不依赖于外加磁场。

耦合常数（J）因此以绝对频率（Hz）而非ppm表示。对于有机分子中的¹H-¹H耦合，J通常在0−20 Hz范围内。然而，一级异核耦合常数可能大得多，¹H-¹³C和¹H-³¹P可达数百Hz。

J不依赖于场强意味着在更高的磁场下，代谢物的主谱峰在频谱上会分离得更开，可能更容易识别，但它们分裂成的二重峰、三重峰等会显得更紧密，因为它们的偏移（J）在更宽的频率尺度上保持恒定。
应当注意，尽管谱线分裂提供了关于化学结构的宝贵信息，但由于原子核信号必须分散到多个多重子峰上，最大峰高会降低。因此，矛盾的是，从检测的角度来看，谱线分裂可能使谱峰识别更加困难。
